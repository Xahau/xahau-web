title: Bibliotecas de Cliente

Essas bibliotecas de cliente simplificam parte do trabalho comum de acesso e processamento de dados do XAH Ledger e os apresentam em um formato que corresponde às convenções nativas de suas respectivas linguagens de programação.

Para outras linguagens de programação, você pode acessar o XAH Ledger por meio das APIs HTTP.
