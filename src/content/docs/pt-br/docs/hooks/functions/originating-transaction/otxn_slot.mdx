---
title: otxn_slot
description: Carrega a transação de origem em um slot
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

* Insere a transação de origem no slot especificado ou em um novo slot, caso nenhum slot seja especificado

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t otxn_slot (
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_slot(slotno: number): ErrorCode | ByteArray
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t otxn_slot_no = 
		otxn_slot(0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const otxn_slot_no = otxn_slot(0)
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome     | Tipo      | Descrição                                                                   |
| -------- | --------- | ----------------------------------------------------------------------------- |
| slot_no | uint32_t | O número do slot no qual inserir, ou 0 caso deseje escolher o próximo disponível. |
</TabItem>

<TabItem label="JavaScript">
| Nome   | Tipo   | Descrição                 |
| ------ | ------ | --------------------------- |
| slotno | number | O número do slot a ser consultado. |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                                                        |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>O slot no qual a otxn foi colocada<br /><br /><code>INVALID_ARGUMENT</code><br />- o número de slot especificado excede o maior número de slot possível<br /><br /><code>NO_FREE_SLOTS</code><br />- a solicitação não pôde ser concedida porque nenhum slot livre estava disponível para colocar a transação de origem.</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                   | Descrição                                                                                                                            |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| ErrorCode \| ByteArray | <p></p><p>Retorna o valor associado ao número de slot especificado como um array de números, ou um ErrorCode caso a busca falhe.</p> |
</TabItem>
</Tabs>
