---
title: otxn_id
description: Produz como saída o hash canônico da transação de origem
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

<Tabs>
<TabItem label="C">
* Escreve o hash canônico da transação de origem no buffer de saída.
* Se flags = 1 e a transação for uma transação EMIT_FAILURE, escreve o hash canônico da transação de origem que causou a emissão.
</TabItem>

<TabItem label="JavaScript">
* Produz como saída o hash canônico da transação de origem.
* Retorna o hash da transação como um array de números, ou um ErrorCode caso a recuperação falhe.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t otxn_id (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t flags
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_id(flag: number): ErrorCode | ByteArray
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
uint8_t txn_id[32];
int64_t bytes_written = 
    otxn_id(txn_id, 32, 0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const tx_id = otxn_id(0)
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome       | Tipo      | Descrição                                                                                                                                                                                                                 |
| ---------- | --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| write_ptr | uint32_t | Ponteiro para um buffer onde armazenar o hash.                                                                                                                                                                                   |
| write_len | uint32_t | Tamanho do buffer de saída. Deve ter pelo menos 32 bytes.                                                                                                                                                                   |
| flags      | uint32_t | <p>Se <code>0</code>:<br />Escreve o hash canônico da transação de origem.<br /><br />Se <code>1</code> E a transação de origem for uma EMIT_FAILURE:<br />Escreve o hash canônico da transação emissora.</p> |
</TabItem>

<TabItem label="JavaScript">
| Nome  | Tipo   | Descrição                                                                                                                                                                                                                 |
| ----- | ------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| flags | number | <p>Se <code>0</code>:<br />Escreve o hash canônico da transação de origem.<br /><br />Se <code>1</code> E a transação de origem for uma EMIT_FAILURE:<br />Escreve o hash canônico da transação emissora.</p> |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>O número de bytes escritos<br /><br />Se negativo, um erro:<br /><code>OUT_OF_BOUNDS</code><br />- ponteiros/tamanhos especificados fora da memória do hook.<br /><br /><code>TOO_SMALL</code><br />- o buffer de saída não era grande o suficiente para conter o objeto serializado</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                   | Descrição                                                                                  |
| ---------------------- | -------------------------------------------------------------------------------------------- |
| ErrorCode \| ByteArray | Retorna o hash da transação como um array de números, ou um ErrorCode caso a recuperação falhe. |
</TabItem>
</Tabs>
