---
title: otxn_burden
description: Obtém o burden da transação de origem
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

<Tabs>
<TabItem label="C">
* Retorna o burden da transação de origem ou `1` se nenhum campo burden estiver presente.
</TabItem>

<TabItem label="JavaScript">
* Recupera o burden da transação de origem.
* Retorna o burden como um número, ou um ErrorCode caso a recuperação falhe.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t otxn_burden (
    void
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_burden(): ErrorCode | number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t burden = 
  otxn_burden();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const burden = otxn_burden()
```
</TabItem>
</Tabs>



### Parâmetros

Nenhum

### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                |
| -------- | ---------------------------------------------------------------------------------------------------------- |
| int64_t | O burden da transação de origem, ou `1` se nenhum burden estava presente na transação de origem. |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descrição                                                             |
| ------------------- | ----------------------------------------------------------------------- |
| number ou ErrorCode | Retorna o burden como um número, ou um ErrorCode caso a recuperação falhe. |
</TabItem>
</Tabs>
