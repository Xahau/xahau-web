---
title: otxn_generation
description: Obtém a generation da transação de origem
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

<Tabs>
<TabItem label="C">
* Retorna a generation da transação de origem ou `1` se nenhum campo generation estiver presente.
</TabItem>

<TabItem label="JavaScript">
* Recupera o número de generation da transação de origem.
* Retorna o número de generation como um número.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t otxn_generation (
    void
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_generation(): number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t generation = 
  otxn_generation();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const generation = otxn_generation()
```
</TabItem>
</Tabs>



### Parâmetros

Nenhum

### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo   | Descrição                                |
| ------ | ------------------------------------------ |
| number | Retorna o número de generation como um número. |
</TabItem>

<TabItem label="JavaScript">
| Tipo     | Descrição                                                                                                        |
| -------- | ------------------------------------------------------------------------------------------------------------------ |
| int64_t | A generation da transação de origem, ou `1` se nenhuma generation estava presente na transação de origem. |
</TabItem>
</Tabs>
