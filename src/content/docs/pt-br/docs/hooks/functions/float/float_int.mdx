---
title: float_int
description: Converte um ponto flutuante XFL em um inteiro (floor)
---
import { Tabs, TabItem, LinkButton, Aside } from '@astrojs/starlight/components';

### Conceitos

<LinkButton href="/pt-br/docs/hooks/concepts/floating-point-numbers-xfl">Números de Ponto Flutuante (XFL)</LinkButton>

### Comportamento

<Tabs>
<TabItem label="C">
* Desloca à esquerda (multiplica por 10) o XFL pelo número de casas decimais especificado
* Converte o XFL resultante em um inteiro, descartando qualquer resto
* Retorna o inteiro
</TabItem>

<TabItem label="JavaScript">
* Converte uma representação de float em um inteiro com casas decimais especificadas.
* Um código de erro ou o inteiro resultante como um número.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t float_int (
    int64_t float1,
    uint32_t decimal_places,
    uint32_t absolute
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function float_int(
    f1: bigint,
    decimal_places: number,
    abs: number
  ): ErrorCode | number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t drops =
    float_int(xahbalance, 6, 0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const drops =
    float_int(xahbalance, 6, 0);
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome            | Tipo      | Descrição                                                                                              |
| --------------- | --------- | -------------------------------------------------------------------------------------------------------- |
| float1          | int64_t  | Um número de ponto flutuante de envoltório XFL representando o primeiro operando da adição                    |
| decimal_places | uint32_t | O número de casas a deslocar a vírgula decimal para a direita antes de calcular o floor do ponto flutuante. |
| absolute        | uint32_t | Se `1`, também aplica o valor absoluto ao valor antes de retorná-lo.                                          |

<Aside type="tip">
Valores de retorno negativos são reservados para códigos de erro. Portanto, se você precisar executar esta função sobre um XFL negativo, use `absolute = 1`
</Aside>
</TabItem>

<TabItem label="JavaScript">


| Nome            | Tipo   | Descrição                                   |
| --------------- | ------ | --------------------------------------------- |
| f1              | bigint | O float a ser convertido                          |
| decimal_places | number | O número de casas decimais a considerar.     |
| abs             | number | Indica se deve ser usado o valor absoluto. |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                                                                                                                                                                       |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>O inteiro positivo calculado<br /><br />Se negativo, um erro:<br /><code>INVALID_FLOAT</code><br />- um dos parâmetros fornecidos não era um número de envoltório XFL válido<br /><br /><code>INVALID_ARGUMENT</code><br />- tentativa de especificar mais de 15 casas decimais.<br /><br /><code>CANT_RETURN_NEGATIVE</code><br />- tentativa de retornar um inteiro negativo, mas isso não é permitido, use <code>absolute = 1</code></p> |
</TabItem>

<TabItem label="JavaScript">


| Tipo                | Descrição                                                 |
| ------------------- | ----------------------------------------------------------- |
| number ou ErrorCode | Retorna um código de erro ou o inteiro resultante como um número. |
</TabItem>
</Tabs>
