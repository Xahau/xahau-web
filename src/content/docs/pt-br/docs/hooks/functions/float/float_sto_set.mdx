import { Tabs, TabItem, LinkButton } from '@astrojs/starlight/components';

### Conceitos

<LinkButton href="/pt-br/docs/hooks/concepts/floating-point-numbers-xfl">Números de Ponto Flutuante (XFL)</LinkButton>
<LinkButton href="/pt-br/docs/hooks/concepts/serialized-objects">Objetos Serializados</LinkButton>

### Comportamento

<Tabs>
<TabItem label="C">
* Lê um número de ponto flutuante serializado.
* Se houver mais campos/dados após o número de ponto flutuante serializado, eles são ignorados.
* Retorna como um número de envoltório XFL
</TabItem>

<TabItem label="JavaScript">
* Define o buffer para armazenar representações de float.
* Retorna ErrorCode ou o resultado como um número.
</TabItem>
</Tabs>



### Definição

<Tabs>
<TabItem label="C">
```c
int64_t float_sto_set (
    uint32_t read_ptr,
    uint32_t read_len
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function float_sto_set(buf: ByteArray | HexString): ErrorCode | number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t vault_pusd = float_sto_set(vault, 8);
if (vault_pusd < 0)
  rollback("Failed to parse serialized float.", 33, 1);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
const vault_pusd = float_sto_set(vault)
if (typeof vault_pusd === 'string')
  rollback("Failed to parse serialized float.", 1)
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome      | Tipo      | Descrição                                                       |
| --------- | --------- | ----------------------------------------------------------------- |
| read_ptr | uint32_t | Ponteiro para um buffer contendo o XFL serializado. _Pode ser nulo._ |
| read_len | uint32_t | O tamanho do buffer.                                         |


</TabItem>

<TabItem label="JavaScript">
| Nome | Tipo                   | Descrição        |
| ---- | ---------------------- | ------------------ |
| buf  | ByteArray \| HexString | O buffer a ser definido. |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                                                      |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| int64_t | <p>O número de bytes escritos no buffer de saída.<br /><br />Se negativo, um erro:<br /><code>NOT_AN_OBJECT</code><br />- o buffer fornecido não continha um número de ponto flutuante serializado válido<br /><br /><code>OUT_OF_BOUNDS</code><br />- ponteiros/tamanhos especificados fora da memória do hook.</p>
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descrição                              |
| ------------------- | ---------------------------------------- |
| ErrorCode ou number | Um código de erro ou o resultado como um número. |
</TabItem>
</Tabs>
