import { Tabs, TabItem, LinkButton, Aside } from '@astrojs/starlight/components';

### Conceitos

<LinkButton href="/pt-br/docs/hooks/concepts/floating-point-numbers-xfl">Números de Ponto Flutuante (XFL)</LinkButton>

### Comportamento

<Tabs>
<TabItem label="C">
* Calcula a multiplicação de dois números de ponto flutuante XFL (xls17)
* Retorna um novo XFL como um int64_t
</TabItem>

<TabItem label="JavaScript">
* Calcula a multiplicação de dois números de ponto flutuante XFL (xls17)
* Retorna um código de erro ou um novo XFL como um bigint.
</TabItem>
</Tabs>



### Definição

<Tabs>
<TabItem label="C">
```c
int64_t float_multiply (
    int64_t float1,
    int64_t float2
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function float_multiply(f1: bigint, f2: bigint): ErrorCode | bigint
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t max_vault_pusd =
    float_multiply(vault_xrp, exchange_rate);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
const max_vault_pusd =
    float_multiply(vault_xrp, exchange_rate)
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">


| Nome   | Tipo     | Descrição                                                                                  |
| ------ | -------- | -------------------------------------------------------------------------------------------- |
| float1 | int64_t | Um número de ponto flutuante de envoltório XFL representando o primeiro operando da multiplicação  |
| float2 | int64_t | Um número de ponto flutuante de envoltório XFL representando o segundo operando da multiplicação |

<Aside type="caution">
Certas multiplicações podem causar overflow, retornando um erro `INVALID_FLOAT`. No entanto, um **underflow** retorna como XFL Canonical Zero (ou seja, número de envoltório = 0).
</Aside>
</TabItem>

<TabItem label="JavaScript">
| Nome | Tipo   | Descrição                                                                                  |
| ---- | ------ | -------------------------------------------------------------------------------------------- |
| f1   | bigint | Um número de ponto flutuante de envoltório XFL representando o primeiro operando da multiplicação  |
| f2   | bigint | Um número de ponto flutuante de envoltório XFL representando o segundo operando da multiplicação |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">


| Tipo     | Descrição                                                                                                                                                                                                                                                                       |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>O número de envoltório XFL (xls17)<br /><br />Se negativo, um erro:<br /><code>INVALID_FLOAT</code><br />- um dos parâmetros fornecidos não era um número de envoltório XFL válido<br /><br /><code>OVERFLOW</code><br />- o resultado da multiplicação era muito grande para ser armazenado em um XFL.</p> |
</TabItem>

<TabItem label="JavaScript">


| Tipo               | Descrição                                       |
| ------------------ | ------------------------------------------------- |
| ErrorCode ou bigint | Um código de erro ou o número de envoltório XFL (xls17) |
</TabItem>
</Tabs>
