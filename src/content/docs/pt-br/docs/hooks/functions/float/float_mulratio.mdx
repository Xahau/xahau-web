import { Tabs, TabItem, LinkButton, Aside } from '@astrojs/starlight/components';

### Conceitos

<LinkButton href="/pt-br/docs/hooks/concepts/floating-point-numbers-xfl">Números de Ponto Flutuante (XFL)</LinkButton>

### Comportamento

<Tabs>
<TabItem label="C">
* Calcula a multiplicação de um número de ponto flutuante XFL (xls17) pelo quociente de dois inteiros
* Retorna um novo XFL como um int64_t
</TabItem>

<TabItem label="JavaScript">
* Multiplica um float por uma razão definida por um numerador e um denominador.
* Retorna um código de erro ou um novo XFL como um bigint.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t float_mulratio (
    int64_t float1,
    uint32_t round_up,
    uint32_t numerator,
    uint32_t denominator
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function float_mulratio(
    f1: bigint,
    round_up: number,
    numerator: number,
    denominator: number
  ): ErrorCode | bigint
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t max_vault_pusd =
    float_mulratio(max_vault_pusd, 0,
        COLLATERALIZATION_NUMERATOR, COLLATERALIZATION_DENOMINATOR);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const max_vault_pusd =
    float_mulratio(max_vault_pusd, 0,
        COLLATERALIZATION_NUMERATOR, COLLATERALIZATION_DENOMINATOR);
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome        | Tipo      | Descrição                                                                                 |
| ----------- | --------- | ------------------------------------------------------------------------------------------- |
| float1      | int64_t  | Um número de ponto flutuante de envoltório XFL representando o primeiro operando da multiplicação |
| round_up   | uint32_t | Se diferente de zero, todos os cálculos serão arredondados para cima                                             |
| numerator   | uint32_t | O numerador do quociente pelo qual o float será multiplicado                          |
| denominator | uint32_t | O denominador do quociente pelo qual o float será multiplicado                        |

<Aside type="caution">
Certas multiplicações podem causar overflow, retornando um erro `INVALID_FLOAT`. No entanto, um **underflow** retorna como XFL Canonical Zero (ou seja, número de envoltório = 0).
</Aside>
</TabItem>

<TabItem label="JavaScript">


| Nome        | Tipo   | Descrição                                                                                 |
| ----------- | ------ | ------------------------------------------------------------------------------------------- |
| f1          | bigint | Um número de ponto flutuante de envoltório XFL representando o primeiro operando da multiplicação |
| round_up   | number | Se diferente de zero, todos os cálculos serão arredondados para cima                                             |
| numerator   | number | O numerador do quociente pelo qual o float será multiplicado                          |
| denominator | number | O denominador do quociente pelo qual o float será multiplicado                        |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                                                                                                                    |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>O número de envoltório XFL (xls17)<br /><br />Se negativo, um erro:<br /><code>INVALID_FLOAT</code><br />- um dos parâmetros fornecidos não era um número de envoltório XFL válido<br /><br /><code>OVERFLOW</code><br />- o resultado da multiplicação era muito grande para ser armazenado em um XFL.<br /><br /><code>DIVISION_BY_ZERO</code><br />- o denominador fornecido era zero.</p>


</TabItem>

<TabItem label="JavaScript">


| Tipo                | Descrição                                                 |
| ------------------- | ----------------------------------------------------------- |
| ErrorCode ou bigint | Retorna um código de erro ou o número de envoltório XFL (xls17). |
</TabItem>
</Tabs>
