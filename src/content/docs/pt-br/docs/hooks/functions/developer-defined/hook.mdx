### Comportamento

* `hook` é uma função definida pelo usuário, chamada pelo `xahaud` para disparar seu hook.
* Sua função `hook` chama `accept` ou `reject` para aprovar ou rejeitar a transação de origem.
* Se a execução chegar ao final da função, isso é implicitamente um `accept`.
