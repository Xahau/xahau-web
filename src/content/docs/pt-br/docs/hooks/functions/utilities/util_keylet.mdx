Nem todo tipo de Keylet é suportado por esta função utilitária. Se você precisar de outro tipo de Keylet, pode derivá-lo você mesmo usando [util_sha512h](https://xrpl-hooks.readme.io/reference/util_sha512h) e verificando os campos necessários [aqui](https://github.com/Xahau/xahaud/blob/dev/src/ripple/protocol/impl/Indexes.cpp). Uma ferramenta adicional de Keylet pode [te ajudar.](https://richardah.github.io/xrpl-keylet-tools/)
</Aside>

### Comportamento

* Calcula um keylet do `keylet_type` especificado de acordo com os parâmetros `a` a `f`, dependendo do tipo.
* Escreve o keylet serializado de 34 bytes em `write_ptr`

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t util_keylet (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t keylet_type,
    uint32_t a,
    uint32_t b,
    uint32_t c,
    uint32_t d,
    uint32_t e,
    uint32_t f
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function util_keylet(
    keylet_type: number,
    accountid: ByteArray | HexString,
    statekey: ByteArray | HexString,
    namespace: ByteArray | HexString
  ): ErrorCode | ByteArray
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
uint8_t keylet[34];
if (util_keylet(keylet, 34, KEYLET_LINE,
              hook_accid, 20,
              account_field, 20,
              currency_code, 20) != 34)
	rollback("Keylet Failed.", 14, 1);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
if (util_keylet(
        KEYLET_LINE, hook_accid, account_field, currency_code
        ).length < 34) 
    rollback("Keylet Failed.", 1)
```
</TabItem>
