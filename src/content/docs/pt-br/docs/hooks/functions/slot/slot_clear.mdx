---
title: slot_clear
description: Libera um slot atualmente ocupado
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

* Libera o slot especificado, liberando qualquer objeto que estivesse alocado nele

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t slot_clear (
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_clear(slotno: number): ErrorCode | number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
slot_clear(1); // assumes a transaction is slotted into slot=1
```
</TabItem>

<TabItem label="JavaScript">
```javascript
slot_clear(1) // assumes a transaction is slotted into slot=1
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome     | Tipo      | Descrição     |
| -------- | --------- | --------------- |
| slot_no | uint32_t | O número do slot |
</TabItem>

<TabItem label="JavaScript">
| Nome   | Tipo   | Descrição     |
| ------ | ------ | --------------- |
| slotno | number | O número do slot |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                               |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p><code>1</code> ou um erro<br /><br />Se negativo, um erro:<br /><code>DOESNT_EXIST</code><br />- o slot especificado não contém nenhum objeto ou é um slot inválido</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descrição                                                 |
| ------------------- | ----------------------------------------------------------- |
| ErrorCode \| number | Retorna um código de erro ou o resultado da operação de limpeza. |
</TabItem>
</Tabs>
