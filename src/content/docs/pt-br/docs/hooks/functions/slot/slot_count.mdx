---
title: slot_count
description: Conta os elementos de um objeto array em um slot
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

* Conta os elementos de um array no slot especificado
* Retorna a contagem

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t slot_count (
    uint32_t slot_no
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_count(slotno: number): ErrorCode | number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
slot_count(1); // assumes an array is slotted into slot=1
```
</TabItem>

<TabItem label="JavaScript">
```javascript
slot_count(1) // assumes an array is slotted into slot=1
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome     | Tipo      | Descrição     |
| -------- | --------- | --------------- |
| slot_no | uint32_t | O número do slot |
</TabItem>

<TabItem label="JavaScript">
| Nome   | Tipo   | Descrição     |
| ------ | ------ | --------------- |
| slotno | number | O número do slot |
</TabItem>
</Tabs>

### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                                              |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>O número de elementos dentro do array alocado no slot<br /><br />Se negativo, um erro:<br /><code>DOESNT_EXIST</code><br />- o slot especificado não contém nenhum objeto ou é um slot inválido<br /><br /><code>NOT_AN_ARRAY</code><br />- o slot especificado não contém um objeto array</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descrição                                    |
| ------------------- | ---------------------------------------------- |
| ErrorCode or number | Retorna um código de erro ou a contagem de entradas. |
</TabItem>
</Tabs>
