---
title: slot_float
description: >-
  Analisa o STI_AMOUNT no slot especificado e o retorna como um número
  encapsulado em XFL
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

<Tabs>
<TabItem label="C">
* Analisa o STI_AMOUNT no slot especificado e o retorna como um número encapsulado em XFL
</TabItem>

<TabItem label="JavaScript">
* Recupera o valor bigint associado ao slot especificado.
* Retorna um código de erro ou o valor bigint.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t slot_float (
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_float(slotno: number): ErrorCode | bigint
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t xfl =
  	slot_float(amt_slot);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
const xfl = slot_float(amt_slot)
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome     | Tipo      | Descrição     |
| -------- | --------- | --------------- |
| slot_no | uint32_t | O número do slot |
</TabItem>

<TabItem label="JavaScript">
| Nome   | Tipo   | Descrição                                        |
| ------ | ------ | -------------------------------------------------- |
| slotno | number | O número do slot do qual recuperar o valor bigint. |
</TabItem>
</Tabs>

### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                                          |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| int64_t | <p>O número encapsulado em XFL<br /><br />Se negativo, um erro:<br /><code>DOESNT_EXIST</code><br />- o slot especificado não contém nenhum objeto ou é um slot inválido<br /><br /><code>NOT_AN_AMOUNT</code><br />- o slot especificado não contém um objeto <code>STI_AMOUNT</code></p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descrição                                |
| ------------------- | ------------------------------------------ |
| bigint or ErrorCode | Retorna um código de erro ou o valor bigint. |
</TabItem>
</Tabs>
