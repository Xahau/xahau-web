---
title: slot_type
description: >-
  Recupera o código de campo de um objeto em um slot e, opcionalmente, outras
  informações
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

<Tabs>
<TabItem label="C">
* Localiza o objeto apontado pelo `slot_no` especificado
* Determina seu código de campo `sf` e o retorna, ou retorna outras informações (veja abaixo) caso `flags` seja usado
</TabItem>

<TabItem label="JavaScript">
* Recupera o tipo do slot especificado.
* Retorna um código de erro ou o tipo do slot.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t slot_type (
    uint32_t slot_no,
    uint32_t flags
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_type(slotno: number, flags: number): ErrorCode | number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t fidle_code = 
    slot_type(1, 0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const field_code = slot_type(1, 0)
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome     | Tipo      | Descrição                                                                                                                                                                                                          |
| -------- | --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| slot_no | uint32_t | O número do slot                                                                                                                                                                                                      |
| flags    | uin32_t  | <p>Para operação normal, isso deve ser <code>0</code>.<br /><br />Para determinar se um tipo <code>STI_AMOUNT</code> contém um valor nativo (XAH) ou um valor de ponto flutuante (IOU), defina como <code>1</code>.</p> |
</TabItem>

<TabItem label="JavaScript">
| Nome   | Tipo   | Descrição                           |
| ------ | ------ | ------------------------------------- |
| slotno | number | O número do slot do qual verificar o tipo. |
| flags  | number | Flags para determinar o tipo.          |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>Se <code>flags</code> for <code>0</code>, então:<br />O código de campo <code>sf</code> do objeto alocado no slot<br /><br />Se <code>flags</code> for <code>1</code>, então:<br /><code>1</code> se, e somente se, o objeto alocado no slot for um <code>STI_AMOUNT</code> e o tipo do valor for XAH.<br /><br />Se negativo, um erro:<br /><code>DOESNT_EXIST</code><br />- o <code>slot_no</code> especificado não contém um objeto.<br /><br /><code>NOT_AN_AMOUNT</code><br />- <code>flags</code> foi definido como <code>1</code>, mas o objeto alocado no slot não é um objeto <code>STI_AMOUNT</code></p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descrição                                    |
| ------------------- | ---------------------------------------------- |
| ErrorCode or number | Retorna um código de erro ou o tipo do slot. |
</TabItem>
</Tabs>
