---
title: slot_size
description: Calcula o tamanho serializado de um objeto em um slot
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

<Tabs>
<TabItem label="C">
* Retorna o número de bytes que o objeto no slot especificado ocupa quando serializado
</TabItem>

<TabItem label="JavaScript">
* Recupera o tamanho do slot especificado.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t slot_size (
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_size(slotno: number): ErrorCode | number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t bytes_needed = slot_size(1); // get size of slot 1
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const bytes_needed = slot_size(1); // get size of slot 1
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome     | Tipo      | Descrição     |
| -------- | --------- | --------------- |
| slot_no | uint32_t | O número do slot |
</TabItem>

<TabItem label="JavaScript">
| Nome   | Tipo   | Descrição                           |
| ------ | ------ | ------------------------------------- |
| slotno | number | O número do slot do qual verificar o tamanho. |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                            |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| int64_t | <p>O número de bytes que o objeto ocupa quando serializado<br /><br />Se negativo, um erro:<br /><code>DOESNT_EXIST</code><br />- o slot especificado não contém nenhum objeto ou é um slot inválido</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descrição                                    |
| ------------------- | ---------------------------------------------- |
| ErrorCode \| number | Retorna um código de erro ou o tamanho do slot. |
</TabItem>
</Tabs>
