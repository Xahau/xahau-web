title: state_foreign_set
description: Define o Estado do Hook em outra conta para uma determinada chave, valor e namespace

import { Tabs, TabItem, Aside } from '@astrojs/starlight/components';

### Comportamento

<Tabs>
<TabItem label="C">
* Lê uma chave do Estado do Hook de 32 bytes a partir de `kread_ptr`
* Lê uma quantidade arbitrária de dados a partir de `read_ptr` (o valor)
* Lê um Namespace de 32 bytes a partir de `nread_ptr`
* Lê um ID de Conta de 20 bytes a partir de `aread_ptr`
* Atualiza a chave do Estado do Hook na conta especificada, dentro do namespace especificado, com o valor
* Mas apenas se uma [Concessão](https://xrpl-hooks.readme.io/docs/grants) naquela conta permitir isso.
* Se a Conta Hook for especificada em `aread_ptr`, então o comportamento é o mesmo de state_set, mas ainda permite a especificação do namespace através de `nread_ptr`
</TabItem>

<TabItem label="JavaScript">
* Define o Estado do Hook Estrangeiro com o valor, chave, namespace e ID de conta especificados.
* Esta operação requer autorização e uma Concessão para permitir esta ação.
* Retorna o número de bytes escritos no Estado do Hook (o tamanho dos dados), negativo em caso de erro.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t state_foreign_set (
    uint32_t read_ptr,
    uint32_t read_len,
    uint32_t kread_ptr,
    uint32_t kread_len,
    uint32_t nread_ptr,
    uint32_t nread_len,
    uint32_t aread_ptr,
    uint32_t aread_len  
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function state_foreign_set(
    value: ByteArray | HexString | undefined | null,
    key: ByteArray | HexString,
    namespace: ByteArray | HexString | undefined | null,
    accountid: ByteArray | HexString | undefined | null
  ): ErrorCode | number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
#define SBUF(str) (uint32_t)(str), sizeof(str)
if (state_foreign_set(SBUF(vault), SBUF(vault_key), SBUF(namespace), SBUF(account)) < 0)
    rollback(SBUF("Error: could not set foreign state!"), 1);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
if (state_foreign_set(vault, vault_key, namespace, account) < 0)
    rollback("Error: could not set foreign state!", 1);
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome       | Tipo      | Descrição                                                                                                                                              |
| ---------- | --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| read_ptr  | uint32_t | <p>Ponteiro para os dados (valor) a serem escritos no Estado do Hook.<br />Se for <code>0</code> (nulo), então os dados nessa chave são excluídos. <em>Pode ser nulo.</em></p> |
| read_len  | uint32_t | <p>O tamanho dos dados.<br />Se for <code>0</code> (nulo), então os dados nessa chave são excluídos. <em>Pode ser nulo.</em></p>                               |
| kread_ptr | uint32_t | Um ponteiro para a chave do Estado do Hook na qual armazenar o valor.                                                                                             |
| kread_len | uint32_t | O tamanho da chave. (Deve ser sempre 32.)                                                                                                            |
| nread_ptr | uint32_t | Um ponteiro para o namespace ao qual a chave pertence.                                                                                                     |
| nread_len | uint32_t | O tamanho do namespace. (Deve ser sempre 32.)                                                                                                      |
| aread_ptr | uint32_t | Um ponteiro para o ID de Conta cujo estado estamos tentando modificar.                                                                                         |
| aread_len | uint32_t | O tamanho do ID de Conta. (Deve ser sempre 20.)                                                                                                     |

<Aside type="caution">
O Xahau define internamente um tamanho máximo de dados do hook. No momento em que este texto foi escrito, e para a testnet pública, esse valor está fixado em 128 bytes; no entanto, no futuro será um número votável pelos validadores.
</Aside>
</TabItem>

<TabItem label="JavaScript">
| Nome      | Tipo                   | Descrição                                     |
| --------- | ---------------------- | ----------------------------------------------- |
| value     | ByteArray or HexString | O valor dos dados a persistir.                   |
| key       | ByteArray or HexString | A chave do Estado do Hook para a qual definir o valor. |
| namespace | ByteArray or HexString | O namespace do Hook no qual procurar.                  |
| accountid | ByteArray or HexString | O proprietário do estado.                         |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>O número de bytes escritos no Estado do Hook (o tamanho dos dados.)<br /><br />Se negativo, um erro:<br /><code>OUT_OF_BOUNDS</code><br />- ponteiros/tamanhos especificados fora da memória do hook.<br /><br /><code>TOO_BIG</code><br />- <code>kread_len</code> era maior que 32, ou<br />- <code>read_len</code> era maior que o tamanho máximo de dados do hook.<br /><br /><code>TOO_SMALL</code><br />- <code>kread_len</code> era 0.<br /><br /><code>NOT_AUTHORIZED</code><br />- nenhum HookGrant apropriado estava presente na conta estrangeira para permitir essa mutação de estado.<br /><br /><code>PREVIOUS_FAILURE_PREVENTS_RETRY</code><br />- durante esta execução, uma chamada anterior a <code>state_foreign_set</code> falhou com NOT_AUTHORIZED e, consequentemente, nenhuma nova chamada a esta API é permitida durante esta execução.</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descrição                                                                                    |
| ------------------- | ---------------------------------------------------------------------------------------------- |
| number or ErrorCode | Retorna o número de bytes escritos no Estado do Hook (o tamanho dos dados), negativo em caso de erro. |
</TabItem>
</Tabs>
