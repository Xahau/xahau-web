### Comportamento

* Retorna o Timestamp Xahau do último ledger fechado.

<Aside type="tip">
Os timestamps Xahau são idênticos a timestamps unix, exceto que possuem um deslocamento de `-946684800`.

O timestamp unix equivalente é: `ledger_last_time() + 946684800;`
</Aside>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t ledger_last_time();
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function ledger_last_time(): ErrorCode | number
```
</TabItem>
</Tabs>

<Tabs>

### Parâmetros

Esta API não recebe parâmetros.
