---
title: ledger_nonce
description: Gera um nonce de 32 bytes para uso em uma transação emitida
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

<Tabs>
<TabItem label="C">
* Escreve um valor aleatório de 32 bytes no write_ptr
</TabItem>

<TabItem label="JavaScript">
* Recupera o nonce do ledger atual.
</TabItem>
</Tabs>



### Definição

C

<Tabs>
<TabItem label="C">
```c
int64_t ledger_nonce (
    uint32_t write_ptr,
    uint32_t write_len
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function ledger_nonce(): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### Exemplo

<Tabs>
<TabItem label="C">
```c
uint8_t n[32];
int64_t bytes_written = 
    ledger_nonce(n, 32);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
const nonce = ledger_nonce()
```
</TabItem>
</Tabs>



### Parâmetros

<Tabs>
<TabItem label="C">
| Nome       | Tipo      | Descrição                                                                              |
| ---------- | --------- | ---------------------------------------------------------------------------------------- |
| write_ptr | uint32_t | Ponteiro para um buffer de tamanho adequado para armazenar a saída. Deve ter pelo menos 32 bytes. |
| write_len | uint32_t | Tamanho do buffer de saída.                                                             |


</TabItem>

<TabItem label="JavaScript">

</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">


| Tipo                   | Descrição                                                                                                                           |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| ErrorCode \| ByteArray | Retorna um código de erro se ocorrer um erro, ou um array representando o nonce do ledger atual. especificados fora da memória do hook. |
</TabItem>

<TabItem label="JavaScript">


| Tipo     | Descrição                                                                                                                                            |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| int64_t | <p>O número de bytes escritos<br /><br />Se negativo, um erro:<br /><code>OUT_OF_BOUNDS</code><br />- ponteiros/tamanhos especificados fora da memória do hook.</p> |
</TabItem>
</Tabs>
