### Novos recursos

A API `account_info` agora fornecerá, além de seu comportamento normal:

* Uma lista de namespaces atualmente em uso por pelo menos um Objeto de Estado do Hook
* Uma contagem total de todos os Objetos de Estado do Hook que a conta possui

Exemplo de consulta:

JSON

```json
{
  "command": "account_info",
  "account": "raaFre81618XegCrzTzVotAmarBcqNSAvK"
}
```

Ex.

JSON

```json
{
  "result": {
    "account_data": {
      "Account": "raaFre81618XegCrzTzVotAmarBcqNSAvK",
      ...,
      "HookNamespaces": [
        "01EAF09326B4911554384121FF56FA8FECC215FDDE2EC35D9E59F2C53EC665A0"
      ],
      "HookStateCount": 49,
      ...
    },
    "ledger_current_index": 5555046,
    "validated": false
  },
  "status": "success",
  "type": "response"
}
```
