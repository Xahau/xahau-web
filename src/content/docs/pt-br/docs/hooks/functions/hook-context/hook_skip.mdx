---
title: hook_skip
description: Ignora um hook que aparece mais adiante na cadeia de hooks da conta hook
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

<Tabs>
<TabItem label="C">
* Procura na cadeia de hooks por um hook identificado pelo hash do hook em `read_ptr`
* Marca-o como desabilitado para esta execução da cadeia
</TabItem>

<TabItem label="JavaScript">
* Ignora a execução de um hook com base no hash e na flag fornecidos.
* Retorna um código de status indicando o resultado da operação.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t hook_skip (
    uint32_t read_ptr,
    uint32_t read_len,
    uint32_t flags
);
```
</TabItem>
<TabItem label="JavaScript">
```javascript
function hook_skip(
    hash: ByteArray | HexString,
    flag: number
  ): ErrorCode | number
```
</TabItem>
</Tabs>

### Exemplo

<Tabs>
<TabItem label="C">
```c
uint8_t phash[] = { 0x19U, 0xFEU, 0x69U, 0xF1U, 0x53U, 0x66U, 0x4EU, 0x8CU, 
                    0x97U, 0xF4U, 0x4CU, 0x5CU, 0x3CU, 0x65U, 0x63U, 0x79U, 
                    0xC2U, 0xD0U, 0x26U, 0xE7U, 0x90U, 0xEFU, 0x38U, 0xF7U, 
                    0xEDU, 0x73U, 0xE9U, 0xCEU, 0x9CU, 0x9DU, 0xBFU, 0x03U };
int64_t result = 
  	hook_skip(phash, 32, 0);
```
</TabItem>
<TabItem label="JavaScript">
```javascript
const phash = [ 0x19, 0xFE, 0x69, 0xF1, 0x53, 0x66, 0x4E, 0x8C, 
                0x97, 0xF4, 0x4C, 0x5C, 0x3C, 0x65, 0x63, 0x79, 
                0xC2, 0xD0, 0x26, 0xE7, 0x90, 0xEF, 0x38, 0xF7, 
                0xED, 0x73, 0xE9, 0xCE, 0x9C, 0x9D, 0xBF, 0x03 ]
const result = hook_skip(phash, 0);
```
</TabItem>
</Tabs>

### Parâmetros

<Tabs>
<TabItem label="C">


| Nome      | Tipo      | Descrição                                                                                                   |
| --------- | --------- | ------------------------------------------------------------------------------------------------------------- |
| read_ptr | uint32_t | Ponteiro para um buffer contendo o hash do hook                                                                  |
| read_len | uint32_t | Tamanho do hash do hook (sempre 32)                                                                           |
| flags     | uint32_t | <p>Se 0:<br />- adiciona o hash à lista de hooks ignorados<br /><br />Se 1<br />- remove o hash da lista de hooks ignorados</p> |
</TabItem>

<TabItem label="JavaScript">


| Nome | Tipo                   | Descrição                                         |
| ---- | ---------------------- | --------------------------------------------------- |
| hash | ByteArray ou HexString | O hash do hook a ser ignorado.                       |
| flag | number                 | Uma flag indicando o motivo para ignorar o hook. |
</TabItem>
</Tabs>

</TabItem>
</Tabs>
