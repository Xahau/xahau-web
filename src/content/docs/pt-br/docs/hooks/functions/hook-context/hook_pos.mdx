---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamento

* Retorna a posição na cadeia de hooks que o hook atualmente em execução ocupa.

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t hook_pos(void);
```
</TabItem>
<TabItem label="JavaScript">
```javascript
hook_pos()
```
</TabItem>
</Tabs>

### Exemplo

<Tabs>
<TabItem label="C">
```c
int64_t pos = 
  	hook_pos();
```
</TabItem>
<TabItem label="JavaScript">
```javascript
const pos = hook_pos()
```
</TabItem>
</Tabs>

### Parâmetros

Esta API não possui parâmetros

### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                               |
| -------- | ----------------------------------------------------------------------------------------- |
| int64_t | A posição na cadeia que o hook atualmente em execução ocupa. A primeira posição é 0. |
</TabItem>
<TabItem label="JavaScript">
| Tipo   | Descrição                                                                              |
| ------ | ---------------------------------------------------------------------------------------- |
| number | Retorna a posição atual na cadeia de hooks, ou um código de erro se a recuperação falhar. |
</TabItem>
</Tabs>
