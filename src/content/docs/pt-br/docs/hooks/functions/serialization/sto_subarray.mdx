### Conceitos

<LinkButton href="/pt-br/docs/hooks/concepts/serialized-objects">Objetos Serializados</LinkButton>

### Comportamento

<Tabs>
<TabItem label="C">
* Analisa um STArray apontado por `read_ptr`
* Encontra o índice do array especificado por `array_id`
* Retorna o deslocamento (offset) em bytes e o tamanho do campo serializado dentro do STObject, se encontrado

<Aside type="caution" title="Codificação do Field ID">
As APIs `sto_` aceitam um parâmetro `field_id` codificado da seguinte forma: `(type << 16U) + field`
Assim, o tipo 1 campo 2 seria `0x10002U`.

Neste caso, o field ID do array é `array_id`.
</Aside>
</TabItem>

<TabItem label="JavaScript">
* Solicita o objeto STO (dados de ledger codificados em binário) do qual o subarray será extraído.
* Encontra o índice do array especificado por `array_id`
* Retorna um subarray de um objeto STO.
</TabItem>
</Tabs>

### Definição

<Tabs>
<TabItem label="C">
```c
int64_t sto_subarray (
    uint32_t read_ptr,
    uint32_t read_len,
    uint32_t array_id
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function sto_subarray(
    sto: ByteArray | HexString,
    array_id: number
  ): bigint | number
```
</TabItem>
</Tabs>



### Exemplo

<Tabs>
<TabItem label="C">
```c
#define SUB_OFFSET(x) ((int32_t)(x >> 32))
#define SUB_LENGTH(x) ((int32_t)(x & 0xFFFFFFFFULL))

  int64_t memo_lookup =
    sto_subarray(memos, memos_len, 0);

if (memo_lookup < 0)
{
    // sfMemo was not found in the STObject pointed at by memo_ptr
}
else
{
    // 0th index of the STArray was found and its location is as follows:
    uint8_t*  memo_ptr = SUB_OFFSET(memo_lookup) + memos;
    uint32_t  memo_len = SUB_LENGTH(memo_lookup);
}
```

<Aside type="tip">
O hookmacro.h já contém as macros `SUB_OFFSET` e `SUB_LENGTH`.
</Aside>
</TabItem>

<TabItem label="JavaScript">
```javascript
const SUB_OFFSET = (x) => Number(x >> 32n)
const SUB_LENGTH = (x) => Number(x & 0xFFFFFFFFn)

const memo_lookup = sto_subarray(memos, 0)

if (typeof memo_lookup === 'number')
{
    // sfMemo was not found in the STObject pointed at
}
else
{
    // 0th index of the STArray was found and its location is as follows:
    const memo_start = SUB_OFFSET(memo_lookup)
    const memo_len = SUB_LENGTH(memo_lookup)
    const memo = memos.slice(memo_start, memo_len)
}
```
</TabItem>
</Tabs>

### Parâmetros

<Tabs>
<TabItem label="C">
| Nome      | Tipo      | Descrição                                                               |
| --------- | --------- | ------------------------------------------------------------------------- |
| read_ptr | uint32_t | Ponteiro para o buffer contendo o STArray                              |
| read_len | uint32_t | Tamanho do STArray                                                         |
| array_id | uint32_t | O índice da entrada dentro do STArray que você está buscando. Começa em 0. |


</TabItem>

<TabItem label="JavaScript">


| Nome      | Tipo                   | Descrição                                                                     |
| --------- | ---------------------- | ------------------------------------------------------------------------------- |
| sto       | ByteArray \| HexString | O objeto STO (dados de ledger codificados em binário) do qual o subarray será extraído. |
| array_id | number                 | O ID do array a ser extraído.                                            |
</TabItem>
</Tabs>



### Código de Retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>A localização do campo dentro do buffer especificado:<br />- Os 32 bits superiores são o deslocamento (offset).<br />- Os 32 bits inferiores são o tamanho.<br /><br />Se negativo, um erro:<br /><code>OUT_OF_BOUNDS</code><br />- ponteiros/tamanhos especificados fora da memória do hook.<br /><br /><code>TOO_SMALL</code><br />- O buffer de entrada não é grande o suficiente para possivelmente conter um STArray válido.<br /><br /><code>DOESNT_EXIST</code><br />- O índice buscado não está presente no STArray fornecido.<br /><br /><code>PARSE_ERROR</code><br />- O STArray fornecido está malformado ou não é um STArray.</p> |


</TabItem>

<TabItem label="JavaScript">


| Tipo               | Descrição                                                                                                                                                                                        |
| ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| bigint / ErrorCode | <p>A localização do campo dentro do buffer especificado:<br />- Os 32 bits superiores são o deslocamento (offset).<br />- Os 32 bits inferiores são o tamanho.<br /><br />ou um código de erro se a extração falhar.</p> |
</TabItem>
