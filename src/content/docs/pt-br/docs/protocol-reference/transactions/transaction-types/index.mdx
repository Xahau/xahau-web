## Tipos de Transação

O tipo de uma transação (campo `TransactionType`) é a informação mais fundamental sobre uma transação. Isso indica que tipo de operação a transação deve realizar.

Todas as transações possuem certos campos em comum:

* Campos Comuns

Cada tipo de transação possui campos adicionais relevantes ao tipo de ação que causa.
