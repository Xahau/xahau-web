---
title: otxn_id
description: 输出原始交易的规范哈希
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 行为

<Tabs>
<TabItem label="C">
* 将原始交易的规范哈希写入输出缓冲区
* 如果 `flags = 1` 且该交易是 `EMIT_FAILURE` 交易，则写入导致此次发射的原始交易规范哈希
</TabItem>

<TabItem label="JavaScript">
* 输出原始交易的规范哈希。
* 成功时返回交易哈希数组，获取失败时返回 ErrorCode。
</TabItem>
</Tabs>

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t otxn_id (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t flags
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_id(flag: number): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### 示例

<Tabs>
<TabItem label="C">
```c
uint8_t txn_id[32];
int64_t bytes_written = 
    otxn_id(txn_id, 32, 0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const tx_id = otxn_id(0)
```
</TabItem>
</Tabs>

### 参数

<Tabs>
<TabItem label="C">
| Name       | Type      | Description |
| ---------- | --------- | ----------- |
| write_ptr | uint32_t | 指向用于存储哈希的缓冲区 |
| write_len | uint32_t | 输出缓冲区长度，至少应为 32 字节 |
| flags      | uint32_t | <p>若为 <code>0</code>：<br />写入原始交易的规范哈希。<br /><br />若为 <code>1</code> 且原始交易是 `EMIT_FAILURE`：<br />写入发射交易的规范哈希。</p> |
</TabItem>

<TabItem label="JavaScript">
| Name  | Type   | Description |
| ----- | ------ | ----------- |
| flags | number | <p>若为 <code>0</code>：<br />写入原始交易的规范哈希。<br /><br />若为 <code>1</code> 且原始交易是 `EMIT_FAILURE`：<br />写入发射交易的规范哈希。</p> |
</TabItem>
</Tabs>

### 返回码

<Tabs>
<TabItem label="C">
| Type     | Description |
| -------- | ----------- |
| int64_t | <p>写入的字节数<br /><br />若为负数，则表示错误：<br /><code>OUT_OF_BOUNDS</code><br />- 指针或长度超出 Hook 内存范围。<br /><br /><code>TOO_SMALL</code><br />- 输出缓冲区不足以容纳序列化对象</p> |
</TabItem>

<TabItem label="JavaScript">
| Type                   | Description |
| ---------------------- | ----------- |
| ErrorCode \| ByteArray | 成功时返回交易哈希数组，失败时返回 ErrorCode。 |
</TabItem>
</Tabs>
