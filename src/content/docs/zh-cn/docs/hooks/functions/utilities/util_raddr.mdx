---
title: util_raddr
description: 将 20 字节 Account ID 转换为 r-address
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 行为

<Tabs>
<TabItem label="C">
* 从 `read_ptr` 读取一个 20 字节 Account ID
* 向 `write_ptr` 写入对应的 r-address
</TabItem>

<TabItem label="JavaScript">
* 从 `accountid` 读取一个 20 字节 Account ID
* 返回对应的 r-address
</TabItem>
</Tabs>

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t util_raddr (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t read_ptr,
    uint32_t read_len
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function util_raddr(accountid: ByteArray | HexString): ErrorCode | string
```
</TabItem>
</Tabs>

### 返回码

<Tabs>
<TabItem label="C">
| Type     | Description |
| -------- | ----------- |
| int64_t | <p>写入的字节数（输出 r-address 长度）。<br /><br />若为负数，则表示错误：<br /><code>OUT_OF_BOUNDS</code><br />- 指针或长度超出 Hook 内存范围。<br /><br /><code>INVALID_ARGUMENT</code><br />- <code>read_len</code> 不是 20。<br /><br /><code>TOO_SMALL</code><br />- <code>write_len</code> 不足以容纳生成的 r-address。</p> |
</TabItem>

<TabItem label="JavaScript">
| Type          | Description |
| ------------- | ----------- |
| string/number | 格式化失败时返回 ErrorCode，否则返回 r-address 字符串。 |
</TabItem>
</Tabs>
