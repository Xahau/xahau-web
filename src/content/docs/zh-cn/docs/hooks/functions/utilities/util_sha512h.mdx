---
title: util_sha512h
description: 对数据计算 sha512-half
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 行为

<Tabs>
<TabItem label="C">
* 对 `read_ptr` 指向的数据计算 `SHA512` 哈希
* 将哈希前半部分写入 `write_ptr`
</TabItem>

<TabItem label="JavaScript">
* 对 `data` 指向的数据计算 `SHA512` 哈希
* 返回哈希的前半部分
</TabItem>
</Tabs>

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t util_sha512h (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t read_ptr,
    uint32_t read_len
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function util_sha512h(data: ByteArray | HexString): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### 返回码

<Tabs>
<TabItem label="C">

</TabItem>
</Tabs>
