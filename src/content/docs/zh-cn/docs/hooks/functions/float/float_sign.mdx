title: float_sign

---
import { Tabs, TabItem, LinkButton, Aside } from '@astrojs/starlight/components';

### 概念

<LinkButton href="/docs/hooks/concepts/floating-point-numbers-xfl">浮点数（XFL）</LinkButton>

### 行为

<Tabs>
<TabItem label="C">
* 如果 XFL 为负，则返回 `1`，否则返回 `0`
</TabItem>

<TabItem label="JavaScript">
* 如果 XFL 为负，则返回 `1`，否则返回 `0`
</TabItem>
</Tabs>

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t float_sign (
    int64_t float1
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function float_sign(f1: bigint): ErrorCode | number
```
</TabItem>
</Tabs>

<Aside type="tip">
XFL 内部符号位在负数时为 `0`，但本函数遵循通用计算习惯：负数返回 `1`。
</Aside>

### 返回码

<Tabs>
<TabItem label="C">
| Type     | Description |
| -------- | ----------- |
| int64_t | <p>XFL 的符号：<br /><code>0</code> 表示正，<code>1</code> 表示负。<br /><br />若为负数，则表示错误：<br /><code>INVALID_FLOAT</code><br />- 参数不是合法 XFL</p> |
</TabItem>

<TabItem label="JavaScript">
| Type                | Description |
| ------------------- | ----------- |
| bigint or ErrorCode | 返回错误码或符号结果。 |
</TabItem>
</Tabs>
