### 定义

<Tabs>
<TabItem label="C">
由于指数可能为负，而负数被保留给错误状态，因此无法通过普通函数返回指数，所以此接口改为如下宏：

```c
#define float_exponent(f)\
        (((int32_t)(((f) >> 54U) & 0xFFU)) - 97)
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const float_exponent = (f) => (Number(((f) >> 54n) & 0xFFn) - 97)
```
</TabItem>
