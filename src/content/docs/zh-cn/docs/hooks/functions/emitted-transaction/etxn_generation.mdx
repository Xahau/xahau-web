import { Tabs, TabItem, LinkButton } from '@astrojs/starlight/components';

### 概念

<LinkButton href="/docs/hooks/concepts/emitted-transactions">发射交易</LinkButton>

### 行为

* 返回发射交易将携带的 generation。

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t etxn_generation (
    void
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function etxn_generation(): ErrorCode | number
```
</TabItem>
</Tabs>

### 示例

<Tabs>
<TabItem label="C">
```c
int64_t generation = 
  etxn_generation();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const generation = etxn_generation()
```
</TabItem>
</Tabs>

### 参数

无

### 返回码

<Tabs>
<TabItem label="C">
| Type     | Description |
| -------- | ----------- |
| int64_t | 发射交易为了成功传递给 `emit()` 所需携带的 generation |
</TabItem>

<TabItem label="JavaScript">
| Type   | Description |
| ------ | ----------- |
| number | 如果出错返回 ErrorCode，成功时返回表示 generation 结果的数值。 |
</TabItem>
</Tabs>
