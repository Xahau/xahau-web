---
title: trace
description: 将缓冲区内容写入 Xahaud trace 日志
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 行为

* 将 Hook 内部缓冲区内容写入 trace 日志，并可附带一段消息

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t trace (
    uint32_t mread_ptr,
    uint32_t mread_len,
    uint32_t dread_ptr,
    uint32_t dread_len,
    uint32_t as_hex
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function trace(
    message: string | null,
    data: any,
    hex: boolean | 0 | 1
  ): ErrorCode
```
</TabItem>
</Tabs>

### 参数

<Tabs>
<TabItem label="C">
| Name       | Type      | Description |
| ---------- | --------- | ----------- |
| mread_ptr | uint32_t | 指向日志前置消息的指针，可为 null |
| mread_len | uint32_t | 消息长度，可为 null |
| dread_ptr | uint32_t | 指向要输出缓冲区的指针 |
| dread_len | uint32_t | 缓冲区长度 |
| as_hex    | uint32_t | 为 `1` 时按 hex 输出；为 `0` 时按 utf-8 输出 |
</TabItem>

<TabItem label="JavaScript">
| Name    | Type    | Description |
| ------- | ------- | ----------- |
| message | string  | 输出缓冲区前的日志消息，可为 null |
| data    | any     | 要记录的数据 |
| hex     | boolean | 是否按 HEX 格式输出 |
</TabItem>
</Tabs>

### 返回码

<Tabs>
<TabItem label="C">
| Type     | Description |
| -------- | ----------- |
| int64_t | <p>成功时为 <code>0</code><br /><br />若为负数，则表示错误：<br /><code>OUT_OF_BOUNDS</code><br />- 指针或长度超出 Hook 内存范围。</p> |
</TabItem>

<TabItem label="JavaScript">
| Type      | Description |
| --------- | ----------- |
| ErrorCode | 成功时值为 0；失败时返回错误码。 |
</TabItem>
</Tabs>
