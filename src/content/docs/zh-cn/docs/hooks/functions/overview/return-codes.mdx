Hooks API 中的所有函数都返回带符号整数。

- 负数表示错误
- `0` 或正数表示成功

当返回成功值时，其具体语义由对应 API 决定。例如，有些函数返回写入字节数，有些返回布尔风格的 `1`。

## 通用约定

以下规则适用于所有 Hook API：

- `< 0` 表示错误
- `>= 0` 表示成功

## 说明

部分错误码只会在特定 API 或特定数据类型中出现。  
例如，XFL 相关函数更可能返回与浮点表示有关的错误码。

在阅读单个 API 页面时，应将该页面列出的错误作为第一参考。
