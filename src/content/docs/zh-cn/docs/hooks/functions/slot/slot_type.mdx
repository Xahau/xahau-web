---
title: slot_type
description: 获取槽位中对象的字段代码，并可选返回其他类型信息
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 行为

<Tabs>
<TabItem label="C">
* 定位指定 `slot_no` 指向的对象
* 返回其 `sf` 字段代码；如果使用 `flags`，则返回其他信息
</TabItem>

<TabItem label="JavaScript">
* 获取指定槽位的类型
* 返回错误码或槽位类型
</TabItem>
</Tabs>

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t slot_type (
    uint32_t slot_no,
    uint32_t flags
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_type(slotno: number, flags: number): ErrorCode | number
```
</TabItem>
</Tabs>

### 返回码

<Tabs>
<TabItem label="C">
| Type     | Description |
| -------- | ----------- |
| int64_t | <p>如果 <code>flags</code> 为 <code>0</code>：<br />返回槽位对象的 <code>sf</code> 字段代码<br /><br />如果 <code>flags</code> 为 <code>1</code>：<br />仅当槽位对象是 <code>STI_AMOUNT</code> 且金额类型为 XAH 时返回 <code>1</code><br /><br />若为负数，则表示错误：<br /><code>DOESNT_EXIST</code><br />- 指定 <code>slot_no</code> 不包含对象<br /><br /><code>NOT_AN_AMOUNT</code><br />- <code>flags</code> 为 <code>1</code>，但槽位对象不是 <code>STI_AMOUNT</code></p> |
</TabItem>

<TabItem label="JavaScript">
| Type                | Description |
| ------------------- | ----------- |
| ErrorCode or number | 返回错误码或槽位类型。 |
</TabItem>
</Tabs>
