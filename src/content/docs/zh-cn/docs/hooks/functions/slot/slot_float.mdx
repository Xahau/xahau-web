---
title: slot_float
description: 将指定槽位中的 STI_AMOUNT 解析为 XFL 封装数
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 行为

<Tabs>
<TabItem label="C">
* 解析指定槽位中的 `STI_AMOUNT`
* 返回其对应的 XFL 封装数
</TabItem>

<TabItem label="JavaScript">
* 获取指定槽位中的 bigint 值
* 成功时返回 bigint，失败时返回错误码
</TabItem>
</Tabs>

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t slot_float (
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_float(slotno: number): ErrorCode | bigint
```
</TabItem>
</Tabs>

### 返回码

<Tabs>
<TabItem label="C">
| Type     | Description |
| -------- | ----------- |
| int64_t | <p>XFL 封装数<br /><br />若为负数，则表示错误：<br /><code>DOESNT_EXIST</code><br />- 指定槽位中没有对象，或该槽位无效<br /><br /><code>NOT_AN_AMOUNT</code><br />- 指定槽位中不是 <code>STI_AMOUNT</code> 对象</p> |
</TabItem>

<TabItem label="JavaScript">
| Type                | Description |
| ------------------- | ----------- |
| bigint or ErrorCode | 返回错误码或 bigint 值。 |
</TabItem>
</Tabs>
