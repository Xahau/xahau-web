---
title: slot_set
description: 根据 keylet 定位对象并将其放入槽位
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 行为

<Tabs>
<TabItem label="C">
* 根据 `read_ptr` 中提供的 Keylet 定位对象
* 将定位到的对象放入指定槽位；若槽位为 0，则放入下一个可用槽位
</TabItem>

<TabItem label="JavaScript">
* 为指定槽位设置数据
* 返回错误码或设置结果
</TabItem>
</Tabs>

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t slot_set (
    uint32_t read_ptr,
    uint32_t read_len,
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_set(
    kl: ByteArray | HexString,
    slotno: number
  ): ErrorCode | number
```
</TabItem>
</Tabs>

### 返回码

<Tabs>
<TabItem label="C">
| Type     | Description |
| -------- | ----------- |
| int64_t | <p>对象被放入的槽位编号<br /><br />若为负数，则表示错误：<br /><code>OUT_OF_BOUNDS</code><br />- 指针或长度超出 Hook 内存范围。<br /><br /><code>INVALID_ARGUMENT</code><br />- <code>read_len</code> 必须是 32 或 34，取决于 `read_ptr` 中使用的是交易哈希还是 keylet<br />- 哈希或 keylet 无效<br /><br /><code>DOESNT_EXIST</code><br />- 未找到请求对象</p> |
</TabItem>

<TabItem label="JavaScript">
| Type                | Description |
| ------------------- | ----------- |
| ErrorCode or number | 返回错误码或设置结果。 |
</TabItem>
</Tabs>
