---
title: slot_size
description: 计算槽位中对象的序列化大小
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 行为

<Tabs>
<TabItem label="C">
* 返回指定槽位中对象序列化后占用的字节数
</TabItem>

<TabItem label="JavaScript">
* 获取指定槽位的大小
</TabItem>
</Tabs>

### 定义

<Tabs>
<TabItem label="C">
```c
int64_t slot_size (
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_size(slotno: number): ErrorCode | number
```
</TabItem>
</Tabs>

### 返回码

<Tabs>
<TabItem label="C">
| Type     | Description |
| -------- | ----------- |
| int64_t | <p>对象序列化后占用的字节数<br /><br />若为负数，则表示错误：<br /><code>DOESNT_EXIST</code><br />- 指定槽位中没有对象，或该槽位无效</p> |
</TabItem>

<TabItem label="JavaScript">
| Type                | Description |
| ------------------- | ----------- |
| ErrorCode \| number | 返回错误码或槽位大小。 |
</TabItem>
</Tabs>
