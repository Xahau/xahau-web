title: Help Us
---

Xahau 文档是开源的，也欢迎通过 [GitHub](https://github.com/Xahau/xahau-web) 贡献内容。

文档中出现空白和未覆盖细节是预期中的，因此我们邀请**所有人**一起协作，把它逐步完善到更高标准。

如果你发现有缺失内容，请在仓库中[创建一个 issue](https://github.com/Xahau/xahau-web/issues)，或者直接向主分支提交一个 pull request，附上建议修改或更新后的文档。

如果你想参与贡献，但不知道从哪里开始，可以参考 [GitHub Project board](https://github.com/orgs/Xahau/projects/2/views/1)，里面整理了相关 issue。如果你对 GitHub 还不熟悉，[Community Discord](https://discord.com/invite/UzU58haAn4) 也是一个很适合求助和参与讨论的地方。
