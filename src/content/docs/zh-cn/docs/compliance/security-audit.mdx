作为 XRPL 智能合约基础设施核心组成部分的 Hooks amendment，已经通过 FYEO 的详细安全审计。此次审计检查了 Hooks 框架实现，包括 Hook API、辅助函数以及执行环境。

在没有发现严重安全问题、且所有轻微问题都已解决的前提下，这次审计证明了我们对安全性的重视，也证明了 Hooks 的稳健性。

通过在 XRPL 的 Layer 1 直接启用智能交易逻辑，Hooks 为应用按其独特需求进行定制带来了更高灵活性，也将进一步激发 XRPL 生态内的创新。

开发者如今已经拥有强大的工具，可以在我们的 testnet 上创建并部署自定义逻辑。

### 持续安全性的保障

此次审计的成功完成不仅是一个里程碑，也是 Hooks 网络持续安全与可持续性的基石。

随着已识别漏洞全部被修复，Hooks amendment 将能够在 Xahau 上以更安全的方式推动创新。

</Aside>

迈向一个更光明、更具创新性的 Xahau 未来！
