Hooks 是 Xahau 网络中的一项强大功能，可提供稳健的智能合约能力。它们是专为 XRPL 协议设计的小型高效 WebAssembly 模块，也可以被视为 XRP Ledger 协议上的智能合约。Hooks 可以使用任何能够编译为 WebAssembly 的语言编写，因此能够实现广泛的业务逻辑和智能合约概念。

### 交易类型

Hooks 功能包含两种交易类型：

**SetHook**

这种交易类型用于在账户上设置 Hook。

<LinkCard title="SetHook" href="/docs/protocol-reference/transactions/transaction-types/sethook" />

**Invoke**

这种交易类型用于调用或触发 Hook 的功能。

<LinkCard title="Invoke" href="/docs/protocol-reference/transactions/transaction-types/invoke" />
