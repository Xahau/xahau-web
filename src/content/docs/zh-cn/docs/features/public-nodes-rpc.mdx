title: 公共节点（RPC）
description: >-
  自己运行节点当然很棒；如果想快速开始，也可以使用公开的 Xahau RPC 节点。
---
## Mainnet（网络 21337）

* WebSocket
  * `wss://xahau.network`
  * `wss://xahau.org`（别名，部分广告拦截器会屏蔽 `.network`）
* HTTP POST RPC
  * `https://xahau.network`
  * `https://xahau.org`（别名，部分广告拦截器会屏蔽 `.network`）
* 网络定义（Binary Codec 等）
  * [`https://xahau.network/server_definitions.json`](https://xahau.network/server_definitions.json)
