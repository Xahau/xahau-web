---
title: URITokenBuy
description: URITokenBuy 交易允许用户购买一个 URIToken，并把其所有权转移给买方。
---
\[[Source](https://github.com/Xahau/xahaud/blob/dev/src/ripple/app/tx/impl/URIToken.cpp)]

_(由 [URIToken amendment][] 引入。)_

### 示例

```json
{
    "TransactionType": "URITokenBuy",
    "Account": "rWYkbWkCeg8dP6rXALnjgZSjjLyih5NXm",
    "URITokenID": "C1AE6DDDEEC05CF2978C0BAD6FE27362498DGS691DC749DCDD3B95992978C0BA",
    "Amount": {
      "issuer": "rf1BiGeXwwQoi8Z2ueFYTEXSwuJYfV2Jpn",
      "currency": "USD",
      "value": "100"
    }
}
```

| 字段 | JSON 类型 | [内部类型][] | 说明 |
| --- | --- | --- | --- |
| `Account` | String | AccountID | 买方账户地址。 |
| `URITokenID` | String | Hash256 | 要购买的 URIToken 的唯一标识符。 |
| `Amount` | [Currency Amount](/docs/protocol-reference/data-types/currency-formats/#specifying-currency-amounts) | Amount | 用于支付该 URIToken 的货币金额。 |

### 特殊交易费用

URITokenBuy 交易具有标准交易费用，也就是 Xahau 任意交易所需的最低费用。

### 错误场景

除所有交易都可能出现的错误外，URITokenBuy 还可能返回以下结果码：

| 错误码 | 说明 |
| --- | --- |
| `tecCANT_ACCEPT_OWN_NFTOKEN_OFFER` | 代币所有者自己试图接受该挂单。 |
| `tecDST_TAG_NEEDED` | 目标账户要求 destination tag，但交易未提供。 |
| `tecNO_PERMISSION` | 卖方没有将该代币挂牌出售，或该代币的 `Destination` 不是买方账户。 |
| `temBAD_CURRENCY` | 购买所用货币与挂单货币不匹配。 |
| `tecINSUFFICIENT_PAYMENT` | 购买金额低于挂单金额。 |
| `tecINSUFFICIENT_FUNDS` | 买方资金不足，无法在支付 `Fee` 后完成购买。 |
