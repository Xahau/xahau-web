title: ご協力ください
---

Xahauのドキュメントはオープンソースで、[GitHub](https://github.com/Xahau/xahau-web)から貢献できます。

このドキュメントには不足や未解説の点が多く残っているはずです。そこで、**誰でも** 一緒に品質向上に取り組んでいただければと思います。

不足している点を見つけた場合は、リポジトリで [issueを作成](https://github.com/Xahau/xahau-web/issues)するか、mainブランチに対して提案変更や更新済みドキュメントを含むpull requestをそのまま送ってください。

貢献したいけれど何をすればよいかわからない場合は、issueが整理されている [GitHub Project board](https://github.com/orgs/Xahau/projects/2/views/1)を参照してください。GitHubに不慣れな場合は、[Community Discord](https://discord.com/invite/UzU58haAn4)で助けを得たり参加したりできます。
