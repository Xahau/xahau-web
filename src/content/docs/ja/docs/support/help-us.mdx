title: ご協力ください
---

Xahau のドキュメントはオープンソースで、[GitHub](https://github.com/Xahau/xahau-web) から誰でも貢献できます。

このドキュメントには、まだ抜けや不足している詳細が多くある前提で、**どなたでも** 改善に参加していただけるよう案内しています。

不足している点を見つけた場合は、リポジトリで [issue を作成](https://github.com/Xahau/xahau-web/issues)するか、そのまま main ブランチ向けに pull request を作成し、修正案や更新したドキュメントを提案してください。

貢献したいが何から始めればよいかわからない場合は、issue が整理されている [GitHub Project board](https://github.com/orgs/Xahau/projects/2/views/1) を参照できます。GitHub に不慣れであれば、[Community Discord](https://discord.com/invite/UzU58haAn4) が質問や参加のきっかけとして役立ちます。
