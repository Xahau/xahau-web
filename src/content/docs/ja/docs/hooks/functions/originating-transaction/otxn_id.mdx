import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 発信トランザクションの正規ハッシュを出力バッファに書き込む。
* `flags = 1`で、かつそのトランザクションが`EMIT_FAILURE`トランザクションである場合は、その発行を引き起こした発信トランザクションの正規ハッシュを書き込む。
</TabItem>

<TabItem label="JavaScript">
* 発信トランザクションの正規ハッシュを出力する。
* トランザクションハッシュを数値配列として返し、取得に失敗した場合はErrorCodeを返す。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t otxn_id (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t flags
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_id(flag: number): ErrorCode | ByteArray
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
uint8_t txn_id[32];
int64_t bytes_written = 
    otxn_id(txn_id, 32, 0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const tx_id = otxn_id(0)
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前       | 型      | 説明                                                                                                                                                                                                                 |
| ---------- | --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| write_ptr | uint32_t | ハッシュを格納するバッファへのポインター。                                                                                                                                                                                   |
| write_len | uint32_t | 出力バッファの長さ。少なくとも32バイト必要。                                                                                                                                                                   |
| flags      | uint32_t | <p><code>0</code> の場合:<br />発信トランザクションの正規ハッシュを書き込む。<br /><br /><code>1</code> かつ発信トランザクションが <code>EMIT_FAILURE</code> の場合:<br />発行トランザクションの正規ハッシュを書き込む。</p> |
</TabItem>

<TabItem label="JavaScript">
| 名前  | 型   | 説明                                                                                                                                                                                                                 |
| ----- | ------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| flags | number | <p><code>0</code> の場合:<br />発信トランザクションの正規ハッシュを書き込む。<br /><br /><code>1</code> かつ発信トランザクションが <code>EMIT_FAILURE</code> の場合:<br />発行トランザクションの正規ハッシュを書き込む。</p> |
</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">
| 型     | 説明                                                                                                                                                                                                                                                |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>書き込まれたバイト数。<br /><br />負の場合はエラー:<br /><code>OUT_OF_BOUNDS</code><br />-指定されたポインター/長さがHookメモリの範囲外である。<br /><br /><code>TOO_SMALL</code><br />-出力バッファがシリアライズ済みオブジェクトを格納するのに十分な大きさではなかった。</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                   | 説明                                                                                  |
| ---------------------- | -------------------------------------------------------------------------------------------- |
| ErrorCode \| ByteArray | トランザクションハッシュを数値配列として返し、取得に失敗した場合はErrorCodeを返す。 |
</TabItem>
</Tabs>
