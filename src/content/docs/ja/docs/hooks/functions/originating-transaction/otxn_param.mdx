---
title: otxn_param
description: 名前付き Invoke トランザクション parameter の値を取得します
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 発信元トランザクション上で `read_ptr` により指定された名前の parameter 値を検索します。
* parameter の値を `write_ptr` に書き込みます。
</TabItem>

<TabItem label="JavaScript">
* 発信元トランザクション上で指定された名前の parameter 値を検索します。
* 指定した parameter の値を返します。取得に失敗した場合は ErrorCode を返します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t otxn_param (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t read_ptr,
    uint32_t read_len
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_param(name: ByteArray | HexString): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
uint8_t pname[] = {0xCAU, 0xFEU};
uint8_t pvalue[32];
int64_t value_len = 
    otxn_param(pvalue, 32, pname, 2);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const pname = [0xCA, 0xFE]
const pvalue = otxn_param(pname)
```
</TabItem>
</Tabs>

### パラメーター

<Tabs>
<TabItem label="C">
| Name      | Type     | Description                                                    |
| --------- | -------- | -------------------------------------------------------------- |
| write_ptr | uint32_t | 出力を格納するのに十分なサイズのバッファへのポインタです。少なくとも 32 バイト必要です。 |
| write_len | uint32_t | 出力バッファの長さです。                                       |
| read_ptr  | uint32_t | parameter 名を含むバッファへのポインタです。                   |
| read_len  | uint32_t | parameter 名の長さです。                                       |
</TabItem>

<TabItem label="JavaScript">
| Name | Type                   | Description                                           |
| ---- | ---------------------- | ----------------------------------------------------- |
| name | ByteArray \| HexString | 検索対象 parameter 名です。ByteArray または文字列で指定します。 |
</TabItem>
</Tabs>

<Tabs>

</TabItem>

<TabItem label="JavaScript">

| ---------------------- | ----------------------------------------------------------- |

</TabItem>
</Tabs>
