import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* 発信トランザクションのトランザクションタイプを返す

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t otxn_type (
    void
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_type(): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t tt = 
  otxn_type();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const tt = txn_type()
```
</TabItem>
</Tabs>

### パラメーター

なし

### 戻り値

<Tabs>
<TabItem label="C">
| 型     | 説明                                                                                                                         |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | 発信トランザクションのトランザクションタイプ。記述時点で既知のトランザクションタイプ一覧は下の表を参照。 |
</TabItem>

<TabItem label="JavaScript">
| 型                | 説明                                                                       |
| ------------------- | --------------------------------------------------------------------------------- |
| ErrorCode \| number | トランザクションタイプを数値として返す。取得に失敗した場合はErrorCodeを返す。 |
</TabItem>
</Tabs>
