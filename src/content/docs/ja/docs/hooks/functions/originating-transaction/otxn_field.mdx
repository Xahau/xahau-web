description: 発信トランザクションのフィールドをシリアライズして出力する

import { Tabs, TabItem, Aside } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 発信トランザクション内で指定された `sf` フィールドを検索する
* フィールドのシリアライズ済み表現を出力バッファに書き込む
</TabItem>

<TabItem label="JavaScript">
* 発信トランザクション内の特定のフィールドの値を取得する。
* 指定したフィールドの値を数値配列として返し、検索に失敗した場合はErrorCodeを返す。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t otxn_field (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t field_id
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_field(field_id: number): ErrorCode | ByteArray
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t account_field_len = 
    otxn_field(account_field, 20, sfAccount);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
const account_field = otxn_field(sfAccount)
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前       | 型      | 説明                                                                                                                                                                                                                                                                                                                                                                                                |
| ---------- | --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| write_ptr | uint32_t | 出力を格納するのに適したサイズのバッファへのポインター。                                                                                                                                                                                                                                                                                                                                                |
| write_len | uint32_t | 出力バッファの長さ。                                                                                                                                                                                                                                                                                                                                                                               |
| field_id  | uint32_t | <p>検索対象フィールドの <code>sf</code> コード。<br /><br />手動で計算するには、シリアライズされた <code>type</code> をuint32_tの上位16ビットにシフトし、<code>field</code> を下位16ビットに配置する。<br /><br />例:<br /><code>sfEmitNonce</code> は <code>type</code> が5、<code>field</code> が11なので、その値は <code>0x050BU</code> になる。</p> |

<Aside type="caution" title="重要">
フィールドコードは出力バッファに_書き込まれず_、フィールドの_ペイロード_のみが書き込まれる。

記述時点では、Hooks Public Testnetにおいて `sfAccount` のような `STI_ACCOUNT` フィールドは、先頭の可変長バイトなしで返される。
</Aside>
</TabItem>

<TabItem label="JavaScript">
| 名前      | 型   | 説明                                                                                         |
| --------- | ------ | --------------------------------------------------------------------------------------------------- |
| field_id | number | <p></p><p>指定したフィールドの値を数値配列として返す。検索に失敗した場合はErrorCodeを返す。</p> |
</TabItem>
</Tabs>

### 戻り値

<Tabs>
<TabItem label="C">
| 型     | 説明                                                                                                                                                                                                                                                                                                                                                                                                                            |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>書き込まれたバイト数。<br /><br />負の場合はエラー:<br /><code>OUT_OF_BOUNDS</code><br />-指定されたポインター/長さがHookメモリの範囲外である。<br /><br /><code>TOO_SMALL</code><br />-出力バッファがシリアライズされたフィールドを格納するのに十分な大きさではなかった。<br /><br /><code>INVALID_FIELD</code><br />- <code>sf</code> のfield_idが無効である。<br /><br /><code>DOESNT_EXIST</code><br />-発信トランザクション内にそのフィールドが見つからなかった。</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                   | 説明                                                                                                         |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------- |
| ByteArray or ErrorCode | <p></p><p>指定したフィールドの値を数値配列として返す。検索に失敗した場合はErrorCodeを返す。</p> |
</TabItem>
</Tabs>
