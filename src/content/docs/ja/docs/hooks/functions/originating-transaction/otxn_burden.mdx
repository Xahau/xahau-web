---
title: otxn_burden
description: 発信トランザクションのバーデンを取得する
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 発信トランザクションのバーデンを返す。バーデンフィールドが存在しない場合は`1`を返す。
</TabItem>

<TabItem label="JavaScript">
* 発信トランザクションのバーデンを取得する。
* バーデンを数値として返し、取得に失敗した場合はErrorCodeを返す。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t otxn_burden (
    void
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function otxn_burden(): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t burden = 
  otxn_burden();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const burden = otxn_burden()
```
</TabItem>
</Tabs>



### パラメーター

なし

### 戻り値

<Tabs>
<TabItem label="C">
| 型     | 説明                                                                                                |
| -------- | ---------------------------------------------------------------------------------------------------------- |
| int64_t | 発信トランザクションのバーデン。発信トランザクションにバーデンが存在しない場合は`1`を返す。 |
</TabItem>

<TabItem label="JavaScript">
| 型                | 説明                                                             |
| ------------------- | ----------------------------------------------------------------------- |
| number or ErrorCode | バーデンを数値として返し、取得に失敗した場合はErrorCodeを返す。 |
</TabItem>
</Tabs>
