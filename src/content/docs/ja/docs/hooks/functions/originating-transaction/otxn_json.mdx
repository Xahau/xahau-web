import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="JavaScript">
* 発信トランザクションをJSON形式で出力する。
* 発信トランザクションをJSONオブジェクトまたはTransactionとして返し、取得に失敗した場合はErrorCodeを返す。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="JavaScript">
```javascript
function otxn_json(): ErrorCode | Record<string, any> | Transaction
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="JavaScript">
```javascript
const tx_json = otxn_json()
```
</TabItem>
</Tabs>



### パラメーター

パラメーターはありません

### 戻り値

<Tabs>
<TabItem label="JavaScript">
| 型                                             | 説明                                                                                                  |
| ------------------------------------------------ | ------------------------------------------------------------------------------------------------------------ |
| ErrorCode \| Record\<string, any> \| Transaction | 発信トランザクションをJSONオブジェクトまたはTransactionとして返し、取得に失敗した場合はErrorCodeを返す。 |
</TabItem>
</Tabs>
