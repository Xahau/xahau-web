---
title: meta_slot
description: 発信トランザクションのメタデータをSlotに読み込む
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* Hookが [弱実行](/ja/docs/hooks/concepts/weak-and-strong)されている場合、発信トランザクションのメタデータを指定したSlotに配置する。Slotが指定されていない場合は、新しいSlotに配置する。

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t meta_slot (
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function meta_slot(slotno: number): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t meta_slot_no = 
		meta_slot(0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const meta_slot_no = meta_slot(0)
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前     | 型      | 説明                                                                   |
| -------- | --------- | ----------------------------------------------------------------------------- |
| slot_no | uint32_t | 配置先のSlot番号。次に利用可能なSlotを選ぶ場合は `0`。 |
</TabItem>

<TabItem label="JavaScript">
| 名前   | 型   | 説明                               |
| ------ | ------ | ----------------------------------------- |
| slotno | number | メタデータを取得するSlot番号。 |
</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">
| 型     | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>otxnが配置されたSlot。<br /><br /><code>INVALID_ARGUMENT</code><br />-指定されたSlot番号が利用可能な最大Slot番号を超えている。<br /><br /><code>NO_FREE_SLOTS</code><br />-発信トランザクションを配置できる空きSlotがなく、要求を受け付けられなかった。<br /><br /><code>PREREQUISITE_NOT_MET</code><br />- Hookが <a href="/ja/docs/hooks/concepts/weak-and-strong">強実行</a> されているため、トランザクションメタデータを利用できない。</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                | 説明                                   |
| ------------------- | --------------------------------------------- |
| ErrorCode \| number | エラーコードまたはSlotのメタデータを返す。 |
</TabItem>
</Tabs>
