### 動作

* `hook` は、Hook を発火させるために `xahaud` から呼び出されるユーザー定義関数です。
* `hook` 関数は、発信元トランザクションを通過させるか拒否するために `accept` または `rollback` を呼び出します。
* 実行が関数の末尾まで到達した場合、それは暗黙的に `accept` になります。
