### 動作

* `hook` は、Hookを実行するために `xahaud` によって呼び出されるユーザー定義関数です。
* `hook` 関数は `accept` または `reject` を呼び出して、発信トランザクションを受理または拒否します。
* 実行が関数の末尾に到達した場合、暗黙に `accept` になります。
