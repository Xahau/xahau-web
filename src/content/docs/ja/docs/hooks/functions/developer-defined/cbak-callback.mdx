### 動作

* `cbak` は、以前に emit されたトランザクションの状態を Hook に通知するために `xahaud` から呼び出されるユーザー定義関数です。
* state の変更や追加の emit 呼び出しは cbak から行えますが、トランザクションを `rollback` することはできません。
* cbak の実行時には、このコールバックが対応する emit 済みトランザクションが現在の発信元トランザクションになります。

### 定義

### パラメーター

<Tabs>
<TabItem label="C">
| Name     | Type     | Description                                                                                                                                                                                                 |
| -------- | -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| reserved | uint32_t | <p><code>0</code> の場合:<br />- このコールバックに対応する emit 済みトランザクションは、レジャーに正常に受け入れられました。<br /><br /><code>1</code> の場合<br />- 対応する emit 済みトランザクションは、有効期限切れまでにレジャーへ正常に受け入れられませんでした。</p> |
</TabItem>

<TabItem label="JavaScript">

</TabItem>
