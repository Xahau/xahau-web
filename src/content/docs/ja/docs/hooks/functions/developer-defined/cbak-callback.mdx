### 概念

<LinkButton href="/ja/docs/hooks/concepts/compiling-hooks">Hooksのコンパイル</LinkButton>

### 動作

* `cbak`は、以前に発行したトランザクションの状態をHookに通知するために`xahaud`によって呼び出されるユーザー定義関数です。
* `cbak`からは状態変更や追加のemit呼び出しが可能ですが、トランザクションを`rollback`することはできません。
* `cbak`が実行されると、コールバックが関連付けられている発行トランザクションが発信トランザクションになります。

### 定義

### パラメーター

<Tabs>
<TabItem label="C">
| 名前     | 型        | 説明                                                                                                                                                                                                                                                                         |
| -------- | --------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| reserved | uint32_t | <p><code>0</code> の場合:<br />-このコールバックが関連付けられている発行トランザクションは、レジャーに正常に受け入れられました。<br /><br /><code>1</code> の場合:<br />-このコールバックが関連付けられている発行トランザクションは、有効期限までにレジャーに正常に受け入れられませんでした。</p> |
</TabItem>

<TabItem label="JavaScript">
| 名前     | 型      | 説明                                                                                                                                                                                                                                                                         |

| reserved | number  | <p><code>0</code> の場合:<br />-このコールバックが関連付けられている発行トランザクションは、レジャーに正常に受け入れられました。<br /><br /><code>1</code> の場合:<br />-このコールバックが関連付けられている発行トランザクションは、有効期限までにレジャーに正常に受け入れられませんでした。</p> |
