### 戻りコードの設計

Web assembly は例外（trap）を許可していますが、この言語機能は Hooks では使用されません。代わりに、どの Hook API からも戻る方法は 1 つしかありません（すべての Hook API が `noexcept` だと考えてよいでしょう）。

効率的なエラーハンドリングのため、次のルールがあります。

1. すべての Hook API 関数は _符号付き_ 整数を返します。
2. 負の戻りコードはすべてエラーです。
3. `0` 以上の戻りコードはその関数固有の出力で、通常は読み書きしたバイト数ですが、常にそうとは限りません。

### エラーコード

エラーコードはすべての Hook API で共通であり、以下の表に示されています。

| Name                           | Value  | Description                                                                                                                                                        |
| ------------------------------ | ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| SUCCESS                        | >= 0   | 0 以上の戻りコードは常に成功を示し、通常は対象 API に応じて書き込まれたバイト数や実行されたイベント数を表します。                                               |
| OUT_OF_BOUNDS                  | -1     | パラメーターとして指定されたポインタまたはバッファ長が、Hook に許可されたメモリ領域の外を指していました。                                                      |
| INTERNAL_ERROR                 | -2     | 内部不変条件違反用に予約されています。通常は入力とは無関係です。issue として報告してください。                                                                   |
| TOO_BIG                        | -3     | 許可された領域より大きなパラメーターまたは値を設定しようとしました。                                                                                             |
| TOO_SMALL                      | -4     | 指定された `write_len` が小さすぎたため、API は `write_ptr` に出力を書き込めませんでした。                                                                      |
| DOESNT_EXIST                   | -5     | 要求されたオブジェクトまたは項目が見つかりませんでした。                                                                                                         |
| NO_FREE_SLOTS                  | -6     | Hook が項目をスロットに割り当てようとしましたが、空きスロットがありませんでした。既存スロットを再利用するようにしてください。最大スロット数は 255 です。        |
| INVALID_ARGUMENT               | -7     | 1 つ以上のパラメーターが、その API の仕様に照らして無効でした。                                                                                                 |
| ALREADY_SET                    | -8     | 一部 API では 1 回の実行につき 1 度だけ設定できるパラメーターがあります。2 回目の設定を試みるとこのエラーになります。                                          |
| PREREQUISITE_NOT_MET           | -9     | この API を呼び出す前に Hook が実行すべき前提条件が満たされていませんでした。API のドキュメントを確認してください。                                             |
| FEE_TOO_LARGE                  | -10    | 手数料計算中に異常に大きな手数料が算出された場合に返されます。                                                                                                   |
| EMISSION_FAILURE               | -11    | 何らかの理由で `emit()` による TXN の送信に失敗しました。発信元 TXN を送信している rippled の `trace` ログを確認してください。                                  |
| TOO_MANY_NONCES                | -12    | Hook は 1 回の実行で `nonce()` を最大 256 回までしか使用できません。それを超えるとこのエラーコードになります。                                                 |
| TOO_MANY_EMITTED_TXN           | -13    | <p>Hook は事前に、いくつの TXN を <code>emit()</code> するか宣言しなければなりません。<br />それより少なく emit するのは許可されます。<br />それより多いとこのエラーが返されます。</p> |
| NOT_IMPLEMENTED                | -14    | Hooks の開発中には、その API の一部または全部が未実装の場合にこの値が返されることがあります。                                                                    |
| INVALID_ACCOUNT                | -15    | 20 バイトの Account ID を受け取る API で、その Account ID が何らかの理由で無効と判断された場合に返されます。                                                    |
| GUARD_VIOLATION                | -16    | Hook 内のすべてのループは、ループ冒頭の最初の非自明な命令として、分岐命令より前に最大反復回数を宣言しなければなりません。この約束に違反すると即座に終了します。 |
| INVALID_FIELD                  | -17    | 要求されたシリアライズ済みフィールドが、指定されたオブジェクト内に見つかりませんでした。                                                                        |
| PARSE_ERROR                    | -18    | シリアライズ済みコンテンツの解析中にエラーが発生しました。通常は不正なシリアライズ形式を示します。                                                              |
| RC_ROLLBACK                    | -19    | rollback イベントを内部的に通知するために使用されます。                                                                                                           |
| RC_ACCEPT                      | -20    | accept イベントを内部的に通知するために使用されます。                                                                                                             |
| NO_SUCH_KEYLET                 | -21    | 指定された keylet が見つからないか、keylet が無効です。                                                                                                           |
| NOT_AN_ARRAY                   | -22    | API は解析対象オブジェクトが STArray であると仮定するよう求められましたが、実際にはそうではありませんでした。                                                   |
| NOT_AN_OBJECT                  | -23    | API は解析対象オブジェクトが STObject であると仮定するよう求められましたが、実際にはそうではありませんでした。                                                  |
| INVALID_FLOAT                  | -10024 | 浮動小数点演算の結果が Not-A-Number になったか、API 呼び出しで XFL の表現可能範囲外の浮動小数点数を指定しようとしました。                                      |
| DIVISION_BY_ZERO               | -25    | API 呼び出しが 0 除算になるため、API は早期終了しました。                                                                                                        |

| EXPONENT_OVERSIZED             | -28    | XFL を作成する際、exponent は 80 を超えてはいけません。                                                                                                          |
| EXPONENT_UNDERSIZED            | -29    | XFL を作成する際、exponent は -96 未満であってはいけません。                                                                                                     |
| OVERFLOW                       | -30    | XFL に対する浮動小数点演算で、XFL 形式が表現できる範囲を超える値が生じました。                                                                                  |
| NOT_IOU_AMOUNT                 | -31    | API は STAmount が IOU であると仮定しましたが、実際には XRP でした。                                                                                            |
| NOT_AN_AMOUNT                  | -32    | API は STObject が STAmount であると仮定しましたが、実際にはそうではありませんでした。                                                                          |
| CANT_RETURN_NEGATIVE           | -33    | API は負の整数を返すはずでしたが、負の整数はエラーコード用に予約されているため返せませんでした。                                                                |
| NOT_AUTHORIZED                 | -34    | Hook は foreign state を設定しようとしましたが、権限がありませんでした（grant がない、または無効でした）。                                                     |

| INVALID_KEY                    | -41    | 指定された公開鍵が無効でした。                                                                                                                                   |
| NOT_A_STRING                   | -42    | バッファにヌル終端文字列が含まれていませんでした。                                                                                                               |
| MEM_OVERLAP                    | -43    | 書き込みポインタが読み取りポインタと重なるバッファを指しています。                                                                                               |
| TOO_MANY_STATE_MODIFICATIONS   | -44    | 結合された Hook チェーン全体で変更された state エントリが 5000 を超えました。                                                                                  |
| TOO_MANY_NAMESPACES            | -45    | このアカウント上の namespace 数が 256 を超えました。                                                                                                            |
