import { Tabs, TabItem, LinkButton } from '@astrojs/starlight/components';

### 概念

<LinkButton href="/docs/hooks/concepts/hook-fees">Hook Fees</LinkButton>

### 動作

* 現在のレジャーから手数料ベースを返します。

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t fee_base();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function fee_base(): number
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
int64_t fee =
    fee_base();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const fee = fee_base()
```
</TabItem>
</Tabs>

### パラメーター

この API はパラメーターを取りません。
