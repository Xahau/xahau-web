---
title: ledger_nonce
description: 発行トランザクションで使用する32バイトのnonceを生成します
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 32バイトのランダム値をwrite_ptrに書き込みます
</TabItem>

<TabItem label="JavaScript">
* 現在のレジャーのnonceを取得します。
</TabItem>
</Tabs>



### 定義

C

<Tabs>
<TabItem label="C">
```c
int64_t ledger_nonce (
    uint32_t write_ptr,
    uint32_t write_len
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function ledger_nonce(): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
uint8_t n[32];
int64_t bytes_written = 
    ledger_nonce(n, 32);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
const nonce = ledger_nonce()
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前       | 型        | 説明                                                                                     |
| ---------- | --------- | ---------------------------------------------------------------------------------------- |
| write_ptr | uint32_t | 出力を保存するのに十分なサイズのバッファへのポインター。少なくとも32バイト必要です。     |
| write_len | uint32_t | 出力バッファの長さ。                                                                     |


</TabItem>

<TabItem label="JavaScript">

</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">


| 型                     | 説明                                                                                                                                  |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| ErrorCode \| ByteArray | エラーが発生した場合はエラーコードを返し、それ以外の場合は現在のレジャーのnonceを表す配列を返します。Hookメモリ外が指定されています。 |
</TabItem>

<TabItem label="JavaScript">


| 型       | 説明                                                                                                                                                   |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| int64_t | <p>書き込まれたバイト数<br /><br />負の場合はエラー:<br /><code>OUT_OF_BOUNDS</code><br />-ポインター/長さがHookメモリ外を指定しています。</p> |
</TabItem>
</Tabs>
