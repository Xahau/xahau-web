### 動作

* 最後にクローズされたレジャーの Xahau Timestamp を返します。

<Aside type="tip">
Xahau のタイムスタンプは Unix タイムスタンプと同一ですが、`-946684800` のオフセットがあります。

対応する Unix タイムスタンプは `ledger_last_time() + 946684800;` です。
</Aside>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t ledger_last_time();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function ledger_last_time(): ErrorCode | number
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
int64_t ts =
    ledger_last_time();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const ts = ledger_last_time()
```
</TabItem>
</Tabs>

### パラメーター

この API はパラメーターを取りません。
