### 動作

* 現在のレジャーからシーケンス番号を返します。

### 定義

C

<Tabs>
<TabItem label="C">
```c
int64_t ledger_seq();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function ledger_seq(): ErrorCode | number
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
int64_t seq =
    ledger_seq();
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const seq = ledger_seq()
```
</TabItem>
</Tabs>

### パラメーター

この API はパラメーターを取りません。
