---
title: hook_again
description: 現在実行中のHookがHookチェーン内で占める位置を返す
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* Hookが強い実行中であれば、このチェーン内のこのHookを [Again As Weak Execution](/ja/docs/hooks/concepts/weak-and-strong)にするようフラグを立てる。
* 発信トランザクションが正常に適用された場合、このHookは2回目の弱い実行として再度呼び出される。

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t hook_again(void);
```
</TabItem>
<TabItem label="JavaScript">
```javascript
function hook_again(): ErrorCode | number
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
int64_t result = 
  	hook_again();
```
</TabItem>
<TabItem label="JavaScript">
```javascript
const result = hook_again()
```
</TabItem>
</Tabs>

### パラメーター

このAPIには引数はありません
