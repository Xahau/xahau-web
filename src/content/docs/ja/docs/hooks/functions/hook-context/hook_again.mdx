---
title: hook_again
description: 現在実行中の Hook を再度 Weak Execution するよう予約します
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* Hook が strong 実行中であれば、チェーン内のこの Hook を [Again As Weak Execution](/docs/hooks/concepts/weak-and-strong) 用にマークします。
* 発信元トランザクションが正常に適用された場合、この Hook は 2 回目の Weak Execution で再度呼び出されます。

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t hook_again(void);
```
</TabItem>
<TabItem label="JavaScript">
```javascript
function hook_again(): ErrorCode | number
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
int64_t result = 
  	hook_again();
```
</TabItem>
<TabItem label="JavaScript">
```javascript
const result = hook_again()
```
</TabItem>
</Tabs>

### パラメーター

この API にパラメーターはありません
