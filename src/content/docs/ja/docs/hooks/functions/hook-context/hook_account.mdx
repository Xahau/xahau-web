---
title: hook_account
description: Hook が実行されている 20 バイトの Account ID を取得します
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 20 バイトの Account ID を `write_ptr` に書き込みます。
</TabItem>

<TabItem label="JavaScript">
* Hook が実行されている 20 バイトの Account ID を取得します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t hook_account (
    uint32_t write_ptr,
    uint32_t write_len
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function hook_account(): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
uint8_t hook_acc_id[20];
int64_t bytes_written = 
    hook_account(hook_acc_id, 20);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const hook_acc_id = hook_account()
```
</TabItem>
</Tabs>

### パラメーター

<Tabs>
<TabItem label="C">
| Name      | Type     | Description                                                    |
| --------- | -------- | -------------------------------------------------------------- |
| write_ptr | uint32_t | 出力を格納するのに十分なサイズのバッファへのポインタです。少なくとも 20 バイト必要です。 |
| write_len | uint32_t | 出力バッファの長さです。                                       |
</TabItem>

<TabItem label="JavaScript">
パラメーターはありません
</TabItem>
</Tabs>

### 戻りコード

<Tabs>
<TabItem label="C">
| Type    | Description                                                                                                                                  |
| ------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>書き込まれたバイト数を返します。<br /><br />負の場合はエラーです。<br /><code>OUT_OF_BOUNDS</code><br />- ポインタまたは長さが Hook メモリ外を指しています。</p> |
</TabItem>

<TabItem label="JavaScript">
| Type                   | Description                                                          |
| ---------------------- | -------------------------------------------------------------------- |
| ErrorCode or ByteArray | Hook が実行されている Account ID を返します。取得に失敗した場合はエラーコードを返します。 |
</TabItem>
</Tabs>
