---
title: hook_account
description: Hookが実行されている20バイトのAccount IDを取得する
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 20バイトのAccount IDを`write_ptr`に書き込む
</TabItem>

<TabItem label="JavaScript">
* Hookが実行されている20バイトのAccount IDを取得する。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t hook_account (
    uint32_t write_ptr,
    uint32_t write_len
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function hook_account(): ErrorCode | ByteArray
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
uint8_t hook_acc_id[20];
int64_t bytes_written = 
    hook_account(hook_acc_id, 20);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
const hook_acc_id =hook_account()
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前       | 型        | 説明                                                                                     |
| ---------- | --------- | ---------------------------------------------------------------------------------------- |
| write_ptr | uint32_t | 出力を格納するのに適したサイズのバッファへのポインター。少なくとも20バイト必要。      |
| write_len | uint32_t | 出力バッファの長さ。                                                                     |
</TabItem>

<TabItem label="JavaScript">
引数はありません
</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">
| 型      | 説明                                                                                                                                           |
| ------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>書き込まれたバイト数<br /><br />負の場合はエラー:<br /><code>OUT_OF_BOUNDS</code><br />-指定したポインター/長さがHookメモリの外側にある。</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                     | 説明                                                                                           |
| ---------------------- | ---------------------------------------------------------------------------------------------- |
| ErrorCode or ByteArray | Hookが実行されているAccount IDを返す。取得に失敗した場合はエラーコードを返す。            |
</TabItem>
</Tabs>
