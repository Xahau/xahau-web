import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* 整数を、メッセージがあればそれと共にトレースログへ書き込む

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t trace_num (
    uint32_t mread_ptr,
    uint32_t mread_len,
    int64_t number 
);
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
#define SBUF(str) (uint32_t)(str), sizeof(str)
trace_num(SBUF("This is an integer"), 10);
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前      | 型        | 説明                                                                                               |
| ---------- | --------- | --------------------------------------------------------------------------------------------------- |
| rmead_ptr | uint32_t | Slot内に見つかったhexエンコード済みのシリアライズオブジェクトの前に出力するメッセージへのポインター。_null可。_ |
| mread_len | uint32_t | メッセージの長さ。_null可。_                                                                     |
| number     | int64_t  | 数値。                                                                                             |
</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">
| 型      | 説明                                                                                                                                             |
| ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>成功時は <code>0</code><br /><br />負の場合はエラー:<br /><code>OUT_OF_BOUNDS</code><br />-指定したポインター/長さがHookメモリの外側にある。</p> |
</TabItem>
</Tabs>
