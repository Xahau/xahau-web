title: trace_num
description: 整数を Xahaud の trace ログに書き込みます
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* 必要であればメッセージとともに、整数を trace ログへ書き込みます。

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t trace_num (
    uint32_t mread_ptr,
    uint32_t mread_len,
    int64_t number 
);
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
#define SBUF(str) (uint32_t)(str), sizeof(str)
trace_num(SBUF("This is an integer"), 10);
```
</TabItem>
</Tabs>

### パラメーター

<Tabs>
<TabItem label="C">
| Name      | Type     | Description                                                                                                  |
| --------- | -------- | ------------------------------------------------------------------------------------------------------------ |
| mread_ptr | uint32_t | 数値の前に出力するメッセージへのポインタです。 _null でも構いません。_ |
| mread_len | uint32_t | メッセージの長さです。 _null でも構いません。_                                                                |
| number    | int64_t  | 数値です。                                                                                                   |
</TabItem>
</Tabs>
