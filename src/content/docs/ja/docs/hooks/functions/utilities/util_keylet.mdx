### 概念

<LinkButton href="/ja/docs/hooks/concepts/slots-and-keylets">SlotとKeylet</LinkButton>

<Aside type="tip">
このユーティリティ関数でサポートされているKeyletタイプはすべてではありません。別のKeyletタイプが必要な場合は、[util_sha512h](https://xrpl-hooks.readme.io/reference/util_sha512h)を使い、必要なフィールドを [ここ](https://github.com/Xahau/xahaud/blob/dev/src/ripple/protocol/impl/Indexes.cpp)で確認することで自分で導出できます。別のKeyletツールが [役立つかもしれません。](https://richardah.github.io/xrpl-keylet-tools/)
</Aside>

### 動作

* 種類に応じて、`a` から `f` のパラメーターに従い、指定した `keylet_type` のKeyletを計算する
* シリアライズ済みの34バイトのKeyletを `write_ptr` に書き込む

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t util_keylet (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t keylet_type,
    uint32_t a,
    uint32_t b,
    uint32_t c,
    uint32_t d,
    uint32_t e,
    uint32_t f
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function util_keylet(
    keylet_type: number,
    accountid: ByteArray | HexString,
    statekey: ByteArray | HexString,
    namespace: ByteArray | HexString
  ): ErrorCode | ByteArray
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
uint8_t keylet[34];
if (util_keylet(keylet, 34, KEYLET_LINE,
              hook_accid, 20,
              account_field, 20,
              currency_code, 20) != 34)
	rollback("Keylet Failed.", 14, 1);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
if (util_keylet(
        KEYLET_LINE, hook_accid, account_field, currency_code
        ).length < 34) 
    rollback("Keylet Failed.", 1)
```
</TabItem>
</Tabs>
