---
title: util_sha512h
description: データに対してsha512-halfを計算する
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* `read_ptr` が指すデータに対して `SHA512` ハッシュを計算する
* ハッシュの前半を `write_ptr` に書き込む
</TabItem>

<TabItem label="JavaScript">
* `data` が指すデータに対して `SHA512` ハッシュを計算する
* ハッシュの前半を返す
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t util_sha512h (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t read_ptr,
    uint32_t read_len
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function util_sha512h(data: ByteArray | HexString): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### 例

C

<Tabs>
<TabItem label="C">
```c
uint8_t hash_out[32];
if (util_sha512h(hash_out, 32, data_in_ptr, data_in_len) < 0)
	rollback("Could not generate Hash", 23, 1);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
if (util_sha512h(data) < 0)
    rollback("Could not generate Hash", 1)
```
</TabItem>
</Tabs>
