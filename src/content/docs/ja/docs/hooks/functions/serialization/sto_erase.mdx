import { Tabs, TabItem, LinkButton, Aside } from '@astrojs/starlight/components';

### コンセプト

<LinkButton href="/ja/docs/hooks/concepts/serialized-objects">シリアライズ済みオブジェクト</LinkButton>

### 動作

<Tabs>
<TabItem label="C">
* `read_ptr`が指すSTObjectを解析します
* 元のオブジェクトに`field_id`が存在する場合は、それを除いた新しいSTObjectを`write_ptr`に書き込みます。
</TabItem>

<TabItem label="JavaScript">
* フィールドを削除する対象のSTOオブジェクト（バイナリエンコードされたレジャーデータ）を参照します。
* 削除するフィールドのIDを参照します。
* バイナリエンコードされたレジャーデータ形式の更新済みSTOオブジェクトを返します。操作に失敗した場合はエラーコードを返します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t sto_erase (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t read_ptr,
    uint32_t read_len,
    uint32_t field_id
);
```

<Aside type="caution" title="フィールドIDのエンコード">
`sto_` APIは、次の形式でエンコードされた`field_id`パラメーターを受け取ります: `(type << 16U) + field`
したがって、typeが1、fieldが2の場合は`0x10002U`になります。
</Aside>
</TabItem>

<TabItem label="JavaScript">
```javascript
function sto_erase(
    sto: ByteArray | HexString,
    field_id: number
  ): ErrorCode | ByteArray
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t result = 
  sto_erase(tx_out, sizeof(tx_out),
            tx_in, tx_len, sfSigners);

if (tx_len <= 0)
    rollback("Erasing failed.", 15, 1);
```

<Aside type="tip" title="emplaceとの同等性">
`sto_erase`は、`field_ptr, field_len`パラメーターに`0,0`を指定した`sto_emplace`と同じです。
</Aside>
</TabItem>

<TabItem label="JavaScript">
```javascript
const tx_out = sto_erase(tx_in, sfSigners)

if (typeof tx_out === 'number')
    rollback("Erasing failed.", 1)
```
</TabItem>
</Tabs>

### パラメーター

<Tabs>
<TabItem label="C">
| 名前       | 型        | 説明                                         |
| ---------- | --------- | -------------------------------------------- |
| write_ptr | uint32_t | 変更後のSTObjectを書き込むバッファ           |
| write_len | uint32_t | 出力バッファの長さ                           |
| read_ptr  | uint32_t | ソースSTObjectを読み取るバッファ             |
| read_len  | uint32_t | ソースオブジェクトの長さ                     |
| field_id  | uint32_t | 削除する`sf`コード（位置）                   |


</TabItem>

<TabItem label="JavaScript">
| 名前      | 型                     | 説明                                                                              |
| --------- | ---------------------- | --------------------------------------------------------------------------------- |
| sto       | ByteArray \| HexString | フィールドを削除する対象のSTOオブジェクト（バイナリエンコードされたレジャーデータ）。 |
| field_id | number                 | 削除するフィールドのID。                                                          |
</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">
| 型       | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p><code>write_ptr</code>に書き込まれたバイト数<br /><br />負の場合はエラー:<br /><code>OUT_OF_BOUNDS</code><br />-ポインター/長さがHookメモリ外を指定しています。<br /><br /><code>TOO_SMALL</code><br />-出力バッファは少なくともソースオブジェクトと同じ大きさである必要があります。<br /><br /><code>TOO_BIG</code><br />-削除しようとしているフィールドが大きすぎます<br /><br /><code>PARSE_ERROR</code><br />-指定されたSTObjectの形式が不正であるか、STObjectではありません。<br /><br /><code>DOESNT_EXIST</code><br />-指定された<code>field_id</code>がSTObjectに存在しません。</p> |


</TabItem>

| ---------------------- | ----------------------------------------------------------------------------------------------------- |
| ErrorCode \| ByteArray | バイナリエンコードされたレジャーデータ形式の更新済みSTOオブジェクト。操作に失敗した場合はエラーコード。 |
