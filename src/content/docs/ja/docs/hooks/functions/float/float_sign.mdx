---
title: float_sign
description: XFL格納数の符号を取得します
---
import { Tabs, TabItem, LinkButton, Aside } from '@astrojs/starlight/components';

### コンセプト

<LinkButton href="/ja/docs/hooks/concepts/floating-point-numbers-xfl">浮動小数点数 (XFL)</LinkButton>

### 動作

<Tabs>
<TabItem label="C">
* XFLが負の場合は`1`を返し、それ以外の場合は0を返します
</TabItem>

<TabItem label="JavaScript">
* XFLが負の場合は1を返し、それ以外の場合は0を返します
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t float_sign (
    int64_t float1
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function float_sign(f1: bigint): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t sign =
    float_sign(float_one());
```

<Aside type="tip">
XFL内部の符号ビットは、XFLが負の場合に`0`ですが、この関数は標準的なコンピューティングの慣例に従い、負の場合に`1`を返します。
</Aside>
</TabItem>

<TabItem label="JavaScript">
```javascript
const sign =
   float_sign(float_one());
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前   | 型       | 説明                                   |
| ------ | -------- | -------------------------------------- |
| float1 | int64_t | XFL浮動小数点格納数                    |
</TabItem>

<TabItem label="JavaScript">
| 名前 | 型     | 説明                                     |
| ---- | ------ | ---------------------------------------- |
| f1   | bigint | 仮数部を取得する浮動小数点数。           |
</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">
| 型       | 説明                                                                                                                                                                                                                |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>XFLの符号:<br />正の場合は<code>0</code>、負の場合は<code>1</code>。<br /><br />負の場合はエラー:<br /><code>INVALID_FLOAT</code><br />-指定されたパラメーターが有効なXFL格納数ではありません</p> |


</TabItem>

<TabItem label="JavaScript">
| 型                  | 説明                                       |
| ------------------- | ------------------------------------------ |
| bigint or ErrorCode | エラーコード、または仮数部をbigintとして返します。 |
</TabItem>
</Tabs>
