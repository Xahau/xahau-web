import { Tabs, TabItem, LinkButton, Aside } from '@astrojs/starlight/components';

### コンセプト

<LinkButton href="/ja/docs/hooks/concepts/floating-point-numbers-xfl">浮動小数点数 (XFL)</LinkButton>

### 動作

<Tabs>
<TabItem label="C">
* XFLに`-1`を乗算します
* 新しいXFLをint64_tとして返します
</TabItem>

<TabItem label="JavaScript">
* 浮動小数点数表現を符号反転します。
* エラーコード、または符号反転した浮動小数点数をbigintとして返します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t float_negate (
    int64_t float1
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function float_negate(f1: bigint): ErrorCode | bigint
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t negative_one =
    float_negate(float_one());
```

<Aside type="tip" title="特殊ケース">
正規ゼロの符号反転は正規ゼロです。一部の浮動小数点標準（IEEEなど）とは異なり、XFLに「負のゼロ」はありません。
</Aside>
</TabItem>

<TabItem label="JavaScript">
```javascript
const negative_one =
    float_negate(float_one());
```
</TabItem>
</Tabs>

</TabItem>

<TabItem label="JavaScript">


| 名前 | 型     | 説明                 |
| ---- | ------ | -------------------- |
| f1   | bigint | 符号反転する浮動小数点数。 |
</TabItem>
</Tabs>
