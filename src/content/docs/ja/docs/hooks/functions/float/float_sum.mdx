---
title: float_sum
description: 2つのXFL数値を加算します
---
import { Tabs, TabItem, LinkButton, Aside } from '@astrojs/starlight/components';

### コンセプト

<LinkButton href="/ja/docs/hooks/concepts/floating-point-numbers-xfl">浮動小数点数 (XFL)</LinkButton>

### 動作

<Tabs>
<TabItem label="C">
* 2つのXFL (xls17)浮動小数点数の加算を計算します
* 新しいXFLをint64_tとして返します
</TabItem>

<TabItem label="JavaScript">
* 2つの浮動小数点数表現を合計します。
* エラーコード、または合計をbigintとして返します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t float_sum (
    int64_t float1,
    int64_t float2
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function float_sum(f1: bigint, f2: bigint): ErrorCode | bigint
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t two =
    float_sum(float_one(), float_one());
```


</TabItem>

<TabItem label="JavaScript">
```javascript
const two =
  float_sum(float_one(), float_one());
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">


| 名前   | 型       | 説明                                                                                   |
| ------ | -------- | -------------------------------------------------------------------------------------- |
| float1 | int64_t | 加算の第1オペランドを表すXFL浮動小数点格納数                                           |
| float2 | int64_t | 加算の第2オペランドを表すXFL浮動小数点格納数                                           |

<Aside type="tip">
2つの浮動小数点数を減算するには、第2の浮動小数点数に`float_negate`を使用してから、`float_sum`を使用します。
</Aside>
</TabItem>

<TabItem label="JavaScript">


| 名前 | 型     | 説明                     |
| ---- | ------ | ------------------------ |
| f1   | bigint | 合計する第1の浮動小数点数。 |
| f2   | bigint | 合計する第2の浮動小数点数。 |
</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">
| 型       | 説明                                                                                                                                                                                                                                                                        |
| -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>XFL (xls17)格納数<br /><br />負の場合はエラー:<br /><code>INVALID_FLOAT</code><br />-指定されたパラメーターの1つが有効なXFL格納数ではありません<br /><br /><code>OVERFLOW</code><br />-加算結果が大きすぎてXFLに保存できませんでした。</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                  | 説明                                  |
| ------------------- | ------------------------------------- |
| ErrorCode or bigint | エラーコード、または合計をbigintとして返します。 |
</TabItem>
</Tabs>
