import { Tabs, TabItem, LinkButton, Aside } from '@astrojs/starlight/components';

### 概念

<LinkButton href="/docs/hooks/concepts/floating-point-numbers-xfl">Floating Point Numbers (XFL)</LinkButton>

### 動作

<Tabs>
<TabItem label="C">
* 2 つの XFL (xls17) 浮動小数点数の積を計算します。
* 新しい XFL を int64_t として返します。
</TabItem>

<TabItem label="JavaScript">
* 2 つの XFL (xls17) 浮動小数点数の積を計算します。
* エラーコード、または新しい XFL を bigint として返します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t float_multiply (
    int64_t float1,
    int64_t float2
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function float_multiply(f1: bigint, f2: bigint): ErrorCode | bigint
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
int64_t max_vault_pusd =
    float_multiply(vault_xrp, exchange_rate);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const max_vault_pusd =
    float_multiply(vault_xrp, exchange_rate)
```
</TabItem>
</Tabs>

### パラメーター

<Tabs>
<TabItem label="C">
| Name   | Type     | Description                                                 |
| ------ | -------- | ----------------------------------------------------------- |
| float1 | int64_t  | 乗算の第 1 オペランドを表す XFL 浮動小数点包含数です。      |
| float2 | int64_t  | 乗算の第 2 オペランドを表す XFL 浮動小数点包含数です。      |

<Aside type="caution">
一部の乗算はオーバーフローし、`INVALID_FLOAT` エラーになります。ただし **underflow** は XFL Canonical Zero（包含数 = 0）として返ります。
</Aside>
</TabItem>

<TabItem label="JavaScript">
| Name | Type   | Description                                                 |
| ---- | ------ | ----------------------------------------------------------- |
| f1   | bigint | 乗算の第 1 オペランドを表す XFL 浮動小数点包含数です。      |
| f2   | bigint | 乗算の第 2 オペランドを表す XFL 浮動小数点包含数です。      |
</TabItem>
</Tabs>
