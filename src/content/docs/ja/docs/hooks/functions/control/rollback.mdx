### 動作

Hook の実行を拒否状態で終了します。

* 戻り文字列と戻りコードをトランザクションメタデータに記録します。
* すべての状態変更を破棄します。
* すべての `emit()` トランザクションを破棄します。
* 発信元トランザクションの続行を許可しません。

<Aside type="caution">
  発信元トランザクションは <code>tecHOOK_REJECTED</code> で失敗し、手数料が請求されます。参照: Execution Order。
</Aside>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t rollback (
    uint32_t read_ptr,
    uint32_t read_len,
    uint64_t error_code
);
```
</TabItem>
<TabItem label="JavaScript">
```javascript
function rollback(error_msg: string, error_code: number): number
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
rollback("Rejected!", 9, 100);
```
</TabItem>
<TabItem label="JavaScript">
```javascript
rollback('Rejected!', 100)
```
</TabItem>
</Tabs>
