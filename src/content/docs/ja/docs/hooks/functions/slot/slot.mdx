---
title: slot
description: Slotにあるオブジェクトをシリアライズして出力する
---
import { Tabs, TabItem, Aside } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 指定されたSlotに現在入っているオブジェクトをシリアライズする
* オブジェクトのシリアライズ済みバージョンを出力バッファに書き込む

<Aside type="tip" title="代替の使い方">
小さなオブジェクトなら、バッファを使わずに済ませられます。`write_ptr, write_len` に `0, 0` を指定すると、Slotに入っているオブジェクトをビッグエンディアン形式のパック済みデータとして `int64_t` の戻り値で返せるか試みます。この方法では最大63ビットのデータを返せます。
</Aside>
</TabItem>

<TabItem label="JavaScript">
* 指定されたSlotに関連付けられたデータを取得する。
* エラーコードまたはSlotのデータを返す。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t slot (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t slot_no
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function slot(slotno: number): ErrorCode | ByteArray
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
uint8_t txn[512];
int64_t bytes_written = 
    slot(txn, 512, 1); // assumes a transaction is slotted into slot=1
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const txn = slot(1) // assumes a transaction is slotted into slot=1
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前       | 型      | 説明                                                 |
| ---------- | --------- | ----------------------------------------------------------- |
| write_ptr | uint32_t | 出力を格納するのに適したサイズのバッファへのポインター。 |
| write_len | uint32_t | 出力バッファの長さ。                                        |
| slot_no   | uint32_t | Slot番号                                                   |
</TabItem>

<TabItem label="JavaScript">
| 名前   | 型   | 説明     |
| ------ | ------ | --------------- |
| slotno | number | Slot番号       |
</TabItem>
</Tabs>
