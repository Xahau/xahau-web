---
title: slot_type
description: >-
  Slot内オブジェクトのフィールドコードを取得し、必要に応じて別の
  情報も返す
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 指定された`slot_no`が指すオブジェクトを見つける
* その`sf`フィールドコードを判定し、`flags`が使われている場合はそれか、別の情報（下記参照）を返す
</TabItem>

<TabItem label="JavaScript">
* 指定されたSlotの型を取得する。
* エラーコードまたはSlotの型を返す。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t slot_type (
    uint32_t slot_no,
    uint32_t flags
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_type(slotno: number, flags: number): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t fidle_code = 
    slot_type(1, 0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const field_code = slot_type(1, 0)
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前     | 型      | 説明                                                                                                                                                                                                          |
| -------- | --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| slot_no | uint32_t | Slot番号                                                                                                                                                                                                      |
| flags    | uin32_t  | <p>通常の動作では <code>0</code> にする。<br /><br /><code>STI_AMOUNT</code> がネイティブ（XAH）金額か、浮動小数点（IOU）金額かを判定するには <code>1</code> にする。</p> |
</TabItem>

<TabItem label="JavaScript">
| 名前   | 型   | 説明                           |
| ------ | ------ | ------------------------------------- |
| slotno | number | 型を確認するSlot番号。 |
| flags  | number | 型を判定するフラグ。      |
</TabItem>
</Tabs>



### 戻りコード

<Tabs>
<TabItem label="C">
| 型     | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p><code>flags</code> が <code>0</code> の場合:<br />Slotに入っているオブジェクトの <code>sf</code> フィールドコード<br /><br /><code>flags</code> が <code>1</code> の場合:<br />Slotに入っているオブジェクトが <code>STI_AMOUNT</code> で、かつ金額の型がXAHである場合に限り <code>1</code>。<br /><br />負の値ならエラー:<br /><code>DOESNT_EXIST</code><br />-指定された <code>slot_no</code> にオブジェクトが存在しない。<br /><br /><code>NOT_AN_AMOUNT</code><br />- <code>flags</code> が <code>1</code> に設定されているが、Slotに入っているオブジェクトが <code>STI_AMOUNT</code> オブジェクトではない</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                | 説明                                    |
| ------------------- | ---------------------------------------------- |
| ErrorCode or number | エラーコードまたはSlotの型を返す。 |
</TabItem>
</Tabs>
