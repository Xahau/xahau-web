---
title: slot_type
description: スロット内オブジェクトのフィールドコードと、必要に応じて追加情報を取得します
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* 指定した `slot_no` が指すオブジェクトを特定します。
* その `sf` フィールドコードを返します。`flags` を使用した場合は、代わりに他の情報（下記参照）を返します。
</TabItem>

<TabItem label="JavaScript">
* 指定したスロットの type を取得します。
* エラーコード、またはスロットの type を返します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t slot_type (
    uint32_t slot_no,
    uint32_t flags
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_type(slotno: number, flags: number): ErrorCode | number
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
int64_t field_code = 
    slot_type(1, 0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const field_code = slot_type(1, 0)
```
</TabItem>
</Tabs>

### パラメーター

<Tabs>
<TabItem label="C">
| Name    | Type     | Description                                                                                                                                  |
| ------- | -------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| slot_no | uint32_t | スロット番号です。                                                                                                                           |
| flags   | uint32_t | <p>通常動作では <code>0</code> を指定します。<br /><br /><code>STI_AMOUNT</code> 型が native（XAH）額なのか浮動小数点（IOU）額なのかを判定するには <code>1</code> を指定します。</p> |
</TabItem>

<TabItem label="JavaScript">
| Name   | Type   | Description                   |
| ------ | ------ | ----------------------------- |
| slotno | number | type を確認するスロット番号です。 |
| flags  | number | type 判定用の flags です。    |
</TabItem>
</Tabs>

### 戻りコード

<Tabs>
<TabItem label="C">
| Type    | Description                                                                                                                                                                                                                                                                                                                                                                       |
| ------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p><code>flags</code> が <code>0</code> の場合:<br />スロット済みオブジェクトの <code>sf</code> フィールドコードを返します。<br /><br /><code>flags</code> が <code>1</code> の場合:<br />スロット済みオブジェクトが <code>STI_AMOUNT</code> で、かつその額タイプが XAH のときに限り <code>1</code> を返します。<br /><br />負の場合はエラーです。<br /><code>DOESNT_EXIST</code><br />- 指定した <code>slot_no</code> にオブジェクトが存在しません。<br /><br /><code>NOT_AN_AMOUNT</code><br />- <code>flags</code> が <code>1</code> ですが、スロット済みオブジェクトが <code>STI_AMOUNT</code> ではありません。</p> |
</TabItem>

<TabItem label="JavaScript">
| Type                | Description                     |
| ------------------- | ------------------------------- |
| ErrorCode or number | エラーコード、またはスロットの type を返します。 |
</TabItem>
</Tabs>
