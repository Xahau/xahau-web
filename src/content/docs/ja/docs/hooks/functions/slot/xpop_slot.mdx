title: xpop_slot
description: xpopトランザクションBlobとメタデータをシリアライズして出力する
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* `Import` トランザクション上のxpop Blobを見つける
* 見つかったtxオブジェクトとmetaオブジェクトを指定されたSlotに配置する

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t xpop_slot (
    uint32_t slot_no_tx,
    uint32_t slot_no_meta
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function xpop_slot(slotno_tx: number, slotno_meta: number): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
int64_t bytes_written = 
    xpop_slot(1, 2); // assumes a txn is slotted into slot=1 meta is slotted into slot=2
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const bytes_written = xpop_slot(1, 2)
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前           | 型      | 説明                                    |
| -------------- | --------- | ---------------------------------------------- |
| slot_no_tx   | uint32_t | tx Blobを配置するSlot番号。   |
| slot_no_meta | uin32_t  | meta Blobを配置するSlot番号。 |
</TabItem>

<TabItem label="JavaScript">
| 名前         | 型   | 説明                         |
| ------------ | ------ | ----------------------------------- |
| slotno_tx   | number | 取り出すトランザクションSlot番号。 |
| slotno_meta | number | 取り出すメタデータSlot番号。      |
</TabItem>
</Tabs>



### 戻りコード

<Tabs>
<TabItem label="C">
| 型     | 説明                                                                                                                                                                                                                                                |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p><code>PREREQUISITE_NOT_MET</code><br /><br />-発信トランザクションタイプが <code>Import</code> ではなかった。<br /><br /><code>NO_FREE_SLOTS</code><br /><br />- APIが新しいSlotの割り当てを必要とするが、HookのSlot数はすでに最大である。</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                | 説明                                               |
| ------------------- | --------------------------------------------------------- |
| number or ErrorCode | エラーコードまたはpop操作の結果を返す。 |
</TabItem>
</Tabs>
