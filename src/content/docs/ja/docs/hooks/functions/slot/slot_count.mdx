---
title: slot_count
description: Slot内の配列オブジェクトの要素数を数える
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* 指定されたSlot内の配列の要素数を数える
* 要素数を返す

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t slot_count (
    uint32_t slot_no
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_count(slotno: number): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
slot_count(1); // assumes an array is slotted into slot=1
```
</TabItem>

<TabItem label="JavaScript">
```javascript
slot_count(1) // assumes an array is slotted into slot=1
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前     | 型      | 説明     |
| -------- | --------- | --------------- |
| slot_no | uint32_t | Slot番号 |
</TabItem>

<TabItem label="JavaScript">
| 名前   | 型   | 説明     |
| ------ | ------ | --------------- |
| slotno | number | Slot番号 |
</TabItem>
</Tabs>

### 戻りコード

<Tabs>
<TabItem label="C">
| 型     | 説明                                                                                                                                                                                                                                                                              |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>Slotに入っている配列内の要素数<br /><br />負の値ならエラー:<br /><code>DOESNT_EXIST</code><br />-指定されたSlotにオブジェクトが存在しないか、無効なSlotである<br /><br /><code>NOT_AN_ARRAY</code><br />-指定されたSlotに配列オブジェクトが含まれていない</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                | 説明                                    |
| ------------------- | ---------------------------------------------- |
| ErrorCode or number | エラーコードまたは要素数を返す。 |
</TabItem>
</Tabs>
