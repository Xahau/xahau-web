---
title: slot_subarray
description: スロット済み配列にインデックスし、サブオブジェクトを別スロットへ割り当てます
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* `parent_slot` の配列を参照します。
* `array_id` で指定したインデックスの sub-object を取得します。
* その sub-object を `new_slot`、または `new_slot` が 0 の場合は次の利用可能スロットへ配置します。
* 新しいスロット番号を返します。
</TabItem>

<TabItem label="JavaScript">
* 指定した親スロット内に subarray を作成します。
* エラーコード、または subarray 作成結果を返します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t slot_subarray (
    uint32_t parent_slot,
    uint32_t array_id,
    uint32_t new_slot
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_subarray(
    parent_slotno: number,
    array_id: number,
    new_slotno: number
  ): ErrorCode | number
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
int64_t subslot = 0;
subslot =
    slot_subarray(slot_no, i, (uint32_t)subslot);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
let subslot = 0
subslot = slot_subarray(slot_no, i, subslot)
```
</TabItem>
</Tabs>

### パラメーター

<Tabs>
<TabItem label="C">
| Name        | Type     | Description                                                                                                                                                                                                                                                                 |
| ----------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| parent_slot | uint32_t | 親配列が配置されているスロットです。                                                                                                                                                                                                                                        |
| array_id    | uint32_t | <p>探したいフィールドの <code>sf</code> コードです。<br /><br />手動で計算するには、シリアライズされた <code>type</code> を uint32_t の上位 16 ビットへシフトし、<code>field</code> を下位 16 ビットに配置します。<br /><br />例:<br /><code>sfEmitNonce</code> は <code>type</code> が 5、<code>field</code> が 11 なので値は <code>0x050BU</code> になります。</p> |
| new_slot    | uint32_t | 選択した配列インデックスのオブジェクトを配置する新しいスロット番号です。null の場合は次に利用可能なスロットを選びます。 _null でも構いません。_                                                                                                                       |
</TabItem>

<TabItem label="JavaScript">
| Name          | Type   | Description                         |
| ------------- | ------ | ----------------------------------- |
| parent_slotno | number | 親スロット番号です。                |
| array_id      | number | subarray を作成する配列の ID です。 |
| new_slotno    | number | subarray 用の新しいスロット番号です。 |
</TabItem>
</Tabs>

### 戻りコード

<Tabs>
<TabItem label="C">
| Type    | Description                                                                                                                                                                                                                                                                                                                                                                        |
| ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>新しく割り当てられたオブジェクトのスロット番号です。<br /><br />負の場合はエラーです。<br /><br /><code>DOESNT_EXIST</code><br />- 指定した <code>array_id</code> が <code>parent_slot</code> の配列内に存在しません。<br /><br /><code>NO_FREE_SLOTS</code><br />- 新しいスロットの割り当てが必要ですが、Hook がすでに最大スロット数に達しています。<br /><br /><code>NOT_AN_ARRAY</code><br />- 指定した <code>parent_slot</code> に STArray が入っていません。</p> |
</TabItem>

<TabItem label="JavaScript">
| Type   | Description                             |
| ------ | --------------------------------------- |
| number | エラーコード、または subarray 作成結果を返します。 |
</TabItem>
</Tabs>
