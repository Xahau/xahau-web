---
title: slot_clear
description: 現在使用中のSlotを解放する
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* 指定されたSlotを解放し、そこに入っていたオブジェクトを取り除く

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t slot_clear (
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_clear(slotno: number): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
slot_clear(1); // assumes a transaction is slotted into slot=1
```
</TabItem>

<TabItem label="JavaScript">
```javascript
slot_clear(1) // assumes a transaction is slotted into slot=1
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前     | 型      | 説明     |
| -------- | --------- | --------------- |
| slot_no | uint32_t | Slot番号 |
</TabItem>

<TabItem label="JavaScript">
| 名前   | 型   | 説明     |
| ------ | ------ | --------------- |
| slotno | number | Slot番号 |
</TabItem>
</Tabs>



### 戻りコード

<Tabs>
<TabItem label="C">
| 型     | 説明                                                                                                                                                               |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p><code>1</code> またはエラー<br /><br />負の値ならエラー:<br /><code>DOESNT_EXIST</code><br />-指定されたSlotにオブジェクトが存在しないか、無効なSlotである</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                | 説明                                                 |
| ------------------- | ----------------------------------------------------------- |
| ErrorCode \| number | エラーコードまたはクリア操作の結果を返す。 |
</TabItem>
</Tabs>
