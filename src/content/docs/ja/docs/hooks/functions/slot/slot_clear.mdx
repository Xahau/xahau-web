---
title: slot_clear
description: 現在占有されているスロットを解放します
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### 動作

* 指定したスロットを解放し、そこに割り当てられていたオブジェクトを解除します。

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t slot_clear (
    uint32_t slot_no
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_clear(slotno: number): ErrorCode | number
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
slot_clear(1); // assumes a transaction is slotted into slot=1
```
</TabItem>

<TabItem label="JavaScript">
```javascript
slot_clear(1) // assumes a transaction is slotted into slot=1
```
</TabItem>
</Tabs>

### パラメーター

<Tabs>
<TabItem label="C">
| Name    | Type     | Description   |
| ------- | -------- | ------------- |
| slot_no | uint32_t | スロット番号です。 |
</TabItem>

<TabItem label="JavaScript">
| Name   | Type   | Description   |
| ------ | ------ | ------------- |
| slotno | number | スロット番号です。 |
</TabItem>
</Tabs>

### 戻りコード

<Tabs>
<TabItem label="C">
| Type    | Description                                                                                                                                  |
| ------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p><code>1</code> またはエラーです。<br /><br />負の場合はエラーです。<br /><code>DOESNT_EXIST</code><br />- 指定したスロットにオブジェクトが存在しない、または無効なスロットです。</p> |
</TabItem>

<TabItem label="JavaScript">
| Type                | Description                               |
| ------------------- | ----------------------------------------- |
| ErrorCode \| number | エラーコード、または clear 操作の結果を返します。 |
</TabItem>
</Tabs>
