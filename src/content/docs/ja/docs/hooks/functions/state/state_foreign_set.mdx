---
title: state_foreign_set
description: 指定したキー、値、名前空間について、他のアカウント上のHook状態を設定する
---
import { Tabs, TabItem, Aside } from '@astrojs/starlight/components';

### 動作

<Tabs>
<TabItem label="C">
* `kread_ptr`から32バイトのHook状態キーを読み取る
* `read_ptr`から任意長のデータ（値）を読み取る
* `nread_ptr`から32バイトの名前空間を読み取る
* `aread_ptr`から20バイトのアカウントIDを読み取る
* 指定した名前空間内の指定したアカウントのHook状態キーを、その値で更新する
* ただし、そのアカウント上の [Grant](https://xrpl-hooks.readme.io/docs/grants)がこれを許可している場合に限る。
* Hookアカウントを`aread_ptr`で指定した場合、動作は`state_set`と同じですが、`nread_ptr`を通じて名前空間も指定できます
</TabItem>

<TabItem label="JavaScript">
* 指定した値、キー、名前空間、アカウントIDで外部Hook状態を設定する。
* この操作には承認と、このアクションを許可するGrantが必要です。
* Hook状態に書き込まれたバイト数（データの長さ）を返す。エラー時は負の値。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t state_foreign_set (
    uint32_t read_ptr,
    uint32_t read_len,
    uint32_t kread_ptr,
    uint32_t kread_len,
    uint32_t nread_ptr,
    uint32_t nread_len,
    uint32_t aread_ptr,
    uint32_t aread_len  
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function state_foreign_set(
    value: ByteArray | HexString | undefined | null,
    key: ByteArray | HexString,
    namespace: ByteArray | HexString | undefined | null,
    accountid: ByteArray | HexString | undefined | null
  ): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
#define SBUF(str) (uint32_t)(str), sizeof(str)
if (state_foreign_set(SBUF(vault), SBUF(vault_key), SBUF(namespace), SBUF(account)) < 0)
    rollback(SBUF("Error: could not set foreign state!"), 1);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
if (state_foreign_set(vault, vault_key, namespace, account) < 0)
    rollback("Error: could not set foreign state!", 1);
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前       | 型        | 説明                                                                                                                                              |
| ---------- | --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| read_ptr  | uint32_t | <p>Hook状態に書き込むデータ（値）へのポインター。<br />これが <code>0</code>（null）の場合は、このキーのデータを削除します。<em>null可。</em></p> |
| read_len  | uint32_t | <p>データの長さ。<br />これが <code>0</code>（null）の場合は、このキーのデータを削除します。<em>null可。</em></p>                               |
| kread_ptr | uint32_t | 値を保存するHook状態キーへのポインター。                                                                                                              |
| kread_len | uint32_t | キーの長さ。（常に32である必要があります。）                                                                                                         |
| nread_ptr | uint32_t | キーが属する名前空間へのポインター。                                                                                                                    |
| nread_len | uint32_t | 名前空間の長さ。（常に32である必要があります。）                                                                                                     |
| aread_ptr | uint32_t | 変更しようとしている状態のアカウントIDへのポインター。                                                                                                 |
| aread_len | uint32_t | アカウントIDの長さ。（常に20である必要があります。）                                                                                                 |

<Aside type="caution">
Xahauは内部的にHookデータサイズの上限を設定しています。執筆時点では公開テストネット向けに128バイトへハードコードされていますが、将来的にはバリデーター投票可能な値になります。
</Aside>
</TabItem>

<TabItem label="JavaScript">
| 名前      | 型                     | 説明                                              |
| --------- | ---------------------- | ------------------------------------------------- |
| value     | ByteArray or HexString | 永続化するデータの値。                          |
| key       | ByteArray or HexString | 値を設定するHook状態のキー。                   |
| namespace | ByteArray or HexString | 検索対象のHook名前空間。                       |
| accountid | ByteArray or HexString | 状態の所有者。                                  |
</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">
| 型       | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>Hook状態に書き込まれたバイト数（データの長さ）。<br /><br />負の値の場合はエラーです。<br /><code>OUT_OF_BOUNDS</code><br />-指定されたポインター／長さがHookメモリの外にあります。<br /><br /><code>TOO_BIG</code><br />- <code>kread_len</code> が32を超えている、または<br />- <code>read_len</code> が最大Hookデータサイズを超えています。<br /><br /><code>TOO_SMALL</code><br />- <code>kread_len</code> が0です。<br /><br /><code>NOT_AUTHORIZED</code><br />-この状態変更を許可する適切なHookGrantが対象の外部アカウント上にありません。<br /><br /><code>PREVIOUS_FAILURE_PREVENTS_RETRY</code><br />-この実行中に、以前の <code>state_foreign_set</code> がNOT_AUTHORIZEDで失敗したため、この実行中はそれ以降のこのAPI呼び出しは許可されません。</p> |
</TabItem>

<TabItem label="JavaScript">
| 型                  | 説明                                                                                          |
| ------------------- | ---------------------------------------------------------------------------------------------- |
| number or ErrorCode | Hook状態に書き込まれたバイト数（データの長さ）を返します。エラー時は負の値です。               |
</TabItem>
</Tabs>
