title: etxn_reserve
description: トランザクションが正常に発行されるために必要な手数料を見積もります
---
import { Tabs, TabItem, LinkButton } from '@astrojs/starlight/components';

### 概念

<LinkButton href="/ja/docs/hooks/concepts/emitted-transactions">発行トランザクション</LinkButton>

### 動作

* このHookが実行中に発行する可能性のある発行トランザクション数を指定します。

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t etxn_fee_base (
    uint32_t count
);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
function etxn_reserve(count: number): ErrorCode | number
```
</TabItem>
</Tabs>



### 例

<Tabs>
<TabItem label="C">
```c
etxn_reserve(2);
```


</TabItem>

<TabItem label="JavaScript">
```javascript
etxn_reserve(2)
```
</TabItem>
</Tabs>



### パラメーター

<Tabs>
<TabItem label="C">
| 名前  | 型        | 説明                                                                                 |
| ----- | --------- | ------------------------------------------------------------------------------------------- |
| count | uint32_t | 1回の実行中にこのHookが発行する可能性のあるトランザクションの最大数。 |


</TabItem>

<TabItem label="JavaScript">


| 名前  | 型     | 説明                                                      |
| ----- | ------ | ---------------------------------------------------------------- |
| count | number | このHookが発行を許可されるトランザクションの最大数。 |
</TabItem>
</Tabs>



### 戻り値

<Tabs>
<TabItem label="C">
| 型       | 説明                                                                                                                                                                                                                                                                                                                                                            |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>このHookが発行できる発行トランザクションの最大数。これは常に <code>count</code> パラメーターと同じで、そうでなければ下記のエラーになります。<br /><br />負の場合はエラー:<br /><code>ALREADY_SET</code><br />- Hookがこの関数をすでに以前に呼び出しています。<br /><br /><code>TOO_BIG</code><br />-指定された発行トランザクション数が大きすぎます。</p> |


</TabItem>

<TabItem label="JavaScript">


| 型     | 説明                                                                        |
| ------ | ---------------------------------------------------------------------------------- |
| number | エラーがある場合はErrorCodeを返し、成功すると設定されたトランザクション数を返します。 |
</TabItem>
</Tabs>
