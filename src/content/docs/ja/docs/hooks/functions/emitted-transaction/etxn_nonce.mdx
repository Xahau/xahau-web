---
title: etxn_nonce
description: emit されるトランザクションで使用する 32 バイト nonce を生成します
---
import { Tabs, TabItem, LinkButton } from '@astrojs/starlight/components';

### 概念

<LinkButton href="/docs/hooks/concepts/emitted-transactions">Emitted Transactions</LinkButton>

### 動作

<Tabs>
<TabItem label="C">
* 32 バイトのハッシュを `write_ptr` に書き込みます。
</TabItem>

<TabItem label="JavaScript">
* エラー時は ErrorCode、成功時は nonce 値を含む配列を返します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t etxn_nonce (
    uint32_t write_ptr,
    uint32_t write_len
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function etxn_nonce(): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
uint8_t n[32];
int64_t bytes_written = 
    etxn_nonce(n, 32);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const nonce = etxn_nonce()
```
</TabItem>
</Tabs>

### パラメーター

<Tabs>
<TabItem label="C">
| Name      | Type     | Description                                                    |
| --------- | -------- | -------------------------------------------------------------- |
| write_ptr | uint32_t | 出力を格納するのに十分なサイズのバッファへのポインタです。少なくとも 32 バイト必要です。 |
| write_len | uint32_t | 出力バッファの長さです。                                       |
</TabItem>

<TabItem label="JavaScript">
パラメーターはありません
</TabItem>
</Tabs>
