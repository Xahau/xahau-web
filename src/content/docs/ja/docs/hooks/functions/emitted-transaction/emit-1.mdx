### 概念

<LinkButton href="/docs/hooks/concepts/emitted-transactions">Emitted Transactions</LinkButton>

### 動作

<Tabs>
<TabItem label="C">
* `read_ptr` からトランザクションを読み取ります。
* emission rules に対してトランザクションを検証します。
* 有効であればそのトランザクションをコンセンサスへ emit します。
* canonical transaction hash を `write_ptr` に書き込みます。
</TabItem>

<TabItem label="JavaScript">
* 指定された transaction JSON を emit します。
* 成功時には emit されたトランザクションハッシュを返します。
* エラー時にはエラーコードを返します。
</TabItem>
</Tabs>

### 定義

<Tabs>
<TabItem label="C">
```c
int64_t emit (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t read_ptr,
    uint32_t read_len
);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
function emit(
    txJson: Record<string, any> | Transaction
  ): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### 例

<Tabs>
<TabItem label="C">
```c
if (emit(tx, tx_len) < 0)
    rollback("Failed to emit!", 15, 1);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const emitResult = emit(txJson)
if(typeof emitResult === 'number')
    rollback("Failed to emit!", 1)
```
</TabItem>
</Tabs>
