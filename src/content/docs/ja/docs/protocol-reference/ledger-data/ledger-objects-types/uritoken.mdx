`URIToken` オブジェクトはURIトークンを表し、レジャー内で一意のリソース識別子を表現するために使用できます。

### JSON例

```json
{
  "Owner": "rUn84CUYbNjRoTQ6mSW7BVJPSVJNLb1QLo",
  "OwnerNode": "0000000000000000",
  "Issuer": "rfkE1aSy9G8Upk4JssnwBxhEv5p4mn2KTy",
  "URI": "DEADBEEF",
  "Digest": "46060241FABCF692D4D934BA2A6C4427CD4279083E38C77CBE642243E43BE291",
  "Amount": "100000000",
  "Destination": "rUn84CUYbNjRoTQ6mSW7BVJPSVJNLb1QLo",
  "PreviousTxnID": "5463C6E08862A1FAE5EDAC12D70ADB16546A1F674930521295BC082494B62924",
  "PreviousTxnLgrSeq": 6,
  "LedgerEntryType": "URIToken",
  "index": "49647F0D748DC3FE26BDACBC57F251AADEFFF391403EC9BF87C97F67E9977FB0"
}
```

### フィールド

`URIToken` オブジェクトには次のフィールドがあります。

| フィールド | JSON型 | \[内部型]\[] | 必須? | 説明 |
| --- | --- | --- | --- | --- |
| `Owner` | String | Account | はい | URI Tokenの所有者です。 |
| `OwnerNode` | String | UInt64 | はい | ディレクトリが複数ページに分かれている場合に、所有者のディレクトリのどのページがこのオブジェクトへリンクするかを示すヒントです。 |
| `Issuer` | String | Account | はい | URI Tokenの発行者です。 |
| `URI` | String | VL | はい | このトークンが表すURIです。 |
| `Digest` | String | Hash256 | いいえ | このURI Tokenの特定識別子として所有者が提供する任意の256ビットハッシュです。 |
| `Amount` | String or Object | Amount | いいえ | URI Tokenの金額です。 |
| `Destination` | String | Account | いいえ | URI Tokenの想定受取人です。 |
| `PreviousTxnID` | String | Hash256 | はい | このオブジェクトを最後に変更したトランザクションの識別ハッシュです。 |
| `PreviousTxnLgrSeq` | Number | UInt32 | はい | このオブジェクトを最後に変更したトランザクションを含むレジャーのインデックスです。 |
| `LedgerEntryType` | String | UInt16 | はい | `0x0073` の値は、文字列 `URIToken` に対応し、このオブジェクトがURI Tokenオブジェクトであることを示します。 |

### URITokenのフラグ

`URIToken` オブジェクトには次のフラグ値を設定できます。

| フラグ名 | 16進値 | 10進値 | 対応するURITokenMintフラグ | 説明 |
| --- | --- | --- | --- | --- |
| `lsfBurnable` | `0x00000001` | 1 | `tfBurnable` | 発行者（または発行者により承認された主体）が、ミントされた `URIToken` を破棄できます。（`URIToken` の所有者は_常に_破棄できます。） |

### URIToken ID形式

`URIToken` オブジェクトのIDは、次の値を順番に連結したものの \[SHA-512Half]\[]です。

* URI Tokenのスペースキー（`0x0055`）
* URI Tokenの発行者のAccountID
* URI Tokenが表すURI
