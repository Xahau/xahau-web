---
title: SetFee
---
`SetFee`擬似トランザクションは、Fee Votingの結果としてトランザクションコストまたはリザーブ要件が変わったことを示します。

**注:** 擬似トランザクションを送信することはできませんが、レジャーの処理中に見つかることがあります。

### JSON例

```json
{
    "Account": "rrrrrrrrrrrrrrrrrrrrrhoLvTp",
    "BaseFee": "000000000000000A",
    "Fee": "0",
    "ReferenceFeeUnits": 10,
    "ReserveBase": 20000000,
    "ReserveIncrement": 5000000,
    "Sequence": 0,
    "SigningPubKey": "",
    "TransactionType": "SetFee",
    "date": 439578860,
    "hash": "1C15FEA3E1D50F96B6598607FC773FF1F6E0125F30160144BE0C5CBC52F5151B",
    "ledger_index": 3721729,
}
```

| フィールド          | JSON型          | \[内部型]\[]        | 説明                                                                                                                                                                                                              |
| ------------------- | ---------------- | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `BaseFee`           | String           | UInt64              | 参照トランザクションの手数料をXAHのdropsで表した値を、16進数で示したものです。（これは負荷によるスケーリング前のトランザクションコストです。）                                                                                              |
| `ReferenceFeeUnits` | Unsigned Integer | UInt32              | 参照トランザクションの、fee unitsによるコスト                                                                                                                                                                     |
| `ReserveBase`       | Unsigned Integer | UInt32              | 基本リザーブ。単位はdrops                                                                                                                                                                                               |
| `ReserveIncrement`  | Unsigned Integer | UInt32              | 増分リザーブ。単位はdrops                                                                                                                                                                                        |
| `LedgerSequence`    | Number           | UInt32              | _(一部の歴史的な`SetFee`擬似トランザクションでは省略)_ この擬似トランザクションが現れるレジャーバージョンのインデックスです。同じ変更の他の出現と区別するために使われます。 |

_\[XAHFees amendment]\[]_が有効な場合、`SetFee`擬似トランザクションは代わりに次のフィールドを使います。

| フィールド              | JSON型   | \[内部型]\[]        | 説明                                                                                                                                                                                                              |
| ----------------------- | --------- | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `BaseFeeDrops`          | String    | Amount              | 参照トランザクションの手数料をXAHのdropsで表したものです。（これは負荷によるスケーリング前のトランザクションコストです。）                                                                                                      |
| `ReserveBaseDrops`      | String    | Amount              | 基本リザーブ。単位はdrops                                                                                                                                                                                               |
| `ReserveIncrementDrops` | String    | Amount              | 増分リザーブ。単位はdrops                                                                                                                                                                                        |
| `LedgerSequence`        | Number    | UInt32              | _(一部の歴史的な`SetFee`擬似トランザクションでは省略)_ この擬似トランザクションが現れるレジャーバージョンのインデックスです。同じ変更の他の出現と区別するために使われます。 |
