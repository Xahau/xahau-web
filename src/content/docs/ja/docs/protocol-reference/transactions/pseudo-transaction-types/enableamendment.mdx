---
title: EnableAmendment
---
`EnableAmendment`擬似トランザクションは、提案中のAmendmentの状態が次のいずれかになったときに、その変化を示します。

* バリデーターから過半数超の承認を得たとき。
* 過半数超の承認を失ったとき。
* Xahauプロトコルで有効になったとき。

### JSON例

```json
{
  "Account": "rrrrrrrrrrrrrrrrrrrrrhoLvTp",
  "Amendment": "42426C4D4F1009EE67080A9B7965B44656D7714D104A72F9B4369F97ABF044EE",
  "Fee": "0",
  "LedgerSequence": 21225473,
  "Sequence": 0,
  "SigningPubKey": "",
  "TransactionType": "EnableAmendment"
}  
```

| フィールド       | JSON型   | \[内部型]\[]        | 説明                                                                                                                                        |
| ---------------- | --------- | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Amendment`      | String    | Hash256             | Amendmentの一意識別子です。人間が読める名前を意図したものではありません。既知のAmendmentの一覧はAmendmentsを参照してください。            |
| `LedgerSequence` | Number    | UInt32              | この擬似トランザクションが現れる \[ledger index]\[]です。同じ変更の他の出現と区別するために使われます。 |

### EnableAmendmentフラグ

EnableAmendment擬似トランザクションの`Flags`値は、その擬似トランザクションを含むレジャー時点でのAmendmentの状態を示します。

`Flags`値が `0`（フラグなし）または`Flags`フィールドが省略されている場合、そのAmendmentは有効化されており、その後のすべてのレジャーに適用されます。その他の`Flags`値は次のとおりです。

| フラグ名         | 16進値       | 10進値        | 説明                                                                                                    |
| ---------------- | ------------ | ------------- | -------------------------------------------------------------------------------------------------------------- |
| `tfGotMajority`  | `0x00010000` | 65536         | このAmendmentへの支持が、このレジャーバージョンから信頼済みバリデーターの80%以上に増加しました。  |
| `tfLostMajority` | `0x00020000` | 131072        | このAmendmentへの支持が、このレジャーバージョンから信頼済みバリデーターの80%未満に減少しました。 |
