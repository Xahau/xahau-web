---
title: TEFコード
---
これらのコードは、トランザクションが失敗してレジャーには含まれなかったものの、理論上のレジャーでは成功していた可能性があることを示します。通常、これはそのトランザクションが今後どのレジャーでも成功しなくなったことを意味します。数値は-199から-100の範囲です。各エラーに対する正確なコードは変更される可能性があるため、依存しないでください。

**注意:** `tef`コードのトランザクションはレジャーに適用されず、Xahauの状態を変更することもありません。ただし、暫定的に失敗したトランザクションでも、再適用後に別のコードで成功または失敗する可能性があります。詳細は、結果の確定性およびReliable Transaction Submissionを参照してください。

| コード                           | 説明                                                                                                                                                                                                                                                                                                      |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `tefALREADY`                     | 同一のトランザクションがすでに適用されています。                                                                                                                                                                                                                                                             |
| `tefBAD_ADD_AUTH`                | **DEPRECATED.**                                                                                                                                                                                                                                                                                                  |
| `tefBAD_AUTH`                    | このアカウントに署名するために使われた鍵には、このアカウントを変更する権限がありません。（アカウントがRegular Keyと同じ鍵セットを持っていれば、認可される可能性があります。）                                                                                                                                                     |
| `tefBAD_AUTH_MASTER`             | このトランザクションを認可するために提供された単一署名はmaster keyと一致しませんが、このアドレスにはregular keyが関連付けられていません。                                                                                                                                                                   |
| `tefBAD_LEDGER`                  | トランザクションの処理中に、レジャーが予期しない状態にあることが判明しました。再現できる場合は、[問題を報告](https://github.com/ripple/rippled/issues)してください。                                                                                                         |
| `tefBAD_QUORUM`                  | トランザクションはマルチ署名でしたが、含まれる署名の重みの合計がquorumに達していませんでした。                                                                                                                                                                                                      |
| `tefBAD_SIGNATURE`               | トランザクションはマルチ署名でしたが、送信元アカウントに関連付けられたSignerListに含まれないアドレスの署名が含まれていました。                                                                                                                                                                         |
| `tefCREATED`                     | **DEPRECATED.**                                                                                                                                                                                                                                                                                                  |
| `tefEXCEPTION`                   | トランザクションの処理中に、サーバーが予期しない状態になりました。たとえば、トランザクションのバイナリデータが著しく不正な場合など、予期しない入力が原因かもしれません。再現できる場合は、[問題を報告](https://github.com/ripple/rippled/issues)してください。 |
| `tefFAILURE`                     | トランザクション適用時の原因不明の失敗です。                                                                                                                                                                                                                                                                 |
| `tefINTERNAL`                    | トランザクションを適用しようとしたとき、サーバーが予期しない状態になりました。再現できる場合は、[問題を報告](https://github.com/ripple/rippled/issues)してください。                                                                                                               |
| `tefINVARIANT_FAILED`            | トランザクションコストを回収しようとしたときに、不変条件チェックに失敗しました。\[EnforceInvariants amendment]\[]により追加されました。再現できる場合は、[問題を報告](https://github.com/ripple/rippled/issues)してください。                                                                                        |
| `tefMASTER_DISABLED`             | トランザクションはアカウントのmaster keyで署名されていましたが、そのアカウントでは`lsfDisableMaster`フィールドが設定されています。                                                                                                                                                                                                  |
| `tefMAX_LEDGER`                  | トランザクションに`LastLedgerSequence`パラメーターが含まれていましたが、現在のレジャーのシーケンス番号がすでに指定値を上回っています。                                                                                                                                                                  |
| `tefNFTOKEN_IS_NOT_TRANSFERABLE` | トランザクションはnon-fungible tokenを別のアカウントへ送ろうとしましたが、`NFToken`で`lsfTransferable`フラグが無効になっており、しかも転送先が発行者でも発行者からでもありませんでした。_(NonFungibleTokensV1_1 amendmentにより追加。)_                                                               |
| `tefNO_AUTH_REQUIRED`            | \[TrustSet transaction]\[]がトラストラインを認可済みにしようとしましたが、対応するアカウントでは`lsfRequireAuth`フラグが有効になっていないため、認可は不要です。                                                                                                                          |
| `tefNO_TICKET`                   | トランザクションはTicketを使おうとしましたが、指定された`TicketSequence`番号はレジャーに存在せず、送信者の現在のシーケンス番号より前であるため将来も作成できません。                                                                                          |
| `tefNOT_MULTI_SIGNING`           | トランザクションはマルチ署名でしたが、送信元アカウントに定義されたSignerListがありません。                                                                                                                                                                                                                             |
| `tefPAST_SEQ`                    | トランザクションのシーケンス番号が、トランザクションを送信するアカウントの現在のシーケンス番号より小さいです。                                                                                                                                                                                         |
| `tefTOO_BIG`                     | このトランザクションは、レジャー内のあまりにも多くのオブジェクトに影響を与えます。たとえば、これは \[AccountDelete transaction]\[]ですが、削除対象のアカウントがレジャー内で1000個を超えるオブジェクトを所有しています。                                                                                                                        |
| `tefWRONG_PRIOR`                 | トランザクションには`AccountTxnID`フィールド（または非推奨の`PreviousTxnID`フィールド）が含まれていましたが、そこに指定された値がアカウントの直前のトランザクションと一致しません。                                                                                                                              |
| `tefPAST_IMPORT_SEQ`             | import sequence numberがすでに使用されているため、トランザクションは失敗しました。                                                                                                                                                                                                                                 |
| `tefPAST_IMPORT_VL_SEQ`          | import validator list sequence numberがすでに使用されているため、トランザクションは失敗しました。                                                                                                                                                                                                                  |
