---
title: TECコード
---
これらのコードは、トランザクションが失敗したものの、トランザクションコストを適用するためにレジャーへ適用されたことを示します。数値は100から199の範囲です。数値ではなく文字列コードを使うことを推奨します。

`tec`コードのトランザクションは、トランザクションコストとして支払われたXAHを消費し、シーケンス番号も消費します。大半の場合、そのほかの処理は行いませんが、例外もあります。たとえば、`tecOVERSIZE`になるトランザクションでも、資金不足のOfferをいくつか片付けます。トランザクションが正確に何をしたかは、必ずトランザクションメタデータを確認してください。

**注意:** `tec`コードで暫定的に失敗したトランザクションでも、再適用後に別のコードで成功または失敗する可能性があります。検証済みレジャー版に現れた時点で、その結果は最終確定です。詳細は、結果の確定性およびReliable Transaction Submissionを参照してください。
