---
title: TELコード
---
これらのコードは、トランザクションのローカルサーバー処理におけるエラーを示します。構成や負荷レベルが異なる別のサーバーなら、そのトランザクションを正常に処理できる可能性があります。数値は-399から-300の範囲です。各エラーに対する正確なコードは変更される可能性があるため、依存しないでください。

**注意:** `tel`コードのトランザクションはレジャーに適用されず、Xahauの状態を変更することもありません。ただし、これらのトランザクションは自動的にキャッシュされ、後で再試行される場合があります。暫定的に失敗したトランザクションでも、再適用後に別のコードで成功または失敗する可能性があります。詳細は、結果の確定性およびReliable Transaction Submissionを参照してください。

| コード                                 | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| -------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `telBAD_DOMAIN`                        | トランザクションで指定されたdomain値（たとえば \[AccountSet transaction]\[]の`Domain`フィールド）が使えません。おそらく、レジャーに保存するには長すぎます。                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `telBAD_PATH_COUNT`                    | トランザクションに、ローカルサーバーでは処理しきれないほど多くのパスが含まれています。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `telBAD_PUBLIC_KEY`                    | トランザクションで指定されたpublic key値（たとえば \[AccountSet transaction]\[]の`MessageKey`フィールド）が使えません。おそらく、長さが正しくありません。                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| `telCAN_NOT_QUEUE`                     | トランザクションはopen ledger costを満たしていませんでしたが、このサーバーはキュー制限を満たさなかったため、このトランザクションをキューに入れませんでした。たとえば、送信者がすでにキューに10件の別トランザクションを持っている場合、このコードが返されます。後で再試行するか、`Fee`フィールドでより高いトランザクションコストを指定した置き換えトランザクションに署名して送信してください。                                                                                                                                                                                                                                                                 |
| `telCAN_NOT_QUEUE_BALANCE`             | トランザクションはopen ledger costを満たしておらず、さらに、すでにキューに入っているトランザクションの想定XAHコスト合計がアカウントの予想残高を上回っているため、トランザクションキューにも追加されませんでした。後で再試行するか、別のサーバーへ送信してみてください。\[New in: rippled 0.70.2]\[]                                                                                                                                                                                                                                                                                                                              |
| `telCAN_NOT_QUEUE_BLOCKS`              | トランザクションはopen ledger costを満たしておらず、さらにトランザクションキューにも追加されませんでした。このトランザクションは、認可方法を変更することで同じ送信元からすでにキューにあるトランザクションを妨げてしまうため、既存のキュー内トランザクションを置き換えられませんでした。（これにはすべての \[SetRegularKey]\[]と \[SignerListSet]\[]のトランザクション、および`RequireAuth`/`OptionalAuth`、`DisableMaster`、`AccountTxnID`フラグを変更する \[AccountSet]\[]トランザクションが含まれます。）後で再試行するか、別のサーバーへ送信してみてください。\[New in: rippled 0.70.2]\[]                                                      |
| `telCAN_NOT_QUEUE_BLOCKED`             | トランザクションはopen ledger costを満たしておらず、さらにトランザクションキューにも追加されませんでした。同じ送信元から先行してキューに入っているトランザクションがこのトランザクションをブロックしているためです。（これにはすべての \[SetRegularKey]\[]と \[SignerListSet]\[]のトランザクション、および`RequireAuth`/`OptionalAuth`、`DisableMaster`、`AccountTxnID`フラグを変更する \[AccountSet]\[]トランザクションが含まれます。）後で再試行するか、別のサーバーへ送信してみてください。\[New in: rippled 0.70.2]\[]                                                                                                                                                                 |
| `telCAN_NOT_QUEUE_FEE`                 | トランザクションはopen ledger costを満たしておらず、さらにトランザクションキューにも追加されませんでした。このコードは、同じ送信元とシーケンス番号を持つトランザクションがすでにキューにあり、新しいものが既存トランザクションを置き換えるのに十分なトランザクションコストを支払っていない場合に発生します。キュー内のトランザクションを置き換えるには、新しいトランザクションの`Fee`値が、fee levelで見て少なくとも25%高くなければなりません。`Fee` を増やして再試行するか、既存トランザクションを置き換えないようにより高い`Sequence`番号で送信するか、別のサーバーへ送信してみてください。\[New in: rippled 0.70.2]\[] |
| `telCAN_NOT_QUEUE_FULL`                | トランザクションはopen ledger costを満たしておらず、さらにこのサーバーのトランザクションキューがいっぱいだったため、このトランザクションはキューに入りませんでした。`Fee` を増やして再試行するか、しばらく待ってから再試行するか、別のサーバーへ送信してみてください。新しいトランザクションは、fee levelで見て、キュー内で最小のトランザクションコストより高いコストでなければなりません。\[New in: rippled 0.70.2]\[]                                                                                                                                                                                                            |
| `telFAILED_PROCESSING`                 | トランザクションの処理中に原因不明のエラーが発生しました。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| `telINSUF_FEE_P`                       | トランザクションの`Fee`が、負荷レベルとネットワークレベルの要件から導かれる、サーバーの現在のトランザクションコスト要件を満たしていません。個別のサーバーが今は忙しすぎてトランザクションを処理できない場合、そのトランザクションをキャッシュして後で自動再試行することがあります。                                                                                                                                                                                                                                                                                                                                |
| `telLOCAL_ERROR`                       | 原因不明のローカルエラーです。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| `telNO_DST`_`PARTIAL`                 | これは新しいアカウントを資金供給するXAH支払いですが、`tfPartialPayment`フラグが有効になっていました。これは許可されません。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| `telWRONG_NETWORK`                     | トランザクションが、現在のネットワークに対して誤った`NetworkID`値を指定しています。対象ネットワークに正しい`NetworkID`値を指定するか、正しいネットワークに接続されたサーバーへトランザクションを送信してください。                                                                                                                                                                                                                                                                                                                                                                                                                |
| `telREQUIRES_NETWORK_ID`              | トランザクションに`NetworkID`フィールドが指定されていませんが、現在のネットワークではそれが必要です。`NetworkID`を必要とするネットワーク向けのトランザクションなら、そのフィールドを追加して再試行してください。別のネットワーク向けなら、正しいネットワークに接続されたサーバーへ送信してください。                                                                                                                                                                                                                                                                                                                               |
| `telNETWORK_ID_MAKES_TX_NON_CANONICAL` | トランザクションには`NetworkID`フィールドが指定されていましたが、現在のネットワークでは`NetworkID`を送信しないことが求められています。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| `telNON_LOCAL_EMITTED_TXN`             | その発行トランザクションはローカルで生成されたものではないため、適用できません。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `telIMPORT_VL_KEY_NOT_RECOGNISED`      | トランザクションが別のネットワークで署名されたか、誤ったネットワークに送信されました。`Import` トランザクションでは、検証結果が受信ネットワークのvlキーと一致している必要があります。                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| `telCAN_NOT_QUEUE_IMPORT`              | `Import`トランザクションは直接適用できず、キューにも入れられませんでした。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
