## トランザクション リファレンス

_Transaction_ は、Xahauに変更を加える唯一の方法です。トランザクションの結果が最終確定するのは、署名され、送信され、コンセンサスプロセスに従って検証済みレジャーバージョンに受け入れられた場合のみです。一部のレジャールールは、署名も送信もされない _擬似トランザクション_ も生成しますが、それらもコンセンサスによって受け入れられる必要があります。失敗したトランザクションもレジャーに含まれます。これは、アンチスパムの \[transaction cost]\[]を支払うためにXAH残高を変更するためです。
