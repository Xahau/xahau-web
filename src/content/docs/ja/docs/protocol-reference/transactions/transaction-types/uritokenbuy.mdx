---
title: URITokenBuy
description: >-
  URITokenBuyトランザクションを使うと、ユーザーはissuerからURITokenを購入できます。このトランザクションは、URITokenの所有権をissuerからbuyerに移すために使われます。
---
[\[ソース\]](https://github.com/Xahau/xahaud/blob/dev/src/ripple/app/tx/impl/URIToken.cpp)

_(\[URIToken amendment]\[]により追加。)_

### 例

```json
{
    "TransactionType": "URITokenBuy",
    "Account": "rWYkbWkCeg8dP6rXALnjgZSjjLyih5NXm",
    "URITokenID": "C1AE6DDDEEC05CF2978C0BAD6FE27362498DGS691DC749DCDD3B95992978C0BA",
    "Amount": {
      "issuer": "rf1BiGeXwwQoi8Z2ueFYTEXSwuJYfV2Jpn",
      "currency": "USD",
      "value": "100",
    },
}
```

| フィールド        | JSON型                                                                                                                          | \[Internal Type]\[] | 説明                                         |
| ------------ | ---------------------------------------------------------------------------------------------------------------------------------- | ------------------- | --------------------------------------------------- |
| `Account`    | String                                                                                                                             | AccountID           | buyerのアカウントのアドレス。                     |
| `URITokenID` | String                                                                                                                             | Hash256             | 購入するURITokenの固有識別子。                    |
| `Amount`     | [Currency Amount](/ja/docs/protocol-reference/data-types/currency-formats/#specifying-currency-amounts) | Amount              | URITokenの代金として支払う通貨の金額。            |

### 特別なトランザクションコスト

URITokenBuyトランザクションには標準のトランザクションコストが適用されます。これは、Xahauのすべてのトランザクションに必要な最小feeです。

### エラーケース

すべてのトランザクションで発生しうるエラーに加えて、URITokenBuyトランザクションでは次のトランザクション結果コードが返されることがあります。

| エラーコード                         | 説明                                                                                                                  |
| ---------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| `tecCANT_ACCEPT_OWN_NFTOKEN_OFFER` | tokenのownerがそのofferを受け取ろうとしている場合に発生します。                                                      |
| `tecDST_TAG_NEEDED`                | destination accountがdestination tagを必要としているのに、トランザクションに含まれていない場合に発生します。          |
| `tecNO_PERMISSION`                 | sellerがtokenを販売中にしていない場合、またはtokenの`Destination`がtokenを買うアカウントでない場合に発生します。 |
| `temBAD_CURRENCY`                  | 購入通貨がofferの通貨と一致しない場合に発生します。                                                                       |
| `tecINSUFFICIENT_PAYMENT`          | buy amountがoffer amountを下回る場合に発生します。                                                                       |
| `tecINSUFFICIENT_FUNDS`            | buyerがtokenを購入するための`Fee`を含む十分な資金を持っていない場合に発生します。                                    |
