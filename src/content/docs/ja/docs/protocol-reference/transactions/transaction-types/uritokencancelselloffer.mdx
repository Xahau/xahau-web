---
title: URITokenCancelSellOffer
description: >-
  URITokenCancelSellOfferトランザクションは、XahauでURITokenのsell offerをキャンセルします。
---
[\[ソース\]](https://github.com/Xahau/xahaud/blob/dev/src/ripple/app/tx/impl/URIToken.cpp)

_(\[URIToken amendment]\[]により追加。)_

### 例

```json
{
    "TransactionType": "URITokenCancelSellOffer",
    "Account": "r9cZA1mLK5R5Am25ArfXFmqgNwjZgnfk59",
    "URITokenID": "C1AE6DDDEEC05CF2978C0BAD6FE27362498DGS691DC749DCDD3B95992978C0BA",
}
```

### フィールド

| フィールド        | JSON型 | \[Internal Type]\[] | 説明                                                    |
| ------------ | --------- | ------------------- | -------------------------------------------------------------- |
| `Account`    | String    | AccountID           | キャンセルするsell offerを所有しているアカウントのアドレス。 |
| `URITokenID` | String    | Hash256             | sell offerをキャンセルするURITokenのID。                  |

### エラーケース

すべてのトランザクションで発生しうるエラーに加えて、URITokenCancelSellOfferトランザクションでは次のトランザクション結果コードが返されることがあります。

| エラーコード         | 説明                                                             |
| ------------------ | ----------------------------------------------------------------------- |
| `tecNO_PERMISSION` | トランザクションを実行するアカウントがtokenのownerでない場合に発生します。 |
