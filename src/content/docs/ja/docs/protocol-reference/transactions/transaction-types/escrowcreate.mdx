\[[ソース](https://github.com/Xahau/xahaud/blob/dev/src/ripple/app/tx/impl/URIToken.cpp)]

_(\[Escrow amendment]\[]により追加。)_

### 例

```json
{
    "Account": "rf1BiGeXwwQoi8Z2ueFYTEXSwuJYfV2Jpn",
    "TransactionType": "EscrowCreate",
    "Amount" : {
       "currency" : "USD",
       "value" : "1",
       "issuer" : "rf1BiGeXwwQoi8Z2ueFYTEXSwuJYfV2Jpn"
    },
    "Destination": "rsA2LpzuawewSBQXkiju3YQTMzW13pAAdW",
    "CancelAfter": 533257958,
    "FinishAfter": 533171558,
    "Condition": "A0258020E3B0C44298FC1C149AFBF4C8996FB92427AE41E4649B934CA495991B7852B855810100",
    "DestinationTag": 23480,
    "SourceTag": 11747
}
```

### フィールド

| フィールド            | JSON型                                                                                                                          | \[Internal Type]\[] | 説明                                                                                                                                                                                                                             |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------- | ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Amount`         | [Currency Amount](/ja/docs/protocol-reference/data-types/currency-formats/#specifying-currency-amounts) | Amount              | 送信者の残高から差し引いてescrowに入れる金額。escrow化された後は、`Destination`アドレスに送ることも（`FinishAfter`時刻の後）、送信者に戻すことも（`CancelAfter`時刻の後）できます。 |
| `Destination`    | String                                                                                                                             | AccountID           | escrowされたAmountを受け取るアドレス。                                                                                                                                                                                               |
| `CancelAfter`    | Number                                                                                                                             | UInt32              | _(任意)_ このescrowが期限切れになる時刻。\[seconds since the Ripple Epoch]\[]で指定します。この値は不変であり、この時刻以降にのみ資金を送信者へ戻せます。                                    |
| `FinishAfter`    | Number                                                                                                                             | UInt32              | _(任意)_ escrowされたAmountを受取人に解放できる時刻。\[seconds since the Ripple Epoch]\[]で指定します。この値は不変であり、この時刻に達するまで資金は移動できません。               |
| `Condition`      | String                                                                                                                             | Blob                | _(任意)_ [PREIMAGE-SHA-256 crypto-condition](https://tools.ietf.org/html/draft-thomas-crypto-conditions-02#section-8.1)を表すhexadecimal値。この条件が満たされた場合にのみ資金を受取人へ送れます。 |
| `DestinationTag` | Number                                                                                                                             | UInt32              | _(任意)_ このescrow支払いの送信先をさらに特定する任意のタグ。たとえば送信先アドレスでのホスト型受取人などです。                                                                                 |

`CancelAfter`または`FinishAfter`のいずれかを指定する必要があります。両方を含める場合、`FinishAfter`時刻は`CancelAfter`時刻より前でなければなりません。

\[fix1571 amendment]\[]が有効な場合は、`FinishAfter`、`Condition`、またはその両方を指定する必要があります。\[New in: rippled 1.0.0]\[]
