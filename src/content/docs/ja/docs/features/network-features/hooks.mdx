HooksはXahauネットワークの強力な機能であり、堅牢なスマートコントラクト機能を提供します。HooksはXRP専用に設計された小型で効率的なWebAssemblyモジュールで、XRP Ledger Protocol向けのSmart Contractsと呼ぶことができます。HooksはWebAssemblyへコンパイル可能な任意の言語で記述でき、幅広いビジネスロジックやスマートコントラクトの概念を実装できます。

### トランザクションタイプ

Hooks機能には2種類のトランザクションがあります。

**SetHook**

このトランザクションタイプは、アカウントにHookを設定するために使用されます。

<LinkCard title="SetHook" href="/ja/docs/protocol-reference/transactions/transaction-types/sethook" />

**Invoke**

このトランザクションタイプは、Hookの機能を呼び出し、または起動するために使用されます。

<LinkCard title="Invoke" href="/docs/protocol-reference/transactions/transaction-types/invoke" />
