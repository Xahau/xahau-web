HooksはXRPLネットワークの強力な機能であり、堅牢なスマートコントラクト機能を提供します。XRPLに特化して設計された小さく効率的なWebAssemblyモジュールで、XRP LedgerプロトコルのスマートコントラクトとしてとしてReferenceされることがあります。Hooksは、WebAssemblyでコンパイル可能なあらゆる言語で書くことができ、幅広いビジネスロジックとスマートコントラクトの概念を実装できます。

### トランザクションタイプ

Hooks機能には2種類のトランザクションが含まれます：

**SetHook**

このトランザクションタイプは、アカウントにHookを設定するために使用されます。

<LinkCard title="SetHook" href="/docs/protocol-reference/transactions/transaction-types/sethook" />

**Invoke**

このトランザクションタイプは、Hookの機能を呼び出す（Invoke）するために使用されます。

<LinkCard title="Invoke" href="/docs/protocol-reference/transactions/transaction-types/invoke" />
