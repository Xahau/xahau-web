Xahauネットワークのオファー機能は、分散型取引所システムの重要なコンポーネントです。ユーザーがオファーを作成およびキャンセルできるようにし、動的でレスポンシブなトレード環境を促進します。

### トランザクションタイプ

#### OfferCreate

`OfferCreate`トランザクションは、分散型取引所にオファーを出すために使用されます。

<LinkCard title="OfferCreate" href="/docs/protocol-reference/transactions/transaction-types/offercreate" />

#### OfferCancel

`OfferCancel`トランザクションは、既存のオファーをキャンセルするために使用されます。

<LinkCard title="OfferCancel" href="/docs/protocol-reference/transactions/transaction-types/offercancel" />
