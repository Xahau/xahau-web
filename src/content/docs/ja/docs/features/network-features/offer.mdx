XahauネットワークのOffer機能は、分散型取引所システムの重要なコンポーネントです。ユーザーはOfferを作成およびキャンセルでき、動的で反応性の高い取引環境を実現します。

### トランザクションタイプ

#### OfferCreate

`OfferCreate`トランザクションは、分散型取引所にOfferを出すために使用されます。

<LinkCard title="OfferCreate" href="/ja/docs/protocol-reference/transactions/transaction-types/offercreate" />

#### OfferCancel

`OfferCancel`トランザクションは、既存のOfferをキャンセルするために使用されます。このトランザクションタイプのドキュメントはXahau Documentationで参照されていますが、指定されたコンテキストには含まれていません。

<LinkCard title="OfferCancel" href="/ja/docs/protocol-reference/transactions/transaction-types/offercancel" />
