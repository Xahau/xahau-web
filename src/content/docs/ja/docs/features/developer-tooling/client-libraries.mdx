title: クライアントライブラリ
---
これらのクライアントライブラリは、XAH Ledgerデータへのアクセスと処理における一般的な作業の一部を簡素化し、それぞれのプログラミング言語のネイティブ規約に合致した形式で提供します。

他のプログラミング言語については、HTTP APIを通じてXAH Ledgerにアクセスできます。
