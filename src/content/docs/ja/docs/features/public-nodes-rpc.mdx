title: パブリックノード（RPC）
description: >-
  自分のノードを実行することは素晴らしいことです。すぐに始めたいですか？Xahauのパブリックな
  RPCノードを使用してください。
---
## メインネット（ネットワーク 21337）

* WebSocket
  * `wss://xahau.network`
  * `wss://xahau.org`（エイリアス、一部の広告ブロッカーは.networkをブロックします）
* HTTP POST RPC
  * `https://xahau.network`
  * `https://xahau.org`（エイリアス、一部の広告ブロッカーは.networkをブロックします）
* ネットワーク定義（バイナリコーデックなど）
  * [`https://xahau.network/server_definitions.json`](https://xahau.network/server_definitions.json)
