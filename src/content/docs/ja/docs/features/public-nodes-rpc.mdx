title: パブリックノード (RPC)
description: >-
  自分のノードを運用するのは素晴らしいことです。すぐに始めたい場合は、パブリックXahau
  RPCノードを使用してください。

## Mainnet (network 21337)

* Websocket
  * `wss://xahau.network`
  * `wss://xahau.org` (エイリアス。一部の広告ブロッカーは .networkをブロックします)
* HTTP POST RPC
  * `https://xahau.network`
  * `https://xahau.org` (エイリアス。一部の広告ブロッカーは .networkをブロックします)
* ネットワーク定義 (Binary Codec, ...)
  * [`https://xahau.network/server_definitions.json`](https://xahau.network/server_definitions.json)
