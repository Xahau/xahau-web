### パブリックサーバー

* wss://xahau.network または https://xahau.network （メインネット）
* wss://xahau-test.net または https://xahau-test.net （テストネット）

### サンプルリクエスト

APIにサンプルリクエストを送信するには、次のコマンドを使用します。

### WebSocket

```
{
  "id": 3,
  "command": "account_info",
  "account": "rhBDFMmr3jSjgsWMqBAYaATLy3PuXy395y",
  "strict": true,
  "ledger_index": "validated",
  "api_version": 1
}
```

### WebSocketリクエスト構造

`xahaud`サーバーへのWebSocket接続を確立したら、次のフィールドを含むJSONオブジェクトとしてコマンドを送信できます：

<table><thead><tr><th>フィールド</th><th width="220">型</th><th>説明</th></tr></thead><tbody><tr><td>command</td><td>String</td><td>APIメソッドの名前</td></tr><tr><td>id</td><td>（複数）</td><td><em>（オプション）</em> リクエストの一意の識別子。</td></tr><tr><td>api_version</td><td>Number</td><td><em>（オプション）</em> APIバージョンを指定します。</td></tr></tbody></table>

### JSON-RPC

```
POST https://xahau.network/
Content-Type: application/json

{
    "method": "account_info",
    "params": [
        {
            "account": "rhBDFMmr3jSjgsWMqBAYaATLy3PuXy395y",
            "strict": true,
            "ledger_index": "validated",
            "api_version": 1
        }
    ]
}
```

### JSON-RPCリクエスト構造

<table><thead><tr><th>フィールド</th><th width="220">型</th><th>説明</th></tr></thead><tbody><tr><td>method</td><td>String</td><td>APIメソッドの名前</td></tr><tr><td>params</td><td>Array</td><td><em>（オプション）</em> メソッドのパラメーターを含むJSONオブジェクトを1つ含む配列。</td></tr></tbody></table>

### コマンドライン

```
xahaud account_info rhBDFMmr3jSjgsWMqBAYaATLy3PuXy395y validated strict
```

### コマンドラインリクエスト構造

<table><thead><tr><th>フィールド</th><th width="220">説明</th></tr></thead><tbody><tr><td>xahaud</td><td>xahaudサービスの呼び出しを開始します</td></tr><tr><td>method</td><td>APIメソッドの名前</td></tr><tr><td>params</td><td>（オプション）</td></tr></tbody></table>
