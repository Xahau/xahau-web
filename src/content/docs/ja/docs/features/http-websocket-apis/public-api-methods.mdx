---
title: パブリックAPIメソッド
---
パブリックAPIメソッドを使用して、xahaudサーバーと直接やり取りします。これらのメソッドは必ずしも一般公開用途を意図したものではありませんが、サーバーに接続している任意のクライアントからアクセスできます。&#x20;

### アカウントメソッド

| メソッド            | 説明                                                                   |
| ------------------- | ---------------------------------------------------------------------- |
| account_channels   | アカウントがチャネル送信元であるPayment Channelを一覧表示します。      |
| account_currencies | アカウントが送信または受信できる通貨を一覧表示します。                 |
| account_info       | アカウントに関する基本情報を取得します。                               |
| account_lines      | アカウントのトラストライン情報へアクセスします。                       |
| account_objects    | アカウントが所有するすべてのレジャーオブジェクトを取得します。         |
| account_offers     | アカウントの通貨交換Offerを表示します。                                |
| account_tx         | アカウントのトランザクション履歴を取得します。                         |
| gateway_balances   | アカウントの発行済み総額を計算します。                                 |
| noripple_check     | アカウントのDefault RippleとNo Ripple設定への変更を提案します。        |

### レジャーメソッド

| メソッド        | 説明                                               |
| --------------- | -------------------------------------------------- |
| ledger          | 特定のレジャーバージョンに関する情報を取得します。 |
| ledger_closed  | 直近のクローズ済みレジャーバージョンを取得します。 |
| ledger_current | 現在作業中のレジャーバージョンを取得します。       |
| ledger_data    | 生のレジャー内容へアクセスします。                 |
| ledger_entry   | レジャーバージョンから特定の要素を取得します。     |

### トランザクションメソッド

| メソッド            | 説明                                                       |
| ------------------- | ---------------------------------------------------------- |
| submit              | トランザクションをネットワークへ送信します。               |
| submit_multisigned | マルチシグ済みトランザクションを送信します。               |
| transaction_entry  | 特定のレジャー内のトランザクション詳細を取得します。       |
| tx                  | すべてのレジャーにわたるトランザクション情報を取得します。 |
| sign                | (Admin)トランザクションに暗号学的に署名します。           |
| sign_for           | (Admin)マルチシグに署名を提供します。                     |

### オーダーブックメソッド

| メソッド            | 説明                                                        |
| ------------------- | ----------------------------------------------------------- |
| book_offers        | 2つの通貨を交換するOfferを表示します。                      |
| deposit_authorized | あるアカウントが別のアカウントへ直接Paymentを送信できるか確認します。 |

### Payment Channelメソッド

| メソッド           | 説明                                      |
| ------------------ | ----------------------------------------- |
| channel_authorize | Payment Channelの請求に署名します         |
| channel_verify    | Payment Channel請求署名を検証します。     |

### サブスクリプションメソッド

| メソッド    | 説明                            |
| ----------- | ------------------------------- |
| subscribe   | 対象の更新をリッスンします      |
| unsubscribe | 更新の受信を停止します          |

### サーバー情報メソッド

| メソッド      | 説明                                          |
| ------------- | --------------------------------------------- |
| fee           | トランザクションコストに関する情報を取得します。 |
| server_info  | 人間が読める形式でサーバー状態を取得します。  |
| server_state | 機械可読形式でサーバー状態を取得します。      |
| manifest      | バリデーターの公開鍵詳細を取得します。        |

### ユーティリティメソッド

| メソッド | 説明                                                                 |
| ------ | ---------------------------------------------------------------------- |
| json   | JSONパラメーターでコマンドを実行するためのプロキシです。 _(コマンドラインのみ。)_ |
| ping   | サーバー接続性を確認します。                                          |
| random | 乱数を生成します。                                                     |
