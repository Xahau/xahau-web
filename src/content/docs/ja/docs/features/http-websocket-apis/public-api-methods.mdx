---
title: パブリックAPIメソッド
---
パブリックAPIメソッドを使用して、xahaudサーバーと直接やり取りします。これらのメソッドは必ずしも一般公開を意図したものではありませんが、サーバーに接続されているすべてのクライアントがアクセスできます。

### アカウントメソッド

| メソッド               | 説明                                                              |
| ------------------- | ---------------------------------------------------------------- |
| account_channels   | アカウントがチャンネルソースである決済チャンネルをリストアップします。      |
| account_currencies | アカウントが送受信できる通貨をリストアップします。                      |
| account_info       | アカウントに関する基本情報を取得します。                               |
| account_lines      | アカウントのトラストライン情報にアクセスします。                         |
| account_objects    | アカウントが所有するすべてのレジャーオブジェクトを取得します。             |
| account_offers     | アカウントの通貨交換オファーを表示します。                             |
| account_tx         | アカウントのトランザクション履歴を取得します。                          |
| gateway_balances   | アカウントの総発行量を計算します。                                    |
| noripple_check     | アカウントのDefault RippleとNo Ripple設定への変更を提案します。       |

### レジャーメソッド

| メソッド           | 説明                                         |
| --------------- | ------------------------------------------- |
| ledger          | 特定のレジャーバージョンに関する情報を取得します。     |
| ledger_closed  | 最近クローズされたレジャーバージョンを取得します。      |
| ledger_current | 現在の作業レジャーバージョンを取得します。             |
| ledger_data    | 生のレジャーコンテンツにアクセスします。               |
| ledger_entry   | レジャーバージョンから特定の要素を取得します。          |

### トランザクションメソッド

| メソッド               | 説明                                                  |
| ------------------- | ---------------------------------------------------- |
| submit              | ネットワークにトランザクションを送信します。                   |
| submit_multisigned | マルチ署名トランザクションを送信します。                      |
| transaction_entry  | 特定のレジャーのトランザクションに関する詳細を取得します。       |
| tx                  | すべてのレジャーからトランザクション情報を取得します。           |
| sign                | （管理者）トランザクションを暗号的に署名します。               |
| sign_for           | （管理者）マルチシグネチャに貢献します。                     |

### オーダーブックメソッド

| メソッド               | 説明                                                   |
| ------------------- | ----------------------------------------------------- |
| book_offers        | 2つの通貨を交換するためのオファーを表示します。                 |
| deposit_authorized | あるアカウントが別のアカウントに直接支払いを送れるかどうかを確認します。|

### ペイメントチャンネルメソッド

| メソッド              | 説明                                        |
| ------------------ | ------------------------------------------ |
| channel_authorize | ペイメントチャンネルのクレームに署名します。           |
| channel_verify    | ペイメントチャンネルのクレーム署名を確認します。         |

### サブスクリプションメソッド

| メソッド        | 説明                          |
| ----------- | ---------------------------- |
| subscribe   | サブジェクトの更新をリッスンします。  |
| unsubscribe | 更新の受信を停止します。           |

### サーバー情報メソッド

| メソッド          | 説明                                           |
| ------------- | --------------------------------------------- |
| fee           | トランザクションコストに関する情報を取得します。         |
| server_info  | 人間が読める形式でサーバーのステータスを取得します。    |
| server_state | 機械が読める形式でサーバーのステータスを取得します。    |
| manifest      | バリデーターの公開鍵の詳細を取得します。               |

### ユーティリティメソッド

| メソッド  | 説明                                                                  |
| ------ | -------------------------------------------------------------------- |
| json   | JSONパラメーターでコマンドを実行するためのプロキシ。_（コマンドライン限定。）_ |
| ping   | サーバーの接続性を確認します。                                            |
| random | 乱数を生成します。                                                        |
