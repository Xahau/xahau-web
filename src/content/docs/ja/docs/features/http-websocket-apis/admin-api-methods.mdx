---
title: 管理APIメソッド
---
これらのメソッドは、xahaudサーバー運用の保守を担当する信頼済み担当者専用です。

### キー生成メソッド



| メソッド           | 説明                                                                                                            |
| ------------------ | --------------------------------------------------------------------------------------------------------------- |
| validation_create | xhaudノード用にフォーマット済みのキーペアを生成します。(バリデーターはこのメソッドのキーではなくトークンを使用する必要があります。) |
| wallet_propose    | 新しいアカウント用のキーを生成します。                                                                         |

### ログとデータ管理メソッド



| メソッド        | 説明                                                            |
| --------------- | --------------------------------------------------------------- |
| can_delete     | 指定したレジャーまでのオンライン削除を有効にします。            |
| download_shard | レジャー履歴の特定のシャードをダウンロードします。              |
| ledger_cleaner | 破損データを検出して解決するためにレジャークリーナーを設定します。 |
| ledger_request | 特定のレジャーバージョンについてピアサーバーに問い合わせます。  |
| log_level      | ログの詳細度レベルを表示または変更します。                      |
| logrotate       | ログファイルを再オープンします。                                |
| node_to_shard | レジャーストアからシャードストアへデータを転送します。          |

### サーバー制御メソッド

| メソッド       | 説明                                              |
| -------------- | ------------------------------------------------- |
| ledger_accept | スタンドアロンモードでレジャーを閉じて進めます。  |
| stop           | xahaudサーバーをシャットダウンします。             |

### 署名メソッド

| メソッド  | 説明                                  |
| --------- | ------------------------------------- |
| sign      | トランザクションに暗号学的に署名します。 |
| sign_for | マルチシグに署名を提供します。        |

### ピア管理メソッド

| メソッド                 | 説明                                               |
| ------------------------ | -------------------------------------------------- |
| connect                  | サーバーを特定のピアへ強制的に接続します。         |
| peer_reservations_add  | 特定のピア用の予約済みSlotを追加または更新します。 |
| peer_reservations_del  | 特定のピア用の予約済みSlotを削除します。           |
| peer_reservations_list | すべての予約済みピアSlotを表示します。             |
| peers                    | 接続済みピアに関する情報を取得します               |

### 状態/デバッグメソッド



| メソッド               | 説明                                                        |
| ---------------------- | ----------------------------------------------------------- |
| consensus_info        | コンセンサスプロセスの現在の状態を表示します。              |
| feature                | プロトコルAmendmentに関する情報を取得します。               |
| fetch_info            | サーバーのネットワークとの同期状態を確認します。            |
| get_counts            | サーバー内部とメモリ使用量に関する統計を表示します。        |
| manifest               | 既知のバリデーターの公開鍵詳細を取得します。                |
| print                  | 内部サブシステムに関する情報へアクセスします。              |
| validator_info        | サーバーのバリデーター設定詳細を取得します。                |
| validator_list_sites | バリデーターリストを公開しているサイトを表示します。        |
| validators             | 現在のバリデーターに関する情報を取得します。                |
