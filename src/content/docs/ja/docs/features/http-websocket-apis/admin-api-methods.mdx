---
title: 管理者APIメソッド
---
これらのメソッドは、xahaudサーバー運用の保守を担う信頼された担当者専用です。

### キー生成メソッド

| メソッド              | 説明                                                                                              |
| ------------------ | ------------------------------------------------------------------------------------------------ |
| validation_create | xahaudノード用のフォーマット済みキーペアを生成します。（バリデーターはこのメソッドのキーではなくトークンを使用してください。） |
| wallet_propose    | 新しいアカウントのキーを生成します。                                                                 |

### ロギングとデータ管理メソッド

| メソッド           | 説明                                                       |
| --------------- | --------------------------------------------------------- |
| can_delete     | 指定されたレジャーまでのレジャーのオンライン削除を有効にします。       |
| download_shard | レジャー履歴の特定のシャードをダウンロードします。                  |
| ledger_cleaner | 破損したデータを検出して解決するためにレジャークリーナーを設定します。 |
| ledger_request | 特定のレジャーバージョンのためにピアサーバーにクエリします。            |
| log_level      | ログの詳細レベルを表示または変更します。                          |
| logrotate       | ログファイルを再開します。                                       |
| node_to_shard | レジャーストアからシャードストアにデータを転送します。               |

### サーバー制御メソッド

| メソッド          | 説明                                         |
| -------------- | ------------------------------------------- |
| ledger_accept | スタンドアロンモードでレジャーを閉じて進めます。          |
| stop           | xahaudサーバーをシャットダウンします。               |

### 署名メソッド

| メソッド     | 説明                               |
| --------- | --------------------------------- |
| sign      | トランザクションを暗号的に署名します。         |
| sign_for | マルチシグネチャに貢献します。               |

### ピア管理メソッド

| メソッド                    | 説明                                          |
| ------------------------ | -------------------------------------------- |
| connect                  | サーバーを特定のピアに強制的に接続させます。          |
| peer_reservations_add  | 特定のピア用の予約スロットを追加または更新します。      |
| peer_reservations_del  | 特定のピア用の予約スロットを削除します。              |
| peer_reservations_list | すべての予約ピアスロットを表示します。               |
| peers                    | 接続されたピアに関する情報を取得します。              |

### ステータス/デバッグメソッド

| メソッド                  | 説明                                                |
| ---------------------- | -------------------------------------------------- |
| consensus_info        | コンセンサスプロセスの現在の状態を表示します。             |
| feature                | プロトコルAmendmentに関する情報を取得します。             |
| fetch_info            | サーバーのネットワークとの同期状態を確認します。            |
| get_counts            | サーバー内部とメモリ使用量に関する統計を表示します。        |
| manifest               | 既知のバリデーターの公開鍵の詳細を取得します。             |
| print                  | 内部サブシステムに関する情報にアクセスします。             |
| validator_info        | サーバーのバリデーター設定の詳細を取得します。             |
| validator_list_sites | バリデーターリストを公開しているサイトを表示します。         |
| validators             | 現在のバリデーターに関する情報を取得します。              |
