title: Xahauドキュメント

Xahauドキュメントへようこそ。ここはXahauを理解し、活用するための包括的なガイドです。このドキュメントは、Features、Infrastructure、Technicalの3つの主要カテゴリに分かれています。

各カテゴリでは、Xahauのさまざまな側面について、詳細な解説と手順を確認できます。

### Features

Featuresセクションでは、Xahauの独自の特性についての情報を紹介しています。このセクションで特に重要なのがDeveloper Toolingです。

Developer Toolingページでは、Hooks ToolsとClient Librariesを扱っており、HooksとXahau Networkとのやり取りを簡素化します。このセクションは、Xahauの機能と能力を理解できるように、実践的な例を交えて構成されています。

### Protocol Reference

XahauのドキュメントにあるProtocol Referenceセクションでは、AccountDelete、CheckCancel、Paymentなどのさまざまなトランザクションタイプを含め、プラットフォーム独自の側面を詳しく解説しています。

また、AccountRootやAmendmentsのように、ネットワークの動作に不可欠なさまざまなレジャーオブジェクトも取り上げています。この包括的な概要は、Xahauの機能と能力への理解を深め、ユーザーがネットワークの機能を最大限に活用できるようにすることを目的としています。

### Infrastructure

InfrastructureはXahau Networkの中核です。xahaudソフトウェアを実行するサーバーによって、トランザクションの送信、検証、照会が可能になります。このセクションのドキュメントでは、xahaudのインストール、更新、設定、ビルドの手順を説明します。特定のサーバーのIDを証明する方法や、xahaudをバリデーターとして実行する方法などの概念も扱います。


**このドキュメントが、Xahau Networkを理解し、利用し、貢献するために必要な情報を提供できれば幸いです。ぜひご覧ください。**
