## Referência de Transações

Uma _Transação_ é a única maneira de causar mudanças no Xahau. Os resultados das transações só são definitivos se forem assinados, enviados e aceitos em uma versão de ledger validada após o processo de consenso. Algumas regras do ledger também geram _pseudo-transações_, que não são assinadas ou enviadas, mas ainda devem ser aceitas por consenso. Transações que falham também são incluídas nos ledgers porque modificam saldos de XAH para pagar o \[custo de transação]\[] anti-spam.
