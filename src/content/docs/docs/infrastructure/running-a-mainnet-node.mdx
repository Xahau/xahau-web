## Docker Container

To connect to the Xahau Mainnet you can use this docker container:

[https://github.com/Xahau/mainnet-docker](https://github.com/Xahau/mainnet-docker)

1. Clone the repository
2. Run `./build`
3. Run `./up`
4. Check the README for commands, etc. [https://github.com/Xahau/mainnet-docker/blob/main/README.md](https://github.com/Xahau/mainnet-docker/blob/main/README.md)
5. Connect using WebSockets or RPC:\
   WebSocket: `ws://localhost:16006`\
   RPC (HTTP POST): `http://localhost:16007`

## Local Install
The Xahau Mainnet Docker Container repository contains a [script](https://github.com/Xahau/mainnet-docker/blob/main/xahaud-install-update.sh) that automates the local install process. To install locally:

1. Download and run the install script: <code>curl -sL https://raw.githubusercontent.com/Xahau/mainnet-docker/refs/heads/main/xahaud-install-update.sh | bash</code>
2. Edit the configuration file: <code>/opt/xahahud/etc/xahaud.cfg</code>
3. Evaluate and, if needed, edit the contents of the trusted validators file: <code>/opt/xahaud/etc/validators-xahau.txt</code>
4. Start and enable xahaud to run automatically: <code>systemctl enable --now xahaud</code>

## Binary Files
Users seeking to install xahaud in non-traditional environments (e.g., outside Ubuntu) may benefit from the binary release packages, which are built with necessary dependencies. Download and run the binary:
1. Download the latest release from [https://build.xahau.tech](https://build.xahau.tech)
2. Extract the downloaded file
3. Download and edit the configuration and validators' files using the links in the table below.
3. Run <code>./xahaud --net --conf xahaud.cfg</code> (replace 'xahaud.cfg') with the path to your downloaded configuration file. The validators' file path is defined in the configuration file and thus does not need to be specified at runtime.
