## Tipos de Transacciones

El tipo de una transacción (campo `TransactionType`) es la información más fundamental sobre una transacción. Indica qué tipo de operación se supone que debe realizar la transacción.

Todas las transacciones tienen ciertos campos en común:

* Campos Comunes

Cada tipo de transacción tiene campos adicionales relevantes para el tipo de acción que causa.
