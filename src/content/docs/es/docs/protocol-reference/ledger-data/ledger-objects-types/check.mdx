Un objeto `Check` describe un cheque, similar a un cheque personal en papel, que puede cobrarse por su destinatario para obtener dinero de su remitente. (El pago potencial ya fue aprobado por su remitente, pero no se mueve dinero hasta que se cobra. A diferencia de un Escrow, el dinero para un Check no se aparta, por lo que cobrar el Check podrÃ­a fallar por falta de fondos.)
