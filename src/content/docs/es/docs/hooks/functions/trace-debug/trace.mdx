---
title: trace
description: Escribir el contenido de un buffer en el registro de trazas de Xahaud
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamiento

* Escribe un buffer desde el Hook en el log de trazas junto con un mensaje (si se proporciona)

### Definición

<Tabs>
<TabItem label="C">
```c
int64_t trace (
    uint32_t mread_ptr,
    uint32_t mread_len,
    uint32_t dread_ptr,
    uint32_t dread_len,
    uint32_t as_hex
);
````

</TabItem>

<TabItem label="JavaScript">
```javascript
function trace(
    message: string | null,
    data: any,
    hex: boolean | 0 | 1
  ): ErrorCode
```
</TabItem>
</Tabs>

### Ejemplo

<Tabs>
<TabItem label="C">
```c
#define SBUF(str) (uint32_t)(str), sizeof(str)
trace(SBUF("Contenido del buffer"), SBUF(some_buffer), 1);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
trace("Contenido del buffer", some_buffer, 1);
```
</TabItem>
</Tabs>

### Parámetros

<Tabs>
<TabItem label="C">
| Nombre     | Tipo     | Descripción                                                                                              |
| ---------- | -------- | -------------------------------------------------------------------------------------------------------- |
| mread_ptr  | uint32_t | Puntero a un mensaje a mostrar antes del buffer. <em>Puede ser null.</em>                                 |
| mread_len  | uint32_t | Longitud del mensaje. <em>Puede ser null.</em>                                                            |
| dread_ptr  | uint32_t | Puntero al buffer a mostrar                                                                              |
| dread_len  | uint32_t | Longitud del buffer                                                                                      |
| as_hex     | uint32_t | <p>Si es <code>1</code> muestra el buffer en hexadecimal.<br />Si es <code>0</code> lo muestra en UTF-8.</p> |
</TabItem>

<TabItem label="JavaScript">
| Nombre  | Tipo    | Descripción                                                         |
| ------- | ------- | ------------------------------------------------------------------- |
| message | string  | Mensaje (clave de log) a mostrar antes del buffer (puede ser null)  |
| data    | any     | Datos a registrar                                                   |
| hex     | boolean | <p>¿Mostrar en HEX?<br /><br />0 - NO<br />1 - SÍ</p>                |
</TabItem>
</Tabs>

### Código de retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descripción                                                                                                                                             |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p><code>0</code> si tiene éxito<br /><br />Si es negativo, error:<br /><code>OUT_OF_BOUNDS</code><br />- punteros fuera de la memoria del hook.</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo      | Descripción                                                                                      |
| --------- | ------------------------------------------------------------------------------------------------ |
| ErrorCode | <p>int64_t, valor 0 si tiene éxito</p><p></p><p>Si es negativo, error: OUT_OF_BOUNDS</p>         |
</TabItem>
</Tabs>
