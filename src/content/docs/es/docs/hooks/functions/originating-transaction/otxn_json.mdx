### Comportamiento

<Tabs>
<TabItem label="JavaScript">
* Devuelve la transacción origen en formato JSON.

</TabItem>
</Tabs>

<Tabs>
<TabItem label="JavaScript">
```javascript id="otxn-json-js"
function otxn_json(): ErrorCode | Record<string, any> | Transaction
````

</TabItem>
</Tabs>

### Ejemplo

<Tabs>
<TabItem label="JavaScript">
```javascript id="otxn-json-js-ex"
const tx_json = otxn_json()
```
</TabItem>
</Tabs>

Sin parámetros
