### Conceptos

<LinkButton href="/docs/hooks/concepts/slots-and-keylets">Slots y Keylets</LinkButton>

<Aside type="tip">
No todos los tipos de Keylet están soportados por esta función. Si necesitas otro tipo, puedes generarlo manualmente usando [util_sha512h](https://xrpl-hooks.readme.io/reference/util_sha512h) y consultando los campos requeridos [aquí](https://github.com/Xahau/xahaud/blob/dev/src/ripple/protocol/impl/Indexes.cpp). También puedes usar esta herramienta: [Keylet Tool](https://richardah.github.io/xrpl-keylet-tools/)
</Aside>

### Comportamiento

* Calcula un keylet del tipo `keylet_type` según los parámetros `a` a `f`
* Escribe el keylet serializado de 34 bytes en `write_ptr`

### Definición

<Tabs>
<TabItem label="C">
```c
int64_t util_keylet (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t keylet_type,
    uint32_t a,
    uint32_t b,
    uint32_t c,
    uint32_t d,
    uint32_t e,
    uint32_t f
);
````

</TabItem>

<TabItem label="JavaScript">
```javascript id="util-keylet-js"
function util_keylet(
    keylet_type: number,
    accountid: ByteArray | HexString,
    statekey: ByteArray | HexString,
    namespace: ByteArray | HexString
  ): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### Ejemplo

<Tabs>
<TabItem label="C">
```c id="util-keylet-c"
uint8_t keylet[34];
if (util_keylet(keylet, 34, KEYLET_LINE,
              hook_accid, 20,
              account_field, 20,
              currency_code, 20) != 34)
	rollback("Keylet Failed.", 14, 1);
```
</TabItem>

<TabItem label="JavaScript">
```javascript id="util-keylet-js-ex"
if (util_keylet(
        KEYLET_LINE, hook_accid, account_field, currency_code
        ).length < 34) 
    rollback("Keylet Failed.", 1)
```
</TabItem>
</Tabs>
