### Comportamiento

Verifica una firma criptográfica

* Si la clave pública tiene el prefijo `0xED`, se usa `ED25519`
* En caso contrario, se asume `SECP256k1`

### Definición

<Tabs>
<TabItem label="C">
```c
int64_t util_verify (
    uint32_t dread_ptr,
    uint32_t dread_len,
    uint32_t sread_ptr,
    uint32_t sread_len,
    uint32_t kread_ptr,
    uint32_t kread_len
);
````

</TabItem>

<TabItem label="JavaScript">
```javascript
function util_verify(
    signedData: ByteArray | HexString,
    signature: ByteArray | HexString,
    pubkey: ByteArray | HexString
  ): 0 | 1
```
</TabItem>
</Tabs>

### Ejemplo

<Tabs>
<TabItem label="C">
```c
if (!util_verify(payload_ptr,    payload_len,
                 signature_ptr,  signature_len,
                 publickey_ptr,  publickey_len))
	rollback("Firma inválida", 17, 60);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
if (util_verify(signedData, signature, pubkey))
    rollback("Firma inválida", 60)
```
</TabItem>
</Tabs>
