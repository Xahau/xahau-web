---
title: slot_size
description: Calcular el tamaño serializado de un objeto en un slot
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamiento

<Tabs>
<TabItem label="C">
* Devuelve el número de bytes que ocupa el objeto en el slot especificado cuando está serializado
</TabItem>

<TabItem label="JavaScript">
* Obtiene el tamaño del slot especificado
</TabItem>
</Tabs>

### Definición

<Tabs>
<TabItem label="C">
```c
int64_t slot_size (
    uint32_t slot_no
);
````

</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_size(slotno: number): ErrorCode | number
```
</TabItem>
</Tabs>

### Ejemplo

<Tabs>
<TabItem label="C">
```c
int64_t bytes_needed = slot_size(1); // obtener tamaño del slot 1
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const bytes_needed = slot_size(1); // obtener tamaño del slot 1
```
</TabItem>
</Tabs>

### Parámetros

<Tabs>
<TabItem label="C">
| Nombre  | Tipo     | Descripción       |
| ------- | -------- | ----------------- |
| slot_no | uint32_t | Número del slot   |
</TabItem>

<TabItem label="JavaScript">
| Nombre | Tipo   | Descripción                              |
| ------ | ------ | ---------------------------------------- |
| slotno | number | Número del slot del que obtener el tamaño |
</TabItem>
</Tabs>

### Código de retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descripción                                                                                                                                                                                          |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>Número de bytes que ocupa el objeto cuando está serializado<br /><br />Si es negativo, error:<br /><code>DOESNT_EXIST</code><br />- el slot no contiene ningún objeto o es inválido</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descripción                                      |
| ------------------- | ---------------------------------------------- |
| ErrorCode \| number | Devuelve un código de error o el tamaño del slot |
</TabItem>
</Tabs>
