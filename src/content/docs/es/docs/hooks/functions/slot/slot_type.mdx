---
title: slot_type
description: >-
  Obtener el código de campo de un objeto en un slot y, opcionalmente, otra información
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamiento

<Tabs>
<TabItem label="C">
* Localiza el objeto apuntado por el `slot_no` especificado
* Determina su código de campo `sf` y lo devuelve, u otra información (ver abajo) si se usan `flags`
</TabItem>

<TabItem label="JavaScript">
* Obtiene el tipo del slot especificado
* Devuelve un código de error o el tipo del slot
</TabItem>
</Tabs>

### Definición

<Tabs>
<TabItem label="C">
```c
int64_t slot_type (
    uint32_t slot_no,
    uint32_t flags
);
````

</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_type(slotno: number, flags: number): ErrorCode | number
```
</TabItem>
</Tabs>

### Ejemplo

<Tabs>
<TabItem label="C">
```c
int64_t field_code = 
    slot_type(1, 0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const field_code = slot_type(1, 0)
```
</TabItem>
</Tabs>

### Parámetros

<Tabs>
<TabItem label="C">
| Nombre  | Tipo     | Descripción                                                                                                                                                                                                          |
| ------- | -------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| slot_no | uint32_t | Número del slot                                                                                                                                                                                                      |
| flags   | uint32_t | <p>Para uso normal debe ser <code>0</code>.<br /><br />Para determinar si un tipo <code>STI_AMOUNT</code> contiene una cantidad nativa (XAH) o un valor en coma flotante (IOU), usar <code>1</code>.</p> |
</TabItem>

<TabItem label="JavaScript">
| Nombre | Tipo   | Descripción                               |
| ------ | ------ | ----------------------------------------- |
| slotno | number | Número del slot a consultar               |
| flags  | number | Flags para determinar el tipo             |
</TabItem>
</Tabs>

### Código de retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>Si <code>flags</code> es <code>0</code>:<br />Código de campo <code>sf</code> del objeto en el slot<br /><br />Si <code>flags</code> es <code>1</code>:<br /><code>1</code> solo si el objeto es un <code>STI_AMOUNT</code> y el tipo es XAH<br /><br />Si es negativo, error:<br /><code>DOESNT_EXIST</code><br />- el slot no contiene un objeto<br /><br /><code>NOT_AN_AMOUNT</code><br />- <code>flags</code> es <code>1</code> pero el objeto no es un <code>STI_AMOUNT</code></p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descripción                                      |
| ------------------- | ---------------------------------------------- |
| ErrorCode o number  | Devuelve un código de error o el tipo del slot |
</TabItem>
</Tabs>
