---
title: slot_float
description: >-
  Parsear el STI_AMOUNT en el slot especificado y devolverlo como un número XFL encapsulado
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamiento

<Tabs>
<TabItem label="C">
* Parsea el STI_AMOUNT en el slot especificado y lo devuelve como un número en formato XFL
</TabItem>

<TabItem label="JavaScript">
* Obtiene el valor bigint asociado al slot especificado
* Devuelve un código de error o el valor bigint
</TabItem>
</Tabs>

### Definición

<Tabs>
<TabItem label="C">
```c
int64_t slot_float (
    uint32_t slot_no
);
````

</TabItem>

<TabItem label="JavaScript">
```javascript
function slot_float(slotno: number): ErrorCode | bigint
```
</TabItem>
</Tabs>

### Ejemplo

<Tabs>
<TabItem label="C">
```c
int64_t xfl =
  	slot_float(amt_slot);
```
</TabItem>

<TabItem label="JavaScript">
```javascript
const xfl = slot_float(amt_slot)
```
</TabItem>
</Tabs>

### Parámetros

<Tabs>
<TabItem label="C">
| Nombre  | Tipo     | Descripción       |
| ------- | -------- | ----------------- |
| slot_no | uint32_t | Número del slot   |
</TabItem>

<TabItem label="JavaScript">
| Nombre | Tipo   | Descripción                                              |
| ------ | ------ | -------------------------------------------------------- |
| slotno | number | Número del slot del que obtener el valor bigint          |
</TabItem>
</Tabs>

### Código de retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descripción                                                                                                                                                                                                                                                                    |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| int64_t | <p>Número en formato XFL<br /><br />Si es negativo, error:<br /><code>DOESNT_EXIST</code><br />- el slot no contiene ningún objeto o es inválido<br /><br /><code>NOT_AN_AMOUNT</code><br />- el slot no contiene un objeto <code>STI_AMOUNT</code></p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descripción                                |
| ------------------- | ------------------------------------------ |
| bigint o ErrorCode  | Devuelve un código de error o el valor bigint |
</TabItem>
</Tabs>
