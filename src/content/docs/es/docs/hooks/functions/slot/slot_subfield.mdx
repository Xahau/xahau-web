---
title: slot_subfield
description: Indexar dentro de un objeto en un slot y asignar un subobjeto a otro slot
---
import { Tabs, TabItem } from '@astrojs/starlight/components';

### Comportamiento

<Tabs>
<TabItem label="C">
* Busca el objeto en el slot `parent_slot`
* Recupera el subobjeto en `field_id`
* Coloca el subobjeto en el slot `new_slot` o en el siguiente slot disponible si `new_slot` es 0
* Devuelve el número del nuevo slot
</TabItem>

<TabItem label="JavaScript">
* Crea un subcampo en el slot padre especificado
* Devuelve un código de error o el resultado de la creación del subcampo
</TabItem>
</Tabs>

### Definición

<Tabs>
<TabItem label="C">
```c
int64_t slot_subfield (
    uint32_t parent_slot,
    uint32_t field_id,
    uint32_t new_slot
);
````

</TabItem>

<TabItem label="JavaScript">
```javascript id="slot-subfield-js"
function slot_subfield(
    parent_slotno: number,
    field_id: number,
    new_slotno: number
  ): ErrorCode | number
```
</TabItem>
</Tabs>

### Ejemplo

<Tabs>
<TabItem label="C">
```c id="slot-subfield-c"
int64_t amt_slot = 
  	slot_subfield(oslot, sfAmount, 0);
```
</TabItem>

<TabItem label="JavaScript">
```javascript id="slot-subfield-js-ex"
const amt_slot = slot_subfield(oslot, sfAmount, 0)
```
</TabItem>
</Tabs>

### Parámetros

<Tabs>
<TabItem label="C">
| Nombre       | Tipo     | Descripción                                                                                                                                                                                                                                                                                                                                                                                            |
| ------------ | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| parent_slot  | uint32_t | Slot donde se encuentra el objeto padre                                                                                                                                                                                                                                                                                                                                                               |
| field_id     | uint32_t | <p>Código <code>sf</code> del campo que se desea buscar.<br /><br />Para calcularlo manualmente, toma el <code>type</code> serializado y desplázalo a los 16 bits más altos de un uint32_t, luego coloca el <code>field</code> en los 16 bits más bajos.<br /><br />Ejemplo:<br /><code>sfEmitNonce</code> tiene <code>type</code> 5 y <code>field</code> 11, por lo que su valor es <code>0x050BU</code></p> |
| new_slot     | uint32_t | Nuevo número de slot donde colocar el objeto del campo seleccionado. Si es 0, se usará el siguiente slot disponible |
</TabItem>

<TabItem label="JavaScript">
| Nombre        | Tipo   | Descripción                                      |
| ------------- | ------ | ------------------------------------------------ |
| parent_slotno | number | Número del slot padre                            |
| field_id      | number | Identificador del campo del que crear subcampo   |
| new_slotno    | number | Nuevo número de slot para el subcampo            |
</TabItem>
</Tabs>

### Código de retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>Número de slot del nuevo objeto asignado<br /><br />Si es negativo, error:<br /><br /><code>DOESNT_EXIST</code><br />- el campo no existe en el objeto o el slot está vacío<br /><br /><code>NO_FREE_SLOTS</code><br />- no hay slots disponibles<br /><br /><code>INVALID_FIELD</code><br />- el campo no es un <code>sf</code> válido<br /><br /><code>NOT_AN_OBJECT</code><br />- el objeto en el slot no es un STObject válido</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                | Descripción                                                     |
| ------------------- | --------------------------------------------------------------- |
| number o ErrorCode  | Devuelve un código de error o el resultado de la operación      |
</TabItem>
</Tabs>
