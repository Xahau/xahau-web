---
title: state
description: >-
  Recuperar los datos apuntados por una clave de Hook State y escribirlos en un
  buffer de salida
---
import { Tabs, TabItem, Aside } from '@astrojs/starlight/components';

### Comportamiento

<Tabs>
<TabItem label="C">
* Lee una clave de Hook State de 32 bytes desde `kread_ptr`
* Escribe los datos (valor) asociados a esa clave en el buffer apuntado por `write_ptr`
</TabItem>

<TabItem label="JavaScript">
* Recupera el valor del Hook State asociado a la clave especificada
* Devuelve un código de error o el valor del Hook State para la clave
</TabItem>
</Tabs>

### Definición

<Tabs>
<TabItem label="C">
```c
int64_t state (
    uint32_t write_ptr,
    uint32_t write_len,
    uint32_t kread_ptr,
    uint32_t kread_len  
);
````

</TabItem>

<TabItem label="JavaScript">
```javascript id="state-js"
function state(key: ByteArray | HexString): ErrorCode | ByteArray
```
</TabItem>
</Tabs>

### Ejemplo

<Tabs>
<TabItem label="C">
```c id="state-c"
#define SBUF(str) (uint32_t)(str), sizeof(str)
if (state(SBUF(vault), SBUF(vault_key)) != 16)
    rollback(SBUF("Error: no se pudo leer el estado!"), 1);
```
</TabItem>

<TabItem label="JavaScript">
```javascript id="state-js-ex"
const value = state(SBUF(vault), SBUF(vault_key))
if (typeof value === 'number' || value.length != 16)
    rollback("Error: no se pudo leer el estado!", 1);
```
</TabItem>
</Tabs>

### Parámetros

<Tabs>
<TabItem label="C">
| Nombre     | Tipo     | Descripción                                                       |
| ---------- | -------- | ----------------------------------------------------------------- |
| write_ptr  | uint32_t | Puntero al buffer donde escribir los datos del Hook State        |
| write_len  | uint32_t | Longitud del buffer de escritura                                 |
| kread_ptr  | uint32_t | Puntero a un buffer que contiene la clave del Hook State         |
| kread_len  | uint32_t | Longitud de la clave del Hook State (debe ser 32)                |

<Aside type="tip">
Asegúrate de comprobar el valor de retorno. Una consulta de estado puede fallar por varias razones y el buffer contendrá lo que tuviera antes de la llamada (normalmente ceros).
</Aside>
</TabItem>

<TabItem label="JavaScript">
| Nombre | Tipo                   | Descripción                                           |
| ------ | ---------------------- | ----------------------------------------------------- |
| key    | ByteArray o HexString  | La clave del Hook State de la que obtener el valor    |
</TabItem>
</Tabs>

### Código de retorno

<Tabs>
<TabItem label="C">
| Tipo     | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| int64_t | <p>Número de bytes escritos en el buffer de salida<br /><br />Si es negativo, error:<br /><code>OUT_OF_BOUNDS</code><br />- punteros fuera de la memoria del hook<br /><br /><code>DOESNT_EXIST</code><br />- la clave de Hook State no tiene valor asociado<br /><br /><code>TOO_BIG</code><br />- la clave supera los 32 bytes<br /><br /><code>TOO_SMALL</code><br />- el buffer es demasiado pequeño</p> |
</TabItem>

<TabItem label="JavaScript">
| Tipo                   | Descripción                                               |
| ---------------------- | --------------------------------------------------------- |
| ByteArray o ErrorCode  | Devuelve un código de error o el valor del Hook State     |
</TabItem>
</Tabs>
